# Вступ до машинного навчання

У цьому розділі навчальної програми ви ознайомитесь із фундаментальними поняттями машинного навчання, включаючи що це таке, його історію та методи, які дослідники використовують для застосування у реальних сценаріях. Давайте разом досліджувати цей захопливий світ машинного навчання!
