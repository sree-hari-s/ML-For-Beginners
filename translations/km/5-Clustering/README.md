# бһҳбҹүбһјбһҠбҹӮбһӣбһҖбҹ’бһҡбһ»бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“

бһҖбҹ’бһҡбһ»бһҳбһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһ”бҹ’бһҡбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҹбһ·бһҖбҹ’бһҹбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбһҪбһҷ бһҠбҹӮбһӣбһңбһ¶бһҳбһҫбһӣбһҡбһҖбһңбһҸбҹ’бһҗбһ»бһҠбҹӮбһӣбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ бһҫбһҷбһҖбҹҸбһ•бҹ’бһҠбһ»бҹҶбһңбһ¶бһ‘бҹ…бһҮбһ¶бһҖбҹ’бһҡбһ»бһҳбһҠбҹӮбһӣбһ бҹ…бһҗбһ¶ бһҖбҹ’бһҡбһ»бһҳбһҹбҹҶбһҺбһ»бҹҶбҹ” бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһҖбһ¶бһҡбһҖбҹ’бһҡбһ»бһҳ бһ“бһ·бһ„бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“ бһӮбһәбһҗбһ¶ бһўбҹ’бһңбһёбҹ—бһ”бһ¶бһ“бһҖбһҫбһҸбһЎбһҫбһ„бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһҳбһ·бһ“бһҳбҹӮбһ“бһҠбһјбһ…бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҠбһ№бһҖбһ“бһ¶бҹҶбҹ”

## бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҸбҹҶбһ”бһ“бҹӢбҹ– бһҳбҹүбһјбһҠбҹӮбһӣбһҖбҹ’бһҡбһ»бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸбһҸбһ“бҹ’бһҸбҹ’бһҡбһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҹбҹ’бһҸбһ¶бһ”бҹӢбһ“бҹ…бһ“бһёбһ бҹ’бһҹбҹҒбһҡбһёбһҷбҹүбһ¶ рҹҺ§

бһ…бҹҶбһ“бһјбһ“бһўбҹ’бһ“бһҖбһҹбҹ’бһҸбһ¶бһ”бҹӢбһ–бһ бһ»бһҳбһ»бһҒбһҮбһ¶бһҸбһ·бһ“бҹ…бһ“бһёбһ бҹ’бһҹбҹҒбһҡбһёбһҷбҹүбһ¶бһҳбһ¶бһ“бһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸбһҸбһ“бҹ’бһҸбҹ’бһҡбһёбһ•бҹ’бһҹбҹҒбһ„бҹ—бһӮбҹ’бһ“бһ¶бҹ” бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ‘бһ¶бһүбһҷбһҖбһ–бһё Spotify (бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹ’бһҡбһҖбһ–бһё [бһўбһҸбҹ’бһҗбһ”бһ‘бһ“бҹҒбҹҮ](https://towardsdatascience.com/country-wise-visual-analysis-of-music-taste-using-spotify-api-seaborn-in-python-77f5b749b421)) бһҷбһҫбһ„бһҳбһҖбһҳбһҫбһӣбһҸбһ“бҹ’бһҸбҹ’бһҡбһёбһҠбҹӮбһӣбһ–бҹҒбһүбһ“бһ·бһҷбһҳбһ“бҹ…бһ“бһёбһ бҹ’бһҹбҹҒбһҡбһёбһҷбҹүбһ¶бҹ” бһҲбһ»бһҸбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹҒбҹҮбһҡбһҪбһҳбһҳбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбҹҶбһ–бһёбһ–бһ·бһ“бҹ’бһ‘бһ» 'бһҖбһ¶бһҡвҖӢбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢвҖӢбһ…бһӣбһ“бһ¶', 'бһ—бһ¶бһ–бһҹбҹҶбһЎбҹҒбһ„вҖӢбһҹбҹ’бһ„бһ¶бһҸбҹӢ', бһҡбһҹбһҮбһ¶бһҸбһ·бһҹбһҳбҹ’бһӣбҹҒбһ„, 'бһ—бһ¶бһ–бһ“бһ·бһҷбһ¶бһҷ', бһ—бһ¶бһ–бһ–бҹҒбһүбһ“бһ·бһҷбһҳ бһ“бһ·бһ„бһҗбһ¶бһҳбһ–бһӣбһ“бҹғбһ”бһ‘бһ…бһҳбҹ’бһҡбҹҖбһ„бһ“бһ¶бһ“бһ¶бҹ” бһңбһ¶бһ“бһ№бһ„бһӮбһҪбһҡбһұбҹ’бһҷбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһӣбҹҶбһ“бһ¶бҹҶбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹҒбҹҮ!

бһҳбҹҒбһҡбҹҖбһ“бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһҠбҹ„бһҷ рҹҺ¶ бһҮбһ¶бһҳбһҪбһҷ [Jen Looper](https://www.twitter.com/jenlooper) бһҳбһ¶бһ“бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ–бһё [Rishit Dagli](https://rishit_dagli) бһ“бһ·бһ„ [Muhammad Sakib Khan Inan](https://twitter.com/Sakibinan)бҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ K-Means бһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҠбҹӮбһӣбһҮбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң [бһ•бҹ’бһҠбһҫбһҳбһ–бһёбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ iris](https://www.kaggle.com/bburns/iris-exploration-pca-k-means-and-gmm-clustering), бһҹбһүбҹ’бһүбһ¶бһҺбһ¶бһҳбһјбһӣ [бһҹбҹҖбһңбһ—бҹ…бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ](https://www.kaggle.com/prashant111/k-means-clustering-with-python) бһ“бһ·бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ NGO бһҹбҹ’бһңбҹӮбһ„бһҡбһҖ [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҹбһ¶бһҡбһ’бһ¶бһҸбһ»](https://www.kaggle.com/ankandash/pca-k-means-clustering-hierarchical-clustering)бҹ”
