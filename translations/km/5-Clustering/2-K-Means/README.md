бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖвҖӢбһ“бһ№бһ„бһҡбҹҖбһ“бһ–бһёбһҡбһ”бҹҖбһ”бһ”бһ„бҹ’бһҖбһҫбһҸвҖӢбһҖбҹ’бһҡбһ»бһҳбһІбҹ’бһҷбһ”бһ¶бһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Scikit-learn бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡбһӮбһҳбҹ’бһҡбһ”бҹӢбһҸбһ“бҹ’бһҸбҹ’бһҡбһёбһ“бһёбһ бҹ’бһҹбҹҒбһҡбһёбһҷбҹүбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһ“бһ¶бҹҶбһ…бһјбһӣбһҳбһ»бһ“бһ“бҹҒбҹҮбҹ” бһҷбһҫбһ„вҖӢбһ“бһ№бһ„бһҡбҹҖбһ”бһҡбһ¶бһ”бҹӢбһ–бһёбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғ K-Means бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢвҖӢбһҖбһ¶бһҡвҖӢбһ”бҹӮбһ„бһ…бҹӮбһҖвҖӢбһҖбҹ’бһҡбһ»бһҳбҹ” бһҹбһјбһҳбһ…бһ„бһ…бһ¶бҹҶбһҗбһ¶ бһҠбһјбһ…бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ“вҖӢбһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“ бһҳбһ¶бһ“бһҡбһ”бҹҖбһ”бһ…бҹ’бһҡбһҫбһ“вҖӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһҖбҹ’бһҡбһ»бһҳ бһ бһҫбһҷбһңбһ·бһ’бһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ“бһ№бһ„бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҷбһҫбһ„вҖӢбһ“бһ№бһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ K-Means бһ–бҹ’бһҡбҹ„бҹҮбһңбһ¶бһҮбһ¶вҖӢбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһ”бҹӮбһ„бһ…бҹӮбһҖбһҖбҹ’бһҡбһ»бһҳбһ’бһҳбҹ’бһҳбһҸбһ¶вҖӢбһ”бҹҶбһ•бһ»бһҸбҹ” бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ‘бҹ…!

бһ–бһ¶бһҖбҹ’бһҷбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҠбҹӮбһӣбһўбҹ’бһ“бһҖвҖӢбһ“бһ№бһ„бһҡбҹҖбһ“бһ–бһёбҹ–

- бһ–бһ·бһ“бҹ’бһ‘бһ» Silhouette
- бһңбһ·бһ’бһё Elbow
- Inertia
- бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶ (Variance)

## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

[K-Means Clustering](https://wikipedia.org/wiki/K-means_clustering) бһӮбһәвҖӢбһҮбһ¶вҖӢбһңбһ·бһ’бһёбһҳбһҪбһҷвҖӢбһ…бҹҒбһүбһ–бһёвҖӢбһңбһ·бһҹбҹҗбһҷвҖӢбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһҹбһүбҹ’бһүбһ¶бҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹӮбһ„бһ…бҹӮбһҖбһҖбҹ’бһҡбһ»бһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶ 'k' бһҖбҹ’бһҡбһ»бһҳбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҹбһ“бҹ’бһ‘бһҹбҹ’бһҹбһ“бҹҚбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ” бһҹбһ“бҹ’бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ“бһёбһҳбһҪбһҷбҹ—вҖӢбһ“бһ№бһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһҪбһүбһҠбһјбһҡвҖӢбһҠбһ¶бһҖбҹӢвҖӢбһҮбһ¶бһҖбҹ’бһҡбһ»бһҳвҖӢбһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷвҖӢбһҠбҹӮбһӣбһҮбһ·бһҸбһҖбһ“бҹ’бһӣбҹӮбһ„ 'mean' бһ¬вҖӢбһҖбһ¶бһ…бҹҶбһҺбһ»бһ…бһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣбһҡбһ”бһҹбҹӢбһҖбҹ’бһҡбһ»бһҳбҹ”

бһҖбҹ’бһҡбһ»бһҳбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ“бһ¶бҹҶбһҳбһ»бһҒбһҮбһ¶бһҡбһјбһ”бһ—бһ¶бһ–бһҮбһ¶ [бһңбҹүбһҡбҹүбһјбһҺбһјбһҷ](https://wikipedia.org/wiki/Voronoi_diagram) бһҠбҹӮбһӣбһҡбһҪбһҳбһҳбһ¶бһ“бһ…бҹҶбһҺбһ»бһ… (бһ¬ 'бһӮбҹ’бһҡбһ¶бһ”бҹӢ') бһ“бһ·бһ„бһҸбҹҶбһ”бһ“бҹӢвҖӢбһҠбҹӮбһӣбһҹбһҳбһҹбҹ’бһҡбһ”бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бҹ”

![voronoi diagram](../../../../translated_images/km/voronoi.1dc1613fb0439b95.webp)

> бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһҠбҹ„бһҷ [Jen Looper](https://twitter.com/jenlooper)

бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһҖбҹ’бһҡбһ»бһҳ K-Means [бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһёбһҮбҹҶбһ бһ¶бһ“](https://scikit-learn.org/stable/modules/clustering.html#k-means):

1. бһўбһ¶бһӣбҹӢбһ бҹ’бһӮбһјбһҡбһёбһ’бһҳа№ҢбһҮбҹ’бһҡбһҫбһҹбһ…бҹҶбһҺбһ»бһ…бһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣбһ…бҹҶбһ“бһҪбһ“ k бһҠбҹ„бһҷбһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҸбҹҶбһҺбһ¶бһ„бһ–бһёбһҜбһҖбһҹбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһңбһ¶бһӣбһјбһ”:
    1. бһңбһ¶бһҖбҹҶбһҺбһҸбҹӢбһҖбҹҶбһҺбһҸбҹӢбһҸбҹҶбһҺбһ¶бһ„бһ“бһёбһҳбһҪбһҷбҹ—бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ”бһҺбҹ’бһҸбҹ„бһҷбһҮбһ·бһҸбһ”бҹҶбһ•бһ»бһҸбҹ”
    2. бһңбһ¶бһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹҶбһҺбһ»бһ…бһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣбһҗбҹ’бһҳбһёбһҠбҹ„бһҷбһҷбһҖбһҸбһҳбҹ’бһӣбҹғбһҳбһ’бҹ’бһҷбһҳбһ“бҹғбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһ…бһ¶бһҸбҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ…бҹҶбһҺбһ»бһ…бһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣбһҳбһ»бһ“бҹ”
    3. бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһңбһ¶бһӮбһҺбһ“бһ¶бһ…бҹҶбһ“бһҪбһ“бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһ…бҹҶбһҺбһ»бһ…бһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣбһҗбҹ’бһҳбһёбһ“бһ·бһ„бһ…бҹҶбһҺбһ»бһ…бһ…бһ¶бһҹбҹӢ бһ бһҫбһҷбһ’бҹ’бһңбһҫбһЎбһҫбһ„бһңбһ·бһүбһҡбһ бһјбһҸбһҠбһӣбҹӢбһ…бҹҶбһҺбһ»бһ…бһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣбһҳбһ¶бһ“бһҹбҹ’бһҗбһ·бһҡбһ—бһ¶бһ–бҹ”

бһ…бҹҶбһҺбһ»бһ…бһҒбҹ’бһҹбҹ„бһҷбһҳбһҪбһҷбһ“бҹғбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ K-Means бһӮбһәбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһҸ 'k' бһҠбҹӮбһӣбһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһ…бҹҶбһҺбһ»бһ…бһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣбҹ” бһҮбһ¶бһҹбҹҶбһҺбһ¶бһ„бһңбһ·бһ’бһё 'elbow method' бһҮбһҪбһҷбһ”бҹүбһ¶бһ“бҹӢбһ”бҹ’бһҡбһҳбһ¶бһҺбһҸбһҳбҹ’бһӣбҹғбһҠбһҫбһҳбһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ 'k'бҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһңбһ·бһ’бһёбһ“бҹҒбҹҮбһҳбһҪбһҷбһ—бҹ’бһӣбһәбҹ”

## бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҳбһ»бһ“

бһўбҹ’бһ“бһҖвҖӢбһ“бһ№бһ„вҖӢбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ„бһ¶бһҡвҖӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„вҖӢбһҜбһҖбһҹбһ¶бһҡ [_notebook.ipynb_](https://github.com/microsoft/ML-For-Beginners/blob/main/5-Clustering/2-K-Means/notebook.ipynb) бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҠбҹӮбһӣбһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһ¶бһҡбһ“бһ¶бҹҶбһ…бһјбһӣ бһ“бһ·бһ„бһҹбһҳбҹ’бһўбһ¶бһҸбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹҶбһ”бһјбһ„бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһ“бҹ…бһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“бҹ”

## бһўбһ“бһ»бһңбһҸбҹ’бһҸ - бһҡбҹҖбһ”бһ…бҹҶ

бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһҳбһҫбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бһҳбҹ’бһҡбҹҖбһ„бһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбҹ”

1. бһ”бһ„бҹ’бһҖбһҫбһҸ boxplot бһҠбҹ„бһҷбһ бҹ… `boxplot()` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбһҪбһҡбһҲбһҡбһ“бһёбһҳбһҪбһҷбҹ—бҹ–

    plt.subplot(4,3,5)

    plt.subplot(4,3,6)
    sns.boxplot(x = 'loudness', data = df)
    
    plt.subplot(4,3,7)
    sns.boxplot(x = 'speechiness', data = df)
    
    plt.subplot(4,3,8)
    sns.boxplot(x = 'tempo', data = df)
    
    plt.subplot(4,3,9)
    sns.boxplot(x = 'time_signature', data = df)
    
    plt.subplot(4,3,10)
    sns.boxplot(x = 'danceability', data = df)
    
    plt.subplot(4,3,11)
    sns.boxplot(x = 'length', data = df)

    бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹҒбҹҮбһҳбһ¶бһ“бһҹбҹҶбһЎбҹҒбһ„бһҡбҹҶбһҒбһ¶бһ“бһҸбһ·бһ…бһҸбһҪбһ…бҹ– бһҠбҹ„бһҷбһҳбһҫбһӣбһҮбһ¶бһ”бһ‘бһ”бһ„бҹ’бһ бһ¶бһү boxplot бһ“бһёбһҳбһҪбһҷбҹ— бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһүбһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“бһҠбҹӮбһӣбһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбҹ”

бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһӣбһ»бһ”бһ…бҹ„бһӣбһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ–бһёбһҜбһҖбһҹбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ¶бһ“бһ№бһ„бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбһ·бһ…бһҸбһҪбһ…бһ”бһ“бҹ’бһҗбҹӮбһҳбҹ”

1. бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ бһҮбҹ’бһҡбһҫбһҹбһҮбһҪбһҡбһҲбһҡбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбҹҶбһ бһ¶бһҸбҹӢбһ”бҹӮбһ„бһ…бҹӮбһҖбһҖбҹ’бһҡбһ»бһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҮбҹ’бһҡбһҫбһҹбһҮбһҪбһҡбһҲбһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһ…бһ“бҹ’бһӣбҹ„бҹҮбһҹбҹ’бһҡбһҠбҹҖбһ„бһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„бһ•бҹ’бһ‘бҹҒбһҡбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣ `artist_top_genre` бһ‘бҹ…бһҮбһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бҹ–

    ```python
    from sklearn.preprocessing import LabelEncoder
    le = LabelEncoder()
    
    X = df.loc[:, ('artist_top_genre','popularity','danceability','acousticness','loudness','energy')]
    
    y = df['artist_top_genre']
    
    X['artist_top_genre'] = le.fit_transform(X['artist_top_genre'])
    
    y = le.transform(y)
    ```

1. бһҘбһЎбһјбһңбһ бһҫбһҷ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ…бҹҶбһ“бһҪбһ“бһҖбҹ’бһҡбһ»бһҳбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бҹӮбһ„бһ…бҹӮбһҖбҹ” бһўбҹ’бһ“бһҖбһҠбһ№бһ„бһҗбһ¶бһҳбһ¶бһ“бһ…бһҳбҹ’бһҡбҹҖбһ„ 3 бһҳбһ»бһҒбһ…бһҳбҹ’бһҡбҹҖбһ„бһҠбҹӮбһӣбһҷбһҫбһ„бһ”бһ¶бһ“бһҠбһҖбһ…бҹҒбһүбһ–бһёбһҜбһҖбһҹбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ 3бҹ–

    ```python
    from sklearn.cluster import KMeans
    
    nclusters = 3 
    seed = 0
    
    km = KMeans(n_clusters=nclusters, random_state=seed)
    km.fit(X)
    
    # бһҒбһ¶бһҸбһҗбһ¶бһ“бһўбҹҶбһ–бһёбһҖбҹ’бһҡбһ»бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹҶбһҺбһ»бһ…бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бһёбһҳбһҪбһҷбҹ—
    
    y_cluster_kmeans = km.predict(X)
    y_cluster_kmeans
    ```

бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһғбһҫбһүбһўбһ¶бһҡбҹҒбһҳбһҪбһҷбһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһҮбһ¶бһҳбһҪбһҷбһҖбҹ’бһҡбһ»бһҳбһҠбҹӮбһӣбһ”бһ¶бһ“бһ‘бһ¶бһҷбһ‘бҹ„бһӣ (0, 1, бһ¬ 2) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбһҪбһҡбһҠбҹҒбһҸбһ¶бһ”бҹ’бһҡбҹҒбһҳбһҳбһҪбһҷбҹ—бҹ”

1. бһ”бҹ’бһҡбһҫбһўбһ¶бһҡбҹҒбһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһӮбһҺбһ“бһ¶бһ–бһ·бһ“бҹ’бһ‘бһ» 'silhouette':

    ```python
    from sklearn import metrics
    score = metrics.silhouette_score(X, y_cluster_kmeans)
    score
    ```

## бһ–бһ·бһ“бҹ’бһ‘бһ» Silhouette

бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ–бһ·бһ“бҹ’бһ‘бһ» silhouette бһҠбҹӮбһӣбһҶбҹҖбһ„бһ‘бҹ…бһҖбһ¶бһ“бҹӢ 1бҹ” бһ–бһ·бһ“бҹ’бһ‘бһ»бһ“бҹҒбҹҮбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһ–бһё -1 бһ‘бҹ… 1 бһ бһҫбһҷ бһ”бһҫбһ–бһ·бһ“бҹ’бһ‘бһ»бһҮбһ¶бһ”бҹ’бһҡбһ¶бҹҶбһҳбһҪбһҷ бһӮбһәбһҖбҹ’бһҡбһ»бһҳбһҳбһ¶бһ“бһ—бһ¶бһ–бһҡбһ№бһ„бһҳбһ¶бҹҶ бһ“бһ·бһ„бһ”бҹҶбһ”бҹӮбһҖбһ…бҹ’бһ”бһ¶бһҹбҹӢбһ–бһёбһҖбҹ’бһҡбһ»бһҳбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ” бһҸбһҳбҹ’бһӣбҹғбһҮбһ·бһҸ 0 бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбҹ’бһҡбһ»бһҳбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһ…бҹ’бһҡбһ”бһјбһҖбһ…бҹ’бһҡбһ”бһӣбҹӢ бһҮбһҪбһҡбһҠбҹҒбһҸбһ¶бһ“бҹ…бһҮбһ·бһҸбһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“бһҹбһҳбҹ’бһҡбһ¶бһ…бҹӢбһҡбһ”бһҹбҹӢбһҖбҹ’бһҡбһ»бһҳбһҮбһ·бһҸбһҒбһ¶бһ„бҹ” [(бһ”бҹ’бһҡбһ—бһ–)](https://dzone.com/articles/kmeans-silhouette-score-explained-with-python-exam)

бһ–бһ·бһ“бҹ’бһ‘бһ»бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһӮбһә **.53**, бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһңбһ¶бһҮбһ¶бһҖбһҺбҹ’бһҸбһ¶бһӣбҹ” бһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶ бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһӮбҹ’бһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһӣбҹ’бһўбһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹӮбһ”бһ“бҹҒбҹҮбһ“бҹғбһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһҖбҹ’бһҡбһ»бһҳ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҷбһҫбһ„бһ“бһ№бһ„бһ”бһ“бҹ’бһҸбҹ”

### бһўбһ“бһ»бһңбһҸбҹ’бһҸ - бһҹбһ„бҹӢбһҳбҹүбһјбһҠбҹӮбһӣ

1. бһ“бһ¶бҹҶбһ…бһјбһӣ `KMeans` бһ бһҫбһҷбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһҖбҹ’бһҡбһ»бһҳбҹ”

    ```python
    from sklearn.cluster import KMeans
    wcss = []
    
    for i in range(1, 11):
        kmeans = KMeans(n_clusters = i, init = 'k-means++', random_state = 42)
        kmeans.fit(X)
        wcss.append(kmeans.inertia_)
    
    ```

    бһҳбһ¶бһ“бһ•бҹ’бһ“бҹӮбһҖбһҒбҹ’бһӣбҹҮбһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ–бһ“бҹ’бһҷбһӣбҹӢбҹ”

    > рҹҺ“ range: бһңбһ¶бһҮбһ¶бһ…бҹҶбһ“бһҪбһ“ iteration бһ“бҹғбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһҖбҹ’бһҡбһ»бһҳ

    > рҹҺ“ random_state: "бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһӣбҹҒбһҒбһ…бҹғбһҠбһ“бҹ’бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ…бҹҶбһҺбһ»бһ…бһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣбҹ”" [бһ”бҹ’бһҡбһ—бһ–](https://scikit-learn.org/stable/modules/generated/sklearn.cluster.KMeans.html#sklearn.cluster.KMeans)

    > рҹҺ“ WCSS: "бһ•бһӣбһ”бһјбһҖбһ…бҹҶбһ“бһҪбһ“бһҖбҹ’бһҡбһЎбһ¶бһ…бһҸбҹ’бһҡбһ„бҹ’бһӮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбҹ’бһҡбһ»бһҳ" бһңбһ¶бһҹбҹӢбһҒбһ»бһҹбһ”бһҡбһ·бһҳбһ¶бһҺбһҖбҹ’бһҡбһЎбһ¶бһ…бһҸбҹ’бһҡбһ„бҹ’бһӮбһҳбһ’бҹ’бһҷбһҳбһ“бҹғбһ…бҹҶбһҺбһ»бһ…бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбҹ’бһҡбһ»бһҳбһ’бҹҖбһ”бһҮбһ¶бһҳбһҪбһҷбһ…бҹҶбһҺбһ»бһ…бһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣбһҖбҹ’бһҡбһ»бһҳбҹ” [бһ”бҹ’бһҡбһ—бһ–](https://medium.com/@ODSC/unsupervised-learning-evaluating-clusters-bd47eed175ce).

    > рҹҺ“ Inertia: бһўбһ¶бһӣбҹӢбһ бҹ’бһӮбһјбһҡбһёбһ’бһҳбҹҚ K-Means бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҮбҹ’бһҡбһҫбһҹбһ…бҹҶбһҺбһ»бһ…бһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣбһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷ 'inertia' бһҠбҹӮбһӣбһҮбһ¶бһҖбһ¶бһҡбһңбһ¶бһҹбҹӢбһҗбһ¶ бһҖбҹ’бһҡбһ»бһҳбһҳбһ¶бһ“бһ—бһ¶бһ–бһҡбһ№бһ„бһҳбһ¶бҹҶбһҖбҹ’бһ“бһ»бһ„бһҒбҹ’бһӣбһҪбһ“бһҷбҹүбһ¶бһ„бһҺбһ¶бҹ” [бһ”бҹ’бһҡбһ—бһ–](https://scikit-learn.org/stable/modules/clustering.html) бһҸбһҳбҹ’бһӣбҹғбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бҹ…бһўбһҗбҹҒбһҡ wcss бһ“бҹ…бһҡбһ¶бһӣбҹӢбһ…бҹҶбһ“бһҪбһ“ iterationбҹ”

    > рҹҺ“ k-means++: бһ“бҹ… [Scikit-learn](https://scikit-learn.org/stable/modules/clustering.html#k-means) бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹҶбһҺбһ»бһ…бһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣ 'k-means++' бһҠбҹӮбһӣ "бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ…бҹҶбһҺбһ»бһ…бһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣбһІбҹ’бһҷбһ“бҹ…бһҶбҹ’бһ„бһ¶бһҷбһ–бһёбһӮбҹ’бһ“бһ¶бһҠбҹ„бһҷбһ‘бһјбһ‘бҹ… бһ”бһҺбҹ’бһҠбһ¶бһӣбһўбҹ„бһҷбһӣбһ‘бҹ’бһ’бһ•бһӣбһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһ„бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ…бҹғбһҠбһ“бҹ’бһҷбҹ”

### бһңбһ·бһ’бһё Elbow

бһҳбһ»бһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“бһҗбһ¶ бһҠбһјбһ…бһҮбһ¶бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ…бһҳбҹ’бһҡбҹҖбһ„ 3 бһ”бҹ’бһҡбһ—бҹҒбһ‘ бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҮбҹ’бһҡбһҫбһҹбһҖбҹ’бһҡбһ»бһҳ 3 бһҸбҹӮбһҸбһҫбһ–бһ·бһҸбһ‘бҹҒ?

1. бһ”бҹ’бһҡбһҫбһңбһ·бһ’бһё 'elbow method' бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бҹ”

    ```python
    plt.figure(figsize=(10,5))
    sns.lineplot(x=range(1, 11), y=wcss, marker='o', color='red')
    plt.title('Elbow')
    plt.xlabel('Number of clusters')
    plt.ylabel('WCSS')
    plt.show()
    ```

    бһ”бҹ’бһҡбһҫбһўбһҗбҹҒбһҡ `wcss` бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бҹ…бһҮбҹҶбһ бһ¶бһ“бһҳбһ»бһ“ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹ’бһҡбһ¶бһ•бһ·бһҖбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһёбһҠбҹӮбһӣбһҳбһ¶бһ“ 'bend' бһҖбҹ’бһ“бһ»бһ„ elbow бһҠбҹ„бһҷбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһ…бҹҶбһ“бһҪбһ“бһҖбҹ’бһҡбһ»бһҳвҖӢбһўбһ”бҹ’бһ”бһ”бһҡбһҳбһ¶бҹ” бһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶ бһңбһ¶ **бһҮбһ¶бһўбһ…бҹӢ** 3!

    ![elbow method](../../../../translated_images/km/elbow.72676169eed744ff.webp)

## бһўбһ“бһ»бһңбһҸбҹ’бһҸ - бһ”бһ„бҹ’бһ бһ¶бһүбһҖбҹ’бһҡбһ»бһҳ

1. бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбһҳбҹ’бһҖбһӣбҹӢбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ бһҖбҹҶбһҺбһҸбҹӢбһҖбҹ’бһҡбһ»бһҳ 3 бһҠбһ„ бһ бһҫбһҷбһ”бһ„бҹ’бһ бһ¶бһүбһҖбҹ’бһҡбһ»бһҳбһҮбһ¶бһҡбҹүбҹӮбһҸбһјбһ…бҹ–

    ```python
    from sklearn.cluster import KMeans
    kmeans = KMeans(n_clusters = 3)
    kmeans.fit(X)
    labels = kmeans.predict(X)
    plt.scatter(df['popularity'],df['danceability'],c = labels)
    plt.xlabel('popularity')
    plt.ylabel('danceability')
    plt.show()
    ```

1. бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ“бҹғбһҳбҹүбһјбһҠбҹӮбһӣбҹ–

    ```python
    labels = kmeans.labels_
    
    correct_labels = sum(y == labels)
    
    print("Result: %d out of %d samples were correctly labeled." % (correct_labels, y.size))
    
    print('Accuracy score: {0:0.2f}'. format(correct_labels/float(y.size)))
    ```

    бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ“бҹҒбҹҮбһҳбһ·бһ“бһӣбҹ’бһўбһ‘бҹҒ бһ бһҫбһҷбһҡбһјбһ”бһҡбһ¶бһ„бһҖбҹ’бһҡбһ»бһҳбһ•бҹ’бһҸбһӣбҹӢбһҹбһүбҹ’бһүбһ¶бһІбҹ’бһҷбһўбҹ’бһ“бһҖбһҠбһ№бһ„бһҳбһјбһӣбһ бҹҒбһҸбһ»бҹ”

    ![clusters](../../../../translated_images/km/clusters.b635354640d8e4fd.webp)

    бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹҒбҹҮбһҳбһ·бһ“вҖӢбһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢ бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҸбһ·бһ…бһ бһҫбһҷвҖӢбһҳбһ¶бһ“бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ…бҹ’бһҡбһҫбһ“бһҡбһңбһ¶бһ„бһҸбһҳбҹ’бһӣбҹғбһҮбһҪбһҡбһҲбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹӮбһ„бһ…бҹӮбһҖбһҖбҹ’бһҡбһ»бһҳбһІбҹ’бһҷбһ”бһ¶бһ“бһӣбҹ’бһўбҹ” бһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„ бһҖбҹ’бһҡбһ»бһҳбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҹбҹҗбһҖбҹ’бһҠбһ· influenced бһ¬ бһҳбһ¶бһ“бһ—бһ¶бһ–бһ…бҹ’бһҡбһ¶бһҹбһ…бҹ’бһҡбҹ„бһҳбһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҠбҹ„бһҷбһҖбҹ’бһҡбһ»бһҳбһ…бһҳбҹ’бһҡбҹҖбһ„ 3 бһҳбһ»бһҒбһҠбҹӮбһӣбһҷбһҫбһ„бһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ–бһёбһҳбһ»бһ“бҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶!

    бһҜбһҖбһҹбһ¶бһҡбһ“бҹғ Scikit-learn бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһүбһҗбһ¶ бһҳбҹүбһјбһҠбҹӮбһӣбһҠбһјбһ…бһ“бҹҒбҹҮ бһҠбҹӮбһӣбһҖбҹ’бһҡбһ»бһҳбһҳбһ·бһ“бһ”бһ¶бһ“бһ”бҹҶбһ”бҹӮбһҖбһ…бҹ’бһ”бһ¶бһҹбҹӢ бһҳбһ¶бһ“бһ”бһүбҹ’бһ бһ¶ 'variance'бҹ–

## бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶ (Variance)

бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҗбһ¶ "бһҮбһ¶бһҳбһ’бҹ’бһҷбһҳбһ“бҹғбһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ…бһҸбҹ’бһҡбһ„бҹ’бһӮбһ–бһёбһҳбһ’бҹ’бһҷбһҳ" [(бһ”бҹ’бһҡбһ—бһ–)](https://www.mathsisfun.com/data/standard-deviation.html)бҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘бһ“бҹғбһ”бһүбҹ’бһ бһ¶бһ”бҹӮбһ„бһ…бҹӮбһҖбһҖбҹ’бһҡбһ»бһҳбһ“бҹҒбҹҮ бһңбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ–бһёбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ…бҹҶбһ“бһҪбһ“бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бҹ’бһҡбһ¶бҹҶбһ„бһ…бҹҒбһүбһ–бһёбһҳбһ’бҹ’бһҷбһҳбһ…бҹ’бһҡбһҫбһ“бһ–бҹҒбһҖбҹ”

вң… бһ“бҹҒбҹҮбһҮбһ¶бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһӣбҹ’бһўбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбһ·бһҸбһ–бһёбһңбһ·бһ’бһёбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҖбҹӮбһҸбһҳбҹ’бһҡбһјбһңбһ”бһүбҹ’бһ бһ¶бһ“бҹҒбҹҮбҹ” бһ…бҹҶбһ–бҹ„бҹҮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһҪбһҷ? бһ”бҹ’бһҡбһҫбһҮбһҪбһҡбһҲбһҡбһ•бҹ’бһҹбҹҒбһ„? бһ”бҹ’бһҡбһҫбһўбһ¶бһӣбҹӢбһ бҹ’бһӮбһјбһҡбһёбһ’бһҳбҹҚбһ•бҹ’бһҹбҹҒбһ„? бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ•бҹ’бһҸбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚ: бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ [бһҖбһ¶бһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ](https://www.mygreatlearning.com/blog/learning-data-science-with-k-means-clustering/) бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһңбһ¶бһҳбһ¶бһ“бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһ“бһ·бһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҮбһҪбһҡбһҲбһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

> бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ '[бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбһҺбһ“бһ¶ variance](https://www.calculatorsoup.com/calculators/statistics/variance-calculator.php)' бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһӣбҹӢбһ–бһёбһӮбҹҶбһ“бһ·бһҸбһ“бҹҒбҹҮбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ”

бһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһҮбһ¶бһҳбһҪбһҷВ notebook бһ“бҹҒбҹҮ бһ”бҹүбһ·бһ“бһ”бҹ’бһҡбһҳбһ¶бһҺбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ“бҹғбһҳбҹүбһјбһҠбҹӮбһӣбһҖбһ¶бһ“бҹӢбһҸбҹӮбһ”бҹ’бһҡбһҹбһҫбһҡбһҠбҹ„бһҷбһҹбһҳбҹ’бһўбһ¶бһҸбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„бһҳбһ»бһ“ (бһҠбһјбһ…бһҮбһ¶бһӣбһ»бһ”бһ…бҹ„бһӣбһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“)? бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ‘бҹҶбһ„бһ“бҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҠбһӣбҹӢбһ‘бҹҶбһ„бһ“бҹӢбһ…бҹҶбһ–бҹ„бҹҮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҒбҹ’бһӣбҹҮбҹ—бҹ” бһҸбһҫбһҳбһ¶бһ“бһўбҹ’бһңбһёбһ‘бҹҖбһҸбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹ’бһҡбһ»бһҳбһӣбҹ’бһўбһҮбһ¶бһ„бһ“бҹҒбҹҮ?

бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ•бҹ’бһҸбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚбҹ– бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бһүбҹ’бһ…бһјбһӣбһҖбһ¶бһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” бһҳбһ¶бһ“бһҖбһјбһҠбһҠбҹӮбһӣбһ”бһ¶бһ“бһ–бҹ’бһҡбһёбһ“бһ»бһ…бһҖбҹ’бһ“бһ»бһ„ notebook бһҠбҹӮбһӣбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһҮбһҪбһҡбһҲбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ¶бһ“бһҳбҹүбһ¶бһҹбҹ’бһҹбһ¶бһҡбһјбһ”бһҡбһ¶бһ„бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„бҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбһҖбһғбһҫбһүбһҗбһ¶ бһҒбһҺбҹҲбһ–бһ·бһ“бҹ’бһ‘бһ» silhouette бһ”бһ“бҹ’бһҗбһҷ бһҖбҹҶбһҺбһҸбҹӢ 'kink' бһ“бҹ…бһҖбҹ’бһҡбһ¶бһ•бһ·бһҖ elbow бһ…бҹҒбһүбһҡбһ¶бһ”бһҹбҹ’бһҳбһҫбһЎбһҫбһ„бҹ” бһ“бҹҒбҹҮбһҠбҹ„бһҷбһҹбһ¶бһҡбһ–бҹҒбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһ—бҹҒбһ‘ бһҖбҹ’бһҡбһ»бһҳбһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһ–бһҒбһ»бһҹбһҒбһ¶бһҸбһҸбһ·бһ…бһ“бһ№бһ„бһҳбһ¶бһ“бһ‘бҹҶбһ„бһ“бҹӢбһ’бҹ’бһ„бһ“бҹӢбһҮбһ¶бһ„бҹ” бһўбһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһ”бһүбҹ’бһ бһ¶бһ“бҹҒбҹҮ [бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ](https://stats.stackexchange.com/questions/21222/are-mean-normalization-and-feature-scaling-needed-for-k-means-clustering/21226#21226)бҹ”
