- **бһ…бһ“бҹ’бһӣбҹ„бҹҮбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–** бһҖбҹҶбһҺбһҸбҹӢбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ“бҹҒбҹҮ бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҳбһ¶бһ“бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҠбһ¶бһ…бҹӢ бһ бһҫбһҷбһҳбһ¶бһ“бһ–бһёбһҡбһҮбһҳбҹ’бһҡбһҫбһҹ - **бһҶбҹ’бһңбҹҒбһ„** бһ“бһ·бһ„ **бһҹбҹ’бһҠбһ¶бҹҶ**бҹ” (бһ”бҹ’бһӣбһ»бһҖбһҖбһјбһҠ бҹў)

бҹЎ. бһҠбһҫбһҳбҹ’бһ”бһё initialize бһҹбһјбһҳбһңбһ¶бһҷбһҖбһјбһҠбһҠбһјбһ…бһҸбһ‘бҹ…бҹ–

    ```python
    env = gym.make("CartPole-v1")
    print(env.action_space)
    print(env.observation_space)
    print(env.action_space.sample())
    ```

бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҸбҹӢбһҹбһҳбҹ’бһҡбһҪбһӣбһҒбҹ’бһӣбһёбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбҹҶбһ бһ¶бһ“ бҹЎбҹ бҹ бҹ” бһ“бҹ…бһҮбҹҶбһ бһ¶бһ“бһ“бһёбһҳбһҪбһҷбҹ— бһҷбһҫбһ„бһ•бҹ’бһҸбһӣбҹӢбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢвҖӢбһҷбһҖбһўбһ“бһ»бһңбһҸбҹ’бһҸ - бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбһҳбҹ’бһҡбһҪбһӣбһ“бҹҒбҹҮбһҷбһҫбһ„бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҮбҹ’бһҡбһҫбһҹбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҠбҹ„бһҷбһ…бҹғбһҠбһ“бҹ’бһҷбһ–бһё `action_space`бҹ”

бҹЎ. бһҡбһҸбҹӢбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ бһ бһҫбһҷбһҳбһҫбһӣбһ•бһӣбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбҹ”

    вң… бһ…бһ„бһ…бһ¶бҹҶбһҗбһ¶ бһӣбҹ’бһўбһҮбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбһҸбҹӢбһҖбһјбһҠбһ“бҹҒбҹҮбһ“бҹ…бһ“бҹ…бһӣбһҫ Python local бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһӣбҹ’бһў! (бһ”бҹ’бһӣбһ»бһҖбһҖбһјбһҠ бҹЈ)

    ```python
    env.reset()
    
    for i in range(100):
       env.render()
       env.step(env.action_space.sample())
    env.close()
    ```

    бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһғбһҫбһүбһҡбһјбһ”бһ—бһ¶бһ–бһҠбһјбһ…бһ”бһ„бҹ’бһ бһ¶бһүбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

    ![non-balancing cartpole](../../../../8-Reinforcement/2-Gym/images/cartpole-nobalance.gif)

бҹЎ. бһ“бҹ…бһ–бҹҒбһӣбһҹбһҳбҹ’бһҡбһҪбһӣ бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бҹ” бһҸбһ¶бһҳбһ–бһ·бһҸ бһҳбһ»бһҒбһ„бһ¶бһҡ step бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҸбҹ’бһҡбһӣбһ”бҹӢбһ“бһјбһңбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ бһҳбһ»бһҒбһ„бһ¶бһҡбһ‘бһ‘бһҪбһӣбһҡбһ„бҹ’бһҹбҹҗбһҷбһҳбһҪбһҷ бһ“бһ·бһ„ бһ‘бһ„бҹ’бһңбһҫбһҸбһҪбһўбһҖбҹ’бһҹбһҡ done бһҠбҹӮбһӣбһҹбһүбҹ’бһүбһ¶бһҗбһ¶бһҸбһҫбһҸбҹ’бһҡбһјбһңбһ”бһ“бҹ’бһҸбһҹбһҳбҹ’бһҡбһҪбһӣбһ¬бһўбһҸбҹӢбҹ– (бһ”бҹ’бһӣбһ»бһҖбһҖбһјбһҠ 4)

    ```python
    env.reset()
    
    done = False
    while not done:
       env.render()
       obs, rew, done, info = env.step(env.action_space.sample())
       print(f"{obs} -> {rew}")
    env.close()
    ```

    - бһ‘бһёбһҸбһ¶бҹҶбһ„бһҡбһ”бһҹбҹӢ cart

    print(env.observation_space.low)

бһҳбһ¶бһ“бһңбһ·бһ’бһёбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ“бҹҒбҹҮбҹ–

- **бһ…бҹӮбһҖбһ…бҹҒбһүбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ“бҹ’бһҸбҹ’бһҡбһҖ**бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҷбһҫбһ„бһҠбһ№бһ„бһ…бһ“бҹ’бһӣбҹ„бҹҮбһҸбһҳбҹ’бһӣбҹғбһҳбһҪбһҷ бһҷбһҫбһ„бһўбһ¶бһ…бһ…бҹӮбһҖбһ…бһ“бҹ’бһӣбҹ„бҹҮбһ“бҹҒбҹҮбһ‘бҹ…бһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһҖбһ“бҹ’бһҸбҹ’бһҡбһҖ бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ”бһҳбҹ’бһӣбҹӮбһ„бһҸбһҳбҹ’бһӣбҹғбһ‘бҹ…бһҮбһ¶бһӣбҹҒбһҒбһҖбһ“бҹ’бһҸбҹ’бһҡбһҖбһҠбҹӮбһӣбһңбһ¶бһ…бһјбһӣбһҡбһҪбһҳбҹ” бһңбһ¶бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһңбһ·бһ’бһё numpy [`digitize`](https://numpy.org/doc/stable/reference/generated/numpy.digitize.html)бҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһҠбһ№бһ„бһ–бһёбһ‘бҹҶбһ бҹҶбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢ бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮбһңбһ¶бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһ…бҹҶбһ“бһҪбһ“бһҖбһ“бҹ’бһҸбҹ’бһҡбһҖбһҠбҹӮбһӣбһҷбһҫбһ„бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бҹ’бһңбһҫ digitizationбҹ”

вң… бһҷбһҫбһ„бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһ”бҹ’бһҡбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһҸбһ¶бһҳбһҹбһ“бһ·бһ‘бҹ’бһ‘бҹҒбһҹбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһҖбһҸбһҳбҹ’бһӣбҹғбһ‘бҹ…бһ…бһ“бҹ’бһӣбҹ„бҹҮбһҖбҹҶбһҺбһҸбҹӢ (бһ§. бһ–бһё -20 бһ‘бҹ… 20) бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ”бһҳбҹ’бһӣбҹӮбһ„бһҸбһҳбҹ’бһӣбҹғбһ бҹ’бһ“бһ№бһ„бһ‘бҹ…бһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһӮбһҸбҹӢбһҠбҹ„бһҷбһ”бһјбһҖбһ”бһ“бҹ’бһҗбһҷбҹ” бһңбһ¶бһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбһ§бһ”бһҸбҹ’бһҗбһҳбҹ’бһ—бһӣбҹ’бһўбһңбһ¶бһ„ бһ“бһ·бһ„ бһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҸбһ·бһ…бһӣбһҫбһ‘бҹҶбһ бҹҶбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– бһҠбҹ„бһҷбһ–бһ·бһҹбҹҒбһҹбһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҷбһҫбһ„бһҳбһ·бһ“бһҠбһ№бһ„бһ–бһёбһҹбһҳбһ¶бһҳбһ¶бһҸбҹ’бһҡбһ–бһ·бһҸбһ“бҹғбһҸбһҳбҹ’бһӣбҹғбһ…бһјбһӣбҹ” бһ§. бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бҹў бһҸбһҳбҹ’бһӣбҹғбһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһ“бһҪбһ“ бҹӨ бһҳбһ·бһ“бһҳбһ¶бһ“бһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“бһӣбһҫбһ¬бһҖбҹ’бһҡбҹ„бһҳ бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһ“бһ¶бҹҶбһІбҹ’бһҷбһҳбһ¶бһ“бһ…бҹҶбһ“бһҪбһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһўбһ“а©°аЁӨбҹ”

бһҖбҹ’бһ“бһ»бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһ‘бһёбһ–бһёбһҡбҹ” бһҠбһјбһ…бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһүбһҖбҹ’бһҡбҹ„бһҷбһ“бҹҒбҹҮ бһ‘бҹ„бҹҮбһ”бһёбһ…бһ“бҹ’бһӣбҹ„бҹҮбһӣбһҫ/бһҖбҹ’бһҡбҹ„бһҳбһҳбһ·бһ“бһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ бһҸбһҳбҹ’бһӣбҹғбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҖбҹҚбһғбһҫбһүбһ”бһ“бҹ’бһҗбһҷбһҖбҹ’бһ“бһ»бһ„бһ…бһ“бҹ’бһӣбҹ„бҹҮбһҖбҹҶбһҺбһҸбҹӢбһҳбһҪбһҷ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҒбҹ’бһ–бһҹбҹӢбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһҖбһ¶бһ…бҹ’бһҡбһҫбһ“бһҸбһ·бһ…бҹ”

бҹЎ. бһ“бҹҒбҹҮбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһ»бһҒбһ„бһ¶бһҡбһҺбһ¶бһҳбһҪбһҷ бһҠбҹӮбһӣбһ‘бһ‘бһҪбһӣбһҖбһ¶бһҡбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ–бһёбһҳбҹүбһјбһҠбҹӮбһӣбһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһ бһҫбһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶бһҹбҹҠбһ»бһҳбһҖбҹ’бһҡбһ»бһҳ бҹӨ бһ–бһёбһҸбһҳбҹ’бһӣбҹғШ№ШҜШҜбһӮбһҸбҹӢбҹ– (бһ”бҹ’бһӣбһ»бһҖбһҖбһјбһҠ бҹҰ)

    ```python
    def discretize(x):
        return tuple((x/np.array([0.25, 0.25, 0.01, 0.1])).astype(np.int))
    ```

бҹЎ. бһҷбһҫбһ„бһҳбһҖбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бһҳбһҪбһҷбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһҳбҹ’бһӣбҹӮбһ„бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ binбҹ– (бһ”бҹ’бһӣбһ»бһҖбһҖбһјбһҠ бҹ§)

    ```python
    def create_bins(i,num):
        return np.arange(num+1)*(i[1]-i[0])/num+i[0]
    
    print("Sample bins for interval (-5,5) with 10 bins\n",create_bins((-5,5),10))
    
    ints = [(-5,5),(-2,2),(-0.5,0.5),(-2,2)] # бһ”бҹ’бһҡбһҫбһҡбһҷбҹҲбһ–бҹҒбһӣбһ“бҹғбһҸбһҳбҹ’бһӣбҹғбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ“бһёбһҳбһҪбһҷбҹ—
    nbins = [20,20,10,10] # бһ…бҹҶбһ“бһҪбһ“бһ”бҹ’бһҡбһўбһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ“бһёбһҳбһҪбһҷбҹ—
    bins = [create_bins(ints[i],nbins[i]) for i in range(4)]
    
    def discretize_bins(x):
        return tuple(np.digitize(x[i],bins[i]) for i in range(4))
    ```

бҹЎ. бһҷбһҫбһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҡбһҸбҹӢбһҹбһҳбҹ’бһҡбһҪбһӣбһҒбҹ’бһӣбһё бһ бһҫбһҷбһҳбһҫбһӣбһҸбһҳбҹ’бһӣбҹғбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҠбһ¶бһ…бҹӢбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ‘бһ¶бҹҶбһ„ `discretize` бһ“бһ·бһ„ `discretize_bins` бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһҗбһ¶бһҸбһҫбһҳбһ¶бһ“бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ¬бһўбһҸбҹӢбҹ”

    вң… discretize_bins бһ”бһ„бҹ’бһ бһ¶бһүбһӣбҹҒбһҒ bin бһҠбҹӮбһӣбһӮбһ·бһҸбһ…бһ¶бһ”бҹӢбһ–бһё 0 бҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ…бҹҶбһ–бҹ„бҹҮбһҸбһҳбҹ’бһӣбҹғбһ“бҹ…бһ…бһ“бҹ’бһӣбҹ„бҹҮбһ“бҹ…бһҮбһ»бҹҶбһңбһ·бһү 0 бһңбһ¶бһҖбҹҸбһ”бһүбҹ’бһ…бҹҒбһүбһӣбҹҒбһҒбһ–бһёбһ…бһ“бҹ’бһӣбҹ„бҹҮбһҖбһҺбҹ’бһҸбһ¶бһӣбһ“бҹғбһ…бһ“бҹ’бһӣбҹ„бҹҮ (10)бҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ discretize бһҷбһҫбһ„бһҳбһ·бһ“бһӮбһ·бһҸбһ–бһёбһҮбһҪбһҡбһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһӣбһ‘бҹ’бһ’бһ•бһӣ бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһңбһ¶ бһўбһ¶бһ…бһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ бһ бҹ’бһ“бһ№бһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһё бһ бһҫбһҷ 0 бһҸбҹ’бһҡбһјбһңбһ“бһ№бһ„ 0бҹ” (бһ”бҹ’бһӣбһ»бһҖбһҖбһјбһҠ бҹЁ)

    ```python
    env.reset()
    
    done = False
    while not done:
       #env.render()
       obs, rew, done, info = env.step(env.action_space.sample())
       #бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ— discretize_bins(obs)
       print(discretize(obs))
    env.close()
    ```

    вң… бһўбһ¶бһ…бһҠбһҖбһ…бҹҒбһүбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҠбҹӮбһӣбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷ env.render бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһҳбһҫбһӣбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ” бһ”бһҫбһҳбһ·бһ“бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҸбҹӢбһңбһ¶бһҖбҹ’бһҡбҹ„бһҳбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһҖбҹ’бһҡбҹ„бһҷбһ”бһ“бҹ’бһ‘бһ”бҹӢ бһҠбҹӮбһӣбһҡбһ бҹҗбһҹбһҮбһ¶бһ„бҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһҮбһ¶бһўбһ¶бһҗбҹҚбһҖбҹҶбһ”бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣ Q-Learning бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бҹ”

## бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ Q-Table

бһ“бҹ…бһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“ бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһӮбһәбһҮбһ¶бһӮбһјбһӣбҹҒбһҒбһҹбһ¶бһҳбһүбҹ’бһү 0 бһҠбһӣбҹӢ 8 бһҠбһјбһ…бҹ’бһ“бҹ„бҹҮбһңбһ¶бһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһү Q-Table бһҮбһ¶бһҡбһјбһ”бһ—бһ¶бһ– numpy tensor бһ‘бҹҶбһ бҹҶ 8x8x2бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҷбһҫбһ„бһ”бҹ’бһҡбһҫ bin discretization бһ‘бҹҶбһ бҹҶбһңбҹүбһ·бһ…бһ‘бҹҗбһҡбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҖбҹҸбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ бһҫбһҷ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҷбһҫбһ„бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҮбһ¶бһўбһ¶бһҡбҹҒ 20x20x10x10x2 (2 бһӮбһәбһҮбһ¶бһҳбһ¶бһҸбҹ’бһҡбһҠбҹ’бһӢбһ¶бһ“бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– бһ бһҫбһҷбһ‘бҹҶбһ бҹҶбһҠбҹҶбһ”бһјбһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһ…бҹҶбһ“бһҪбһ“ bin бһҠбҹӮбһӣбһҷбһҫбһ„бһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹүбһ¶бһҡбҹүбһ¶бҹүбһҳбҹүбҹӮбһҸбҹ’бһҡбһ“бҹ…бһ…бһ“бҹ’бһӣбҹ„бҹҮбһҹбҹҶбһӮбһ¶бһӣбҹӢ)бҹ”

бһ”бҹүбһ»бһ“бҹ’бһҠбҹӮбһ–бҹҒбһӣбһҒбҹ’бһӣбҹҮ бһ”бһҡбһ·бһҳбһ¶бһҺбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҡбһ”бһҹбҹӢбһ…бһ“бҹ’бһӣбҹ„бҹҮбһҹбһҳбҹ’бһҡбһҪбһӣбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбҹ’бһӮбһ¶бһӣбҹӢбҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһҳбһ»бһҒбһ„бһ¶бһҡ `discretize` бһҷбһҫбһ„бһҳбһ·бһ“бһўбһ¶бһ…бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҳбһҪбһҷ бһ–бҹ’бһҡбҹ„бҹҮбһҸбһҳбҹ’бһӣбҹғбһҠбһҫбһҳбһҒбҹ’бһӣбҹҮбһҳбһ·бһ“бһҳбһ¶бһ“бһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“бҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҒбһ»бһҹбһ”бһ“бҹ’бһҸбһ·бһ… бһҠбҹ„бһҷбһ”бһ„бҹ’бһ бһ¶бһү Q-Table бһҮбһ¶бһҗбһҸбһҖбҹҶбһ“бһҸбҹӢбһҲбҹ’бһҳбҹ„бҹҮ (dictionary)бҹ”

бҹЎ. бһ”бҹ’бһҡбһҫбһӮбһј *(state,action)* бһҮбһ¶бһҡбһ№бһ„бһӮбһ“бҹ’бһӣбһ„ key бһҸбһ¶бһҳ(dictionary) бһ бһҫбһҷбһҸбһҳбҹ’бһӣбҹғбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһ бһ¶бһү Q-Table бһ“бҹ…бһ…бҹҶбһҺбһ»бһ…бһ“бҹ„бҹҮ (бһ”бҹ’бһӣбһ»бһҖбһҖбһјбһҠ бҹ©)

    ```python
    Q = {}
    actions = (0,1)
    
    def qvalues(state):
        return [Q.get((state,a),0) for a in actions]
    ```

    бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ бһҷбһҫбһ„бһҖбҹҶбһҺбһҸбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡ `qvalues()`, бһҠбҹӮбһӣбһҸбҹ’бһҡбһЎбһ”бҹӢбһ”бһүбҹ’бһҮбһёбһҸбһҳбҹ’бһӣбҹғбһҸбһ¶бһҡбһ¶бһ„ Q бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҘбһҡбһ·бһҷбһ¶бһ”бһҗбһҠбҹӮбһӣбһўбһ¶бһ…бһҖбһҫбһҸбһҳбһ¶бһ“бҹ” бһ”бһҫбһҳбһ·бһ“бһҳбһ¶бһ“ бһ…бҹҶбһҺбһ»бһ…бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Q-Table бһӮбһәбһҸбҹ’бһҡбһЎбһ”бҹӢбһ‘бҹ… 0 бһҮбһ¶бһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбҹ”

## бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ Q-Learning

бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһҷбһҫбһ„бһҸбҹ’бһҡбҹҖбһҳбһҒбҹ’бһӣбһҪбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҡбҹҖбһ“ Peter бһҡбһҖбҹ’бһҹбһ¶бһҹбһҳбһҸбһ»бһӣбҹ’бһҷ!

бҹЎ. бһҮбһ¶бһҳбһ»бһ“ бһҹбһјбһҳбһҖбҹҶбһҺбһҸбҹӢвҖӢбһ—бһ¶бһ–бһ•бҹ’бһҸбҹ„бһҸбһ…бһ·бһҸбҹ’бһҸбһҒбҹ’бһӣбҹҮбҹ—бҹ– (бһ”бҹ’бһӣбһ»бһҖбһҖбһјбһҠ бҹЎбҹ )

    ```python
    # бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҒбҹ’бһ–бһҹбҹӢ
    alpha = 0.3
    gamma = 0.9
    epsilon = 0.90
    ```

    бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ `alpha` бһӮбһәбһҮбһ¶ **бһўбһҸбҹ’бһҡбһ¶бһ“бҹғбһҖбһ¶бһҡбһҡбҹҖбһ“** бһҠбҹӮбһӣбһҖбҹҶбһҺбһҸбҹӢбһҗбһ¶бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҸбһҳбҹ’бһӣбҹғ Q-Table бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“ бһ“бҹ…бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҮбҹҶбһ бһ¶бһ“бҹ” бһ“бҹ…бһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“бһҷбһҫбһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһё 1 бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷ `alpha` бһ‘бҹ…бһҸбһҳбҹ’бһӣбҹғбһҸбһ·бһ…бҹ—бһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбҹ” бһ“бҹ…бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһҡбһҖбҹ’бһҹбһ¶бһңбһ¶бһҮбһ¶бһ–бһ·бһҹбҹҒбһҹ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһ–бһҹбһ¶бһҳбһүбҹ’бһү бһ бһҫбһҷбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғ alpha бһ–бһёбһҖбҹ’бһҡбҹ„бһҷбһ”бһ¶бһ“бҹ”

    `gamma` бһӮбһәбһҮбһ¶ **бһҸбҹӮбһңбһ¶бһҠбҹӮбһӣбһ”бһүбҹ’бһ…бһ»бҹҮбһҸбһҳбҹ’бһӣбҹғ** бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһ”бһүбҹ’бһүбһҫбһҹбҹҶбһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһҡбһ„бҹ’бһҹбҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһўбһ“бһ¶бһӮбһҸ бһҖбҹ’бһҡбҹӮбһ„бһӣбһҫбһҡбһ„бҹ’бһҹбҹҗбһҷбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бҹ”

    `epsilon` бһӮбһәбһҮбһ¶ **бһҳбһјбһӣбһ бҹҒбһҸбһ»бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ‘бҹҒбһҹбһҸ/бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ** бһҠбҹӮбһӣбһҖбҹҶбһҺбһҸбҹӢбһҗбһ¶бһҸбһҫбһҷбһҫбһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ (exploration) бһ¬бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһҷбһҫбһ„бһ”бһ¶бһ“бһҡбҹҖбһ“бһҳбһҖбһ бһҫбһҷ (exploitation)бҹ” бһҖбҹ’бһ“бһ»бһ„бһўбһ¶бһӣбҹӢбһ бҹ’бһӮбһҡбһёбһ’бһҳбһёбһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһ“бҹ… `epsilon` бһ—бһ¶бһӮбһҡбһҷбһ“бҹғбһҖбһҡбһҺбһё бһҷбһҫбһ„бһҮбҹ’бһҡбһҫбһҹбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҸбһ¶бһҳбһҸбһ¶бһҡбһ¶бһ„ Q бһ“бһ·бһ„бһ“бҹ…бһҖбһҡбһҺбһёбһҹбһӣбҹӢ бһҷбһҫбһ„бһҮбҹ’бһҡбһҫбһҹбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҠбҹ„бһҷбһ…бҹғбһҠбһ“бҹ’бһҷбҹ” бһңбһ¶бһ“бһ№бһ„бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҷбһҫбһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҸбҹҶбһ”бһ“бҹӢбһ“бҹ…бһӣбһҫбһӣбҹҶбһ бһҡбһҖбҹҶбһҺбһҸбҹӢбһҠбҹӮбһӣбһҳбһ·бһ“бһҠбҹӮбһӣбһ”бһ¶бһ“бһғбһҫбһүбһ–бһёбһҳбһ»бһ“бҹ”

    вң… бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһҹбһҳбһҸбһ»бһӣбҹ’бһҷ - бһҮбҹ’бһҡбһҫбһҹбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһ…бҹғбһҠбһ“бҹ’бһҷ (exploration) бһӮбһәбһўбһ¶бһ…бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһңбһ¶бһҷбһ…бҹғбһҠбһ“бҹ’бһҷбһ‘бҹ…бһҒбҹ’бһҷбһӣбҹӢбһҒбһ»бһҹ бһӮбҹ’бһ“бһ¶ бһ бһҫбһҷбһҒбҹ’бһҹбҹӮбһӮбһҪбһҡбһҸбҹӮбһҡбҹҖбһ“бһңбһ·бһ’бһёбһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбһҸбһ»бһӣбҹ’бһҷбһ–бһёбһҖбһ„бҹ’бһңбҹҮбһҒбһ¶бһҸбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ”

### бһ–бһ„бҹ’бһҡбһ№бһ„бһўбһ¶бһӣбҹӢбһ бҹ’бһӮбһҡбһёбһ’бһҳбһё

бһҷбһҫбһ„бһҖбҹҸбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһ–бһёбһҡбһҷбҹүбһ¶бһ„бһ…бҹҶбһҡбһјбһ„бһҳбһҖбһӣбһҫбһўбһ¶бһӣбҹӢбһ бҹ’бһӮбһҡбһёбһ’бһҳбһёбһ–бһёбһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“бҹ–

- **бһӮбһҺбһ“бһ¶бһ—бһ¶бһ–бһҳбһ’бҹ’бһҷбһҳбһҡбһ„бҹ’бһҹбҹҗбһҷбһ…бһ„бһҖбҹ’бһҡбһ„** бһўбҹҶбһ–бһёбһҡбҹүбһ¶бһ…бҹҶбһ“бһҪбһ“бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ“бҹғбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһҪбһӣбҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһңбһ¶бһҗбһ¶ бһ‘бҹ…бһҮбһ¶бһҡбһҝбһҷбҹ—бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢ бҹҘбҹ бҹ бҹ  бһҮбҹҶбһӣбҹ„бҹҮ бһ бһҫбһҷбһҷбһҫбһ„бһ“бһ№бһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҡбһ„бҹ’бһҹбҹҗбһҷбһҳбһ’бҹ’бһҷбһҳбһ…бһ„бһҖбҹ’бһҡбһ„бһӣбһҫбһҡбһҷбҹҲбһ–бҹҒбһӣбһ“бҹ„бҹҮбҹ” бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһ”бһҫбһҷбһҫбһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҮбһ¶бһ„ бҹЎбҹ©бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ» бһҷбһҫбһ„бһўбһ¶бһ…бһӮбһ·бһҸбһҗбһ¶бһ”бһүбҹ’бһ бһ¶бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһўбһ¶бһ…бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„бһӮбҹҒбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһҹбҹ’бһ“бһҫбҹ”

- **бһӮбһҺбһ“бһ¶бһ–бһ·бһ“бҹ’бһ‘бһ»бһҳбһ’бҹ’бһҷбһҳбһ…бһ„бһҖбҹ’бһҡбһ„бһўбһҸбһ·бһ”бһҡбһҳбһ¶** `Qmax`, бһ бһҫбһҷбһҷбһҫбһ„бһ“бһ№бһ„бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ Q-Table бһҠбҹӮбһӣбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһӣбһ‘бҹ’бһ’бһ•бһӣбһ“бҹ„бҹҮбҹ” бһ“бҹ…бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҡбһҸбҹӢбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҖбһҸбҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҗбһ¶ бһ–бҹҒбһӣбһҒбҹ’бһӣбҹҮбһӣбһ‘бҹ’бһ’бһ•бһӣбһҳбһ’бҹ’бһҷбһҳбһ…бһ„бһҖбҹ’бһҡбһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ’бҹ’бһӣбһ¶бһҖбҹӢбһ…бһ»бҹҮ бһ бһҫбһҷбһҷбһҫбһ„бһ…бһ„бҹӢбһҡбһҖбҹ’бһҹбһ¶бһҸбһҳбҹ’бһӣбҹғ Q-Table бһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ“бһ№бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҠбҹӮбһӣбһ”бһ¶бһ“бһҹбһ„бҹ’бһҖбҹҒбһҸбһғбһҫбһүбһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбҹ”

бҹЎ. бһҹбһјбһҳбһ”бҹ’бһҡбһҳбһјбһӣбһҡбһ„бҹ’бһҹбҹҗбһҷбһ…бһ„бһҖбҹ’бһҡбһ„бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹ…бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҮбҹҶбһ бһ¶бһ“бһҹбһҳбҹ’бһҡбһҪбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбҹүбһ·бһ…бһ‘бҹҗбһҡ `rewards` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӮбһјбһҡбһҮбһ¶бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ” (бһ”бҹ’бһӣбһ»бһҖбһҖбһјбһҠ бҹЎбҹЎ)

    ```python
    def probs(v,eps=1e-4):
        v = v-v.min()+eps
        v = v/v.sum()
        return v
    
    Qmax = 0
    cum_rewards = []
    rewards = []
    for epoch in range(100000):
        obs = env.reset()
        done = False
        cum_reward=0
        # == бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҹбһҳбһҪбһ„ ==
        while not done:
            s = discretize(obs)
            if random.random()<epsilon:
                # бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ - бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһўбһ¶бһҳбһҜбһҖбһҸбһ¶бһҳбһ”бҹ’бһҡбһјбһ”бһ¶бһ” Q-бһҸбһ¶бһҡбһ¶бһ„
                v = probs(np.array(qvalues(s)))
                a = random.choices(actions,weights=v)[0]
            else:
                # бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ - бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҠбҹ„бһҷбһ…бҹғбһҠбһ“бҹ’бһҷ
                a = np.random.randint(env.action_space.n)
    
            obs, rew, done, info = env.step(a)
            cum_reward+=rew
            ns = discretize(obs)
            Q[(s,a)] = (1 - alpha) * Q.get((s,a),0) + alpha * (rew + gamma * max(qvalues(ns)))
        cum_rewards.append(cum_reward)
        rewards.append(cum_reward)
        # == бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһӣбһ‘бҹ’бһ’бһ•бһӣбһ”бҹ’бһҡбһ…бһ¶бҹҶбһ–бҹҒбһӣбһ бһҫбһҷбһӮбһҺбһ“бһ¶бһҡбһ„бҹ’бһңбһ¶бһ“бҹӢбһҳбһ’бҹ’бһҷбһҳ ==
        if epoch%5000==0:
            print(f"{epoch}: {np.average(cum_rewards)}, alpha={alpha}, epsilon={epsilon}")
            if np.average(cum_rewards) > Qmax:
                Qmax = np.average(cum_rewards)
                Qbest = Q
            cum_rewards=[]
    ```

бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһӮбһ·бһҸбһҹбҹ’бһҡбһҳбҹғбһ–бһёбһӣбһ‘бҹ’бһ’бһ•бһӣбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ–

- **бһҮбһ·бһҸбһҠбһӣбҹӢбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„**бҹ” бһҷбһҫбһ„бһҮбһ·бһҸбһҠбһӣбҹӢбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“ бҹЎбҹ©бҹҘ бһҡбһ„бҹ’бһҹбҹҗбһҷбһ…бһ„бһҖбҹ’бһҡбһ„бһӣбһҫбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ· бҹЎбҹ бҹ + бһ•бҹ’бһҸбһ¶бһ…бҹӢбһҳбһ»бһҒбһ¬бһҷбһҫбһ„бһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ бһҫбһҷ! бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶бһӣбһ‘бҹ’бһ’бһ•бһӣбһҸбһ·бһ…бһҖбҹҸбһҠбҹ„бһҷ бһҷбһҫбһ„бһҳбһ·бһ“бһҠбһ№бһ„бһ‘бҹҒ бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮбһҷбһҫбһ„бһӮбһҺбһ“бһ¶бһҳбһ’бҹ’бһҷбһҳбһӣбһҫбһҖбһ¶бһҡбһҡбһҸбҹӢ бҹҘбҹ бҹ бҹ  бһ бһҫбһҷбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ· бҹЎбҹ бҹ  бҹ” бҹ”

- **бһҡбһ„бҹ’бһҹбҹҗбһҷбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ…бһ»бҹҮ**бҹ” бһ–бҹҒбһӣбһҒбҹ’бһӣбҹҮ бһҡбһ„бҹ’бһҹбҹҗбһҷбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ’бҹ’бһӣбһ¶бһҖбҹӢ бһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶бһҷбһҫбһ„бһўбһ¶бһ… "бһ”бҹҶбһ•бҹ’бһӣбһ¶бһү" бһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбҹҖбһ“бһ бһҫбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Q-Table бһҮбһ¶бһҳбһҪбһҷбһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҖбһ¶бһ“бҹӢбһҸбҹӮбһўбһ¶бһҖбҹ’бһҡбһҖбҹӢбҹ”

```python
obs = env.reset()
done = False
while not done:
   s = discretize(obs)
   env.render()
   v = probs(np.array(qvalues(s)))
   a = random.choices(actions,weights=v)[0]
   obs,_,done,_ = env.step(a)
env.close()
```

## бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“

бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһҷбһҫбһ„бһ”бһ¶бһ“бһҡбҹҖбһ“бһ–бһёбһңбһ·бһ’бһёбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһӣбҹ’бһў бһҠбҹ„бһҷбһ•бҹ’бһҸбһӣбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡбһҡбһ„бҹ’бһңбһ¶бһ“бҹӢбһҠбҹӮбһӣбһҖбҹҶбһҺбһҸбҹӢбһҡбһҠбҹ’бһӢбһ—бһ¶бһ–бһҠбҹӮбһӣбһ…бһ„бҹӢбһ”бһ¶бһ“ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ бҹ’бһӮбҹҒбһҳ бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһұбһҖбһ¶бһҹбһІбҹ’бһҷбһ–бһҪбһҖбһӮбҹҒбһ…бһ»бҹҮбһҹбһјбһҳбһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһҖбҹ’бһ“бһ»бһ„бһӣбҹҶбһ бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһҶбһ“бҹ’бһ‘бҹҲбҹ” бһҷбһҫбһ„бһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһўбһ¶бһӣбҹӢбһ бҹ’бһӮбһҡбһёбһ’бһҳ Q-Learning бһҠбҹ„бһҷбһҮбҹ„бһӮбһҮбҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҮбһ¶бһ”бһ“бҹ’бһҸ бһ“бһ·бһ„бһ”бһүбҹ’бһҲбһҡ бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһ”бһүбҹ’бһҲбһҡбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”

бһңбһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢбһ•бһ„бһҠбҹӮбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ·бһҖбҹ’бһҹбһ¶ бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҠбҹӮбһӣбһҡбһҠбҹ’бһӢбһ—бһ¶бһ–бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҖбҹҸбһҮбһ¶бһ”бһ“бҹ’бһҸ бһ“бһ·бһ„бһ–бҹҒбһӣбһҠбҹӮбһӣбһӣбҹҶбһ бһҹбҹҶбһЎбҹҒбһ„бһҹбһ„бҹ’бһҖбһҸбҹӢбһҳбһ¶бһ“бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҮбһ¶бһ„бһ“бҹҒбҹҮ бһҠбһјбһ…бһҮбһ¶бһҡбһјбһ”бһ—бһ¶бһ–бһ–бһёбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбһ бҹ’бһӮбҹҒбһҳ Atariбҹ” бһҖбҹ’бһ“бһ»бһ„бһ”бһүбҹ’бһ бһ¶бһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮ бһҷбһҫбһ„бһҸбҹӮбһ„бһҸбҹӮбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҒбҹ’бһӣбһ¶бҹҶбһ„бһҮбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһӮбҹ’бһҡбһҝбһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“ бһҠбһјбһ…бһҮбһ¶ бһ”бһҺбҹ’бһҸбһ¶бһүбһҹбһ¶бһҡбһ’бһ¶бһҸбһ»бһҖбҹ„бһҹбһ·бһҖбһ¶ neural networks бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһӣбҹ’бһўбҹ” бһ”бһүбҹ’бһ бһ¶бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҮбһ¶бһ„бһ“бҹҒбҹҮбһҮбһ¶бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ“бҹғбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶ AI бһӣбҹҶбһҠбһ¶бһ”бҹӢбһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„бһҳбһҪбһҷбһҠбҹӮбһӣбһҷбһҫбһ„бһ“бһ№бһ„бһ”бһҫбһҖбһҹбһ·бһҖбҹ’бһҹбһ¶бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”
