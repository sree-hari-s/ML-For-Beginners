# бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҢбҹӮбһӣбһ•бҹ’бһҹбһ¶бһҡвҖӢбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Scikit-learn: бһ•бҹ’бһҹбһ¶бһҡвҖӢбһ”бһҪбһ“вҖӢбһҡбһ”бҹҖбһ”

## бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢвҖӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳ

бһҖбһ¶бһҡвҖӢбһ•бҹ’бһҹбһ¶бһҡвҖӢбһҡбҹҖбһ”вҖӢбһҡбҹүбҹҒвҖӢбһҢбһёвҖӢбһҹбҹ’бһҷбһ»бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһҷбһҫбһ„бһ…бһ„бҹӢбһ‘бһ¶бһҷбһ‘бҹ„бһӣвҖӢбһҸбһҳбҹ’бһӣбҹғ **бһ…бҹҶбһ“бһҪбһ“вҖӢбһҹбҹҠбһёбһ•бҹҚ** (бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҸбһҳбҹ’бһӣбҹғбһ•бҹ’бһ‘бҹҮ, бһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ– бһ¬ бһӣбһҖбҹӢбһ…бҹҒбһү)бҹ” бһңбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҡбһ”бһ¶бһҡбһ—бһ¶бһ–вҖӢбһҸбһҳбҹ’бһҡбһ¶бһҹбҹӢвҖӢбһҳбһҪбһҷвҖӢбһҠбҹӮбһӣвҖӢбһҸбҹҶбһҺбһ¶бһ„бһІбҹ’бһҷбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„вҖӢбһҡбһңбһ¶бһ„вҖӢбһӣбһҖбҹ’бһҒбһҺбҹҲбһ”бһүбҹ’бһ…бһјбһӣ бһ“бһ·бһ„бһӣбһ‘бҹ’бһ’бһ•бһӣвҖӢбһ”бһ¶бһ“вҖӢбһӣбҹ’бһўвҖӢбһ”бҹҶбһ•бһ»бһҸбҹ”

вҖӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“вҖӢбһ“бҹҒбҹҮвҖӢбһҷбһҫбһ„бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһӮбҹҶбһ“бһ·бһҸбһҳбһ»бһ“бһ–бҹҒбһӣбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ•бҹ’бһҹбһ¶бһҡвҖӢбһӣбҹҶбһҠбһ¶бһ”бҹӢбһҒбҹ’бһ–бһҹбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бҹҖбһҸбҹ”

### бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһ…бҹҶбһ–бҹ„бҹҮбһ–бҹҒбһӣбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбһ№бһ„бһҗбһ¶бһ•бҹ’бһҹбһ¶бһҡбһҡбҹҖбһ”бһҡбһ¶бһ”бҹӢбһӮбһәбһҮбһ¶бһўбҹ’бһңбһёбһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбҹҶбһҡбһјбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһҳбһјбһӣбһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚбһҸбһҳбҹ’бһӣбҹғбһ•бҹ’бһӣбҹӮбһ•бҹ’бһўбҹӮбһҒбҹ„бһ…бһҠбҹӮбһӣбһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбҹ” бһўбҹ’бһ“бһҖбһҖбҹҸбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҳбһҫбһӣбһғбһҫбһүбһңбһ¶бһҸбһ¶бһҳбһҡбһҷбҹҲ Matplotlib бһ•бһ„бһҠбҹӮбһҡбҹ”

бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҮбҹ’бһҡбҹҖбһҸбһ…бһјбһӣбһҮбҹ’бһҡбһ¶бһӣбһҮбҹ’бһҡбҹ…бһ…бҹҶбһ–бҹ„бҹҮбһ•бҹ’бһҹбһ¶бһҡбһҡбҹҖбһ”бһҡбһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ MLбҹ” бһҒбһҺбҹҲбһ–бҹҒбһӣбһҠбҹӮбһӣбһҖбһ¶бһҡбһҳбһҫбһӣбһғбһҫбһүбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ–бҹ’бһҡбҹҮбһҡбһ¶бһҮбһўбҹҶбһҺбһ¶бһ…бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһҡбһ”бһҹбҹӢбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҳбһ¶бһ“бһ—бһ¶бһ–бһ–бһё _бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҢбҹӮбһӣ_бҹ” бһҳбҹүбһјбһҢбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһҶбһ¶бһ”бҹӢбһҷбһӣбҹӢбһ–бһёбһҖбһ¶бһҡбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҡбһңбһ¶бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ‘бһ¶бһҷбһ‘бҹ„бһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹ’бһҳбһё бһҠбҹӮбһӣбһҳбҹүбһјбһҢбҹӮбһӣбһҳбһ·бһ“бһ”бһ¶бһ“бһғбһҫбһүбһ–бһёбһҳбһ»бһ“бҹ”

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ•бҹ’бһҹбһ¶бһҡбһҡбҹҖбһ”бһҡбһ¶бһ”бҹӢбһ–бһёбһҡ: _бһ•бҹ’бһҹбһ¶бһҡбһҡбҹҖбһ”бһҡбһ¶бһ”бҹӢбһҡбһ¶бһ„бһҸбҹӮбһҳбһҪбһҷбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“_ бһ“бһ·бһ„ _бһ•бҹ’бһҹбһ¶бһҡбһҡбҹҖбһ”бһҡбһ¶бһ”бҹӢбһ–бһ бһ»бһ”бһјбһЎбһёбһҺбһјбһҳбҹ’бһҷбҹӮбһӣ_ бһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷбһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҒбҹ’бһӣбҹҮбһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ” бһҳбҹүбһјбһҢбҹӮбһӣбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ“бһ№бһ„бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҷбһҫбһ„бһ‘бһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ•бҹ’бһӣбҹӮбһ•бҹ’бһўбҹӮбһҒбҹ„бһ…бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһүбҹ’бһ…бһјбһӣбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

> бһӣбһҫбһҖбһ‘бһјбһӣбһ¶бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бҹҒбҹҮ бһҷбһҫбһ„бһӮбһ·бһҸбһҗбһ¶ бһҳбһ¶бһ“бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶вҖӢбһҸбһ·бһ…бһҸбһҪбһ… бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ‘бһ¶бһҳбһ‘бһ¶бһҡбһұбҹ’бһҷбһҳбһ¶бһ“бһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ·бһҹбҹ’бһҹбһҠбҹӮбһӣбһҳбһҖбһ–бһёбһңбһ·бһҹбҹҗбһҷбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҹбһјбһҳбһ”бҹ’бһҡбһҷбҹҗбһҸбҹ’бһ“бһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢ рҹ§® бһҖбһ¶бһҡбһ бҹ…бһ…бҹҒбһү бһңбһ·бһ…бһ¶бһҡбһҺбһ¶бһ’бһ·бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһҹбҹ’бһӣбһ¶бһҖбһ”бһ¶бһ…бҹӢбһ”бһ·бһ‘ бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ”бҹӢбһҡбҹҶбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбһҪбһҷбһІбҹ’бһҷбһҷбһӣбҹӢбҹ”

### бһӣбһҖбҹ’бһҒбһҺбҹ’бһҢбһҳбһ»бһ“бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ

бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһҹбҹ’бһӮбһ¶бһӣбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһӣбҹӮбһ•бҹ’бһўбҹӮбһҒбҹ„бһ…бһҠбҹӮбһӣбһҷбһҫбһ„бһҖбҹҶбһ–бһ»бһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҘбһЎбһјбһңбһ“бҹҒбҹҮбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҖбһ”бһ¶бһ“бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖбһҡбһҪбһ…бһ бһҫбһҷ бһ“бһ·бһ„бһ”бһ¶бһ“бһҹбһҳбҹ’бһўбһ¶бһҸбһҡбһҪбһ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ _notebook.ipynb_ бһ“бҹғбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбҹ” бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ бһӮбһәбһ”бһ„бҹ’бһ бһ¶бһүбһҸбһҳбҹ’бһӣбҹғбһ•бҹ’бһӣбҹӮбһ•бҹ’бһўбҹӮбһҒбҹ„бһ…бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҳбһҪбһҷбһҖбҹҶбһ”бҹүбһ·бһӣбһҖбҹ’бһ“бһ»бһ„бһҸбһ¶бһҡбһ¶бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹ’бһҳбһёбҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҸбҹӢбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҸбҹҶбһҺбһ¶бһ„бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҺбҹ’бһҸбһ»бһҡбһҡбһ”бһҹбҹӢ Visual Studio Codeбҹ”

### бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶ

бһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҶбһӣбһ№бһҖ бһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҪбһҡбһ…бҹҶбһЎбһҫбһҷбһ–бһёбһңбһ¶бҹ”

- бһҸбһҫбһ–бҹҒбһӣбһҺбһ¶вҖӢбһҮбһ¶вҖӢбһ–бҹҒбһӣбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ‘бһ·бһүбһ•бҹ’бһӣбҹӮбһ•бҹ’бһўбҹӮбһҒбҹ„бһ…?
- бһҸбһҫбһҸбһҳбҹ’бһӣбҹғбһўбҹ’бһңбһёбһҠбҹӮбһӣбһҒбҹ’бһүбһ»бҹҶбһўбһ¶бһ…бһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбһ…бҹҶбһ–бҹ„бҹҮбһ”бҹ’бһҡбһўбһ”бҹӢбһ•бҹ’бһӣбҹӮбһ•бҹ’бһўбҹӮбһҒбҹ„бһ…бһҸбһјбһ…?
- бһҸбһҫбһҒбҹ’бһүбһ»бҹҶбһӮбһҪбһҡбһ‘бһ·бһүбһңбһ¶бһҖбҹ’бһ“бһ»бһ„бһ’бһ»бһ„бһҖбҹҚбһҳбһҪбһҷбһ–бһ¶бһҖбҹӢбһҖбһҺбҹ’бһҸбһ¶бһӣбһҖбҹҶбһ”бҹүбһ·бһӣ бһ¬бһҠбҹ„бһҷбһ”бҹ’бһҡбһўбһ”бҹӢ 1 1/9 бһҖбҹҶбһ”бҹүбһ·бһӣ?
бһўбһҫбһҷ! бһҷбһҫбһ„бһҹбһјбһҳбһӣбһҸбҹӢбһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹҒбҹҮбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ”

бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“ бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҸбһ¶бһҡбһ¶бһ„бһҸбҹҶбһҺбһ¶бһ„ Pandas бһ бһҫбһҷбһ•бҹ’бһ‘бһ»бһҖбһңбһ¶бһ“бҹ…бһҮбһ»бҹҶбһңбһ·бһүбһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ“бҹғбһҜбһҖбһҹбһ¶бһҡбһҠбһҫбһҳ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһҸбһҳбҹ’бһҡбһ№бһҳбһҸбһҳбҹ’бһӣбҹғбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҳбһҪбһҷбһҖбҹҶбһ”бҹүбһ·бһӣбҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҳбһјбһӣбһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһ”бҹ’бһҡбһ бҹӮбһӣ бҹӨбҹ бҹ бһ…бҹҶбһҺбһ»бһ…бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ бһҫбһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҒбҹӮбһҡбһҠбһјбһңбһҹбҹ’бһӣбһ№бһҖбһҲбһҫбһҮбҹ’бһҡбһ»бҹҮбһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”

бһҳбһҫбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖбһҡбһҪбһ…бһҮбһ¶бһҳбһ»бһ“бһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҡбһҪбһҳбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбҹ” бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖбһҡбһҪбһ…бһ бһҫбһҷ бһ“бһ·бһ„бһ”бһ¶бһ“бһ…бһ»бҹҮбһҸбҹҶбһҺбһ¶бһ„бһҸбһ¶бһҳбһҡбһҷбҹҲ scatterplot бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбһ¶бһҳбһҒбҹӮбҹ” бһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһҷбһҫбһ„бһўбһ¶бһ…бһҷбһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһӣбһҖбҹ’бһҒбһҺбҹҲбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹ„бһҷбһҹбһҳбҹ’бһўбһ¶бһҸбһңбһ¶бһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бҹҖбһҸбҹ”

## бһҹбһ“бҹ’бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ•бҹ’бһҹбһ¶бһҡвҖӢбһҡбҹҖбһ”бһҡбһ¶бһ”бҹӢбһҡбһ¶бһ„бһҸбҹӮбһҳбһҪбһҷ

бһҠбһјбһ…бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ“бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бҹЎ бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ“бҹғбһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳ Linear Regression бһӮбһәбһҠбһҫбһҳбҹ’бһ”бһёбһӮбһјбһҹбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҳбһҪбһҷбһҠбҹӮбһӣ:

- **бһ”бһ„бҹ’бһ бһ¶бһүбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһўбһҗбҹҒбһҡ**бҹ” бһ”бһ„бҹ’бһ бһ¶бһүбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҡбһңбһ¶бһ„бһўбһҗбҹҒбһҡ
- **бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ‘бһ¶бһҷбһ‘бҹ„бһӣ**бҹ” бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ‘бһ¶бһҷбһ‘бҹ„бһӣбһҷбҹүбһ¶бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҗбһ¶бһ…бҹҶбһҺбһ»бһ…бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹ’бһҳбһёбһ“бһ№бһ„бһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһҺбһ¶бһҮбһ·бһҸбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ“бҹ„бҹҮбҹ”

бһҹбһ·бһҖбҹ’бһҹбһ¶бһҡбҹҖбһ”бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһҳбһ’бҹ’бһҷбҹ„бһ”бһ¶бһҷ **Least-Squares Regression** бһҮбһ¶бһ‘бһјбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбһјбһҹбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ“бҹҒбҹҮбҹ” бһ–бһ¶бһҖбҹ’бһҷ "Least-Squares" бһҳбһ¶бһ“бһ“бҹҗбһҷбһ–бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҖбҹҶбһ бһ»бһҹбһҹбһҡбһ»бһ”бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһјбһҢбҹӮбһӣбһҷбһҫбһ„бҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹҶбһҺбһ»бһ…бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ¶бһӣбҹӢбһӮбһәбһҷбһҫбһ„бһңбһ¶бһҹбҹӢбһ…бһҳбҹ’бһ„бһ¶бһҷбһҲбһҡ (бһ бҹ…бһҗбһ¶ residual) бһҡбһңбһ¶бһ„бһ…бҹҶбһҺбһ»бһ…бһ–бһ·бһҸ бһ“бһ·бһ„бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ•бҹ’бһҹбһ¶бһҡбҹ” 

бһҷбһҫбһ„бһҹбһҳбҹ’бһҡбһҪбһӣбһ…бһҳбҹ’бһ„бһ¶бһҷбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҠбһӣбҹӢбһҖбҹҶбһҡбһ·бһҸбһҖбҹ’бһҡбҹ„бһҳбһ–бһёбһ бҹҒбһҸбһ»бһ•бһӣбһ–бһёбһҡбһ…бһҳбҹ’бһ”бһ„бҹ–

1. **бһ‘бҹҶбһ бҹҶбһӣбһҫбһ‘бһ·бһҹбһҠбҹ…**бҹ– бһҷбһҫбһ„бһ…бһ„бҹӢбһҠбһ¶бһҖбҹӢбһҸбһҳбҹ’бһӣбҹғбһҖбҹҶбһ бһ»бһҹ -5 бһ“бһ·бһ„ +5 бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ” бһҖбһ¶бһҡбһ”бһ„бҹ’бһҡбһҪбһ…бһӮбһ»бһҺвҖӢвҖӢбһ”бһ„бҹ’бһҖбһҫбһҸбһўбҹ„бһҷбһҸбһҳбҹ’бһӣбҹғбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҮбһ¶бһҹбҹ’бһҡбһ”бҹ” 

2. **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбҹҶбһҺбһ»бҹҶбһ“бҹғбһ…бҹҶбһҺбһ»бһ…бһҖбҹ’бһҡбҹ…**бҹ– бһҖбһ¶бһҡбһ”бһ„бҹ’бһҡбһҪбһ…бһӮбһ»бһҺбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҖбҹҶбһ бһ»бһҹбһ’бҹҶбҹ—бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ‘бҹҶбһ„бһ“бҹӢбһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„ бһ бһҫбһҷбһ”бһҺбҹ’бһҸбһ¶бһӣбһІбҹ’бһҷбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ“бҹ…бһҮбһ·бһҸбһ…бҹҶбһҺбһ»бһ…бһ…бһҳбҹ’бһ„бһ¶бһҷбҹ”

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҷбһҫбһ„бһ”бһјбһҖбһҸбһҳбҹ’бһӣбҹғбһҖбҹ’бһҡбҹ„бһҳбһӮбһ»бһҺбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ‘бһ¶бҹҶбһ„бһЎбһ¶бһҷбһӮбҹ’бһ“бһ¶бҹ” бһӮбҹ„бһӣбһҠбҹ…бһӮбһәбһҡбһҖбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҳбһҪбһҷбһҠбҹӮбһӣбһ”бһјбһҖбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһ“бҹҒбҹҮбһҸбһ·бһ…бһҮбһ¶бһ„бһӮбҹҒ (бһҸбһҳбҹ’бһӣбҹғбһҸбһ·бһ…бһ”бҹҶбһ•бһ»бһҸ)вҖ”бһ бҹҒбһҸбһ»бһ“бҹҒбҹҮбһ бҹ…бһҗбһ¶ "Least-Squares"бҹ”

## бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҹбһҳбһ¶бһҳбһ¶бһҸбҹ’бһҡ (Correlation)

бһ–бһ¶бһҖбҹ’бһҷбһҳбһҪбһҷбһ‘бҹҖбһҸбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҷбһӣбҹӢбһӮбһә **бһҹбһҳбһ¶бһҳбһ¶бһҸбҹ’бһҡбһҮбһ¶бһӮбһј** бһҡбһңбһ¶бһ„бһўбһҗбҹҒбһҡ X бһ“бһ·бһ„ Yбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҫбһӣбһҹбһҳбһ¶бһҳбһ¶бһҸбҹ’бһҡбһ“бҹҒбҹҮбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҡбһ бҹҗбһҹбһҸбһ¶бһҳбһҖбһ¶бһҡбһӮбһјбһҹ scatterplotбҹ” бһ”бҹ’бһӣбһ¶бһҸбҹӢбһҠбҹӮбһӣбһҳбһ¶бһ“бһ…бҹҶбһҺбһ»бһ…бһҡбһ¶бһҷбһ–бҹ’бһҡбһҳбһ‘бҹ’бһҡбһ»бһ„бһҖбҹ’бһ“бһ»бһ„бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“бһҹбһҳбһ¶бһҳбһ¶бһҸбҹ’бһҡбһҒбҹ’бһ–бһҹбҹӢ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ”бҹ’бһӣбһ¶бһҸбҹӢбһҠбҹӮбһӣбһ…бҹҶбһҺбһ»бһ…бһҡбһ¶бһҷбһЎбһҫбһ„бһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„ X бһ“бһ·бһ„ Y бһӮбһәбһҳбһ¶бһ“бһҹбһҳбһ¶бһҳбһ¶бһҸбҹ’бһҡбһҸбһ·бһ…бҹ”

бһҳбҹүбһјбһҢбҹӮбһӣбһ•бҹ’бһҹбһ¶бһҡ Linear Regression бһӣбҹ’бһўбһӮбһәбһ”бһ¶бһ“бһҹбһҳбһ¶бһҳбһ¶бһҸбҹ’бһҡбһҒбҹ’бһ–бһҹбҹӢ (бһҮбһ·бһҸ 1 бһҮбһ¶бһ„ 0) бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ Least-Squares Regression бһҮбһ¶бһҳбһҪбһҷбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ•бҹ’бһҹбһ¶бһҡбҹ”

вң… бһҡбһҸбҹӢбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҮбһ¶бһҳбһҪбһҷбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҡбһҪбһ…бһҳбһҫбһӣ scatterplot бһҹбһҳбһ¶бһҳбһ¶бһҸбҹ’бһҡбһ–бһё Month бһ‘бҹ… Priceбҹ” бһҸбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢ Month бһ‘бҹ… Price бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӣбһҖбҹӢбһ•бҹ’бһӣбҹӮбһ•бҹ’бһўбҹӮбһҒбҹ„бһ… бһҷбһӣбҹӢбһҗбһ¶бһҹбһҳбһ¶бһҳбһ¶бһҸбҹ’бһҡбһҒбҹ’бһ–бһҹбҹӢбһ¬бһ‘бһ¶бһ” бһҸбһ¶бһҳбһҖбһ¶бһҡбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӣбһҫ scatterplot? бһҸбһҫбһңбһ¶бһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһңбһ·бһҳбһ¶бһҸбҹ’бһҡвҖӢбһ…бҹҶвҖӢбһҺбһ¶бһҹбҹӢбһҮбһ¶бһ„бһҳбһ»бһ“ бһ бҹ…бһҗбһ¶ *бһҗбҹ’бһ„бҹғбһ“бҹғбһҶбҹ’бһ“бһ¶бҹҶ* (бһ…бҹҶбһ“бһҪбһ“бһҗбҹ’бһ„бҹғбһ…бһ¶бһ”бҹӢбһ–бһёбһҠбһҫбһҳбһҶбҹ’бһ“бһ¶бҹҶ)?

бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ бһ“бһ№бһ„бһӮбһ·бһҸбһҗбһ¶бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҹбһҳбҹ’бһўбһ¶бһҸбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ бһҫбһҷ бһ бһҫбһҷбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“ data frame бһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮ `new_pumpkins` бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

```python
day_of_year = pd.to_datetime(pumpkins['Date']).apply(lambda dt: (dt-datetime(dt.year,1,1)).days)
```

бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһҠбҹ„бһҷбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҷбһӣбҹӢбһ–бһёбһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶бһ“бҹғбһ•бҹ’бһҹбһ¶бһҡбһҡбҹҖбһ”бһҡбһ¶бһ”бҹӢбһҡбһ¶бһ„бһҸбҹӮбһҳбһҪбһҷбһңбһ·бһү бһҹбһјбһҳбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҢбҹӮбһӣбһ•бҹ’бһҹбһ¶бһҡвҖӢбһҡбһҪбһҳвҖӢбһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһҗбһ¶бһҸбһҫбһҷбһҫбһ„бһўбһ¶бһ…бһ‘бһ¶бһҷбһҗбһ¶ бһ”бҹ’бһҡбһўбһ”бҹӢбһ•бҹ’бһӣбҹӮбһ•бҹ’бһўбҹӮбһҒбҹ„бһ…бһҺбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһ•бҹ’бһӣбҹӮбһ•бҹ’бһўбҹӮбһҒбҹ„бһ…бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбҹ” бһўбҹ’бһ“бһҖбһҠбҹӮбһӣбһ‘бһ·бһүбһ•бҹ’бһӣбҹӮбһ•бҹ’бһўбҹӮбһҒбҹ„бһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҡбһ»бһҖбһ•бҹ’бһӣбҹӮбһ•бҹ’бһўбҹӮбһҒбҹ„бһ…бһ“бҹ…бһҗбҹ’бһ„бҹғбһ”бһ»бһҺбҹ’бһҷбһ”бҹ’бһҡбһ бҹӮбһӣбһ“бһ№бһ„бһ…бһ„бҹӢбһ”бһ¶бһ“бһ–бҹҗбһҸбҹҚбһҳбһ¶бһ“бһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ…бһӣбҹҶбһўбҹҖбһ„бһҖбһ¶бһҡбһ‘бһ·бһүбһ”бҹ’бһҡбһўбһ”бҹӢбһ•бҹ’бһӣбҹӮбһ•бҹ’бһўбҹӮбһҒбҹ„бһ…бһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбҹ”

бһ”бһҫбһҷбһҫбһ„бһӮбһҺбһ“бһ¶бһҹбһҳбһ¶бһҳбһ¶бһҸбҹ’бһҡбһҡбһңбһ¶бһ„ `Price` бһ“бһ·бһ„ `DayOfYear` бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `corr` бһңбһ¶бһ“бһ№бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶ `-0.27` - бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһҳбҹүбһјбһҢбҹӮбһӣбһ‘бһ¶бһҷбһ‘бҹ„бһӣбһҳбһ¶бһ“бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһ бһҫбһҷбҹ”

> бһҳбһ»бһ“бһ“бһ№бһ„бһ”бһҺбҹ’бһҸбһ»бҹҮбһҳбҹүбһјбһҢбҹӮбһӣ Linear Regression бһңбһ¶бһҮбһ¶бһҡбһҝбһ„бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҗбһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһ”бһ¶бһ“бһҹбҹ’бһўбһ¶бһҸбҹ” Linear Regression бһҳбһ·бһ“бһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһҳбһҪбһҷбһўбһҗбҹҒбһҡбһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҺбҹӮбһ“бһ¶бҹҶбһІбҹ’бһҷбһҖбһҳбҹ’бһ…бһ¶бһҸбҹӢбһўбһҗбҹҒбһҡбһ‘бһ‘бҹҒбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ–

```python
pie_pumpkins.dropna(inplace=True)
pie_pumpkins.info()
```

бһҳбһ»бһҒбһ„бһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бһӮбҹ’бһ“бһ¶бһӮбһәбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҸбһҳбҹ’бһӣбҹғбһ‘бһ‘бҹҒбһ“бһјбһңбһҸбһҳбҹ’бһӣбҹғбһҳбһ’бҹ’бһҷбһҳбһ–бһёбһҮбһҪбһҡбһҲбһҡбһҸбҹ’бһҡбһјбһңбһ‘бһ¶бһҖбҹӢбһ‘бһ„бҹ”

## бһ•бҹ’бһҹбһ¶бһҡбһҡбҹҖбһ”бһҡбһ¶бһ”бҹӢбһҡбһ¶бһ„бһҸбҹӮбһҳбһҪбһҷбһҹбһ¶бһҳбһүбҹ’бһү

[![ML for beginners - Linear and Polynomial Regression using Scikit-learn](https://img.youtube.com/vi/e4c_UP2fSjg/0.jpg)](https://youtu.be/e4c_UP2fSjg "ML for beginners - Linear and Polynomial Regression using Scikit-learn")

> рҹҺҘ бһ…бһ»бһ…бһҡбһјбһ”бһ—бһ¶бһ–бһҒбһ¶бһ„бһӣбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһҫбһӣбһңбһёбһҠбҹҒбһўбһјбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһҳбҹүбһјбһҢбҹӮбһӣ Linear бһ“бһ·бһ„ Polynomial Regressionбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҺбҹ’бһҸбһ»бҹҮбһҳбҹүбһјбһҢбҹӮбһӣ Linear Regression бһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ **Scikit-learn**бҹ”

```python
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error
from sklearn.model_selection import train_test_split
```

бһҷбһҫбһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһҠбҹ„бһҷбһ”бҹҶбһ”бҹӮбһҖбһ…бҹҶбһ“бһҪбһ“бһ”бһүбҹ’бһ…бһјбһӣ (features) бһ“бһ·бһ„бһ…бҹҒбһүбһҡбһ„бҹӢбһ…бһ¶бҹҶ (label) бһ‘бҹ…бһҮбһ¶ array numpy бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ–

```python
X = pie_pumpkins['DayOfYear'].to_numpy().reshape(-1,1)
y = pie_pumpkins['Price']
```

> бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҗбһ¶бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһң reshape бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһүбҹ’бһ…бһјбһӣ бһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷ package Linear Regression бһҷбһӣбҹӢбһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ” Linear Regression бһҡбһ„бҹӢбһ…бһ¶бҹҶ input array 2D бһҠбҹӮбһӣбһҮбһ¶бһҸбһҪбһҡбһ‘бҹҶбһ¶бһ„бһҡбһ”бһҹбҹӢбһҮбһҪбһҡбһҲбһҡбһ“бһёбһҳбһҪбһҷбҹ—бһҮбһ¶бһҸбһҪбһҡбһ•бҹ’бһӮбһ»бҹҶбһЎбһҫбһ„ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ input бһҸбҹӮбһҳбһҪбһҷ бһҷбһҫбһ„бһ…бһ„бҹӢбһ”бһ¶бһ“ array бһ‘бҹҶбһ бҹҶ NГ—1 бһҠбҹӮбһӣ N бһӮбһәбһ‘бҹҶбһ бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ”бҹӮбһ„бһ…бҹӮбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶ train бһ“бһ·бһ„ test dataset бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҳбҹүбһјбһҢбҹӮбһӣбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһҺбҹ’бһҸбһ»бҹҮбҹ–

```python
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=0)
```

бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһҳбҹүбһјбһҢбҹӮбһӣ Linear Regression бһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„ бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҸбҹӮбһ–бһёбһҡбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһјбһҠбҹ” бһҷбһҫбһ„бһҖбҹҶбһҺбһҸбҹӢбһңбһҸбҹ’бһҗбһ» `LinearRegression` бһ бһҫбһҷбһ—бҹ’бһҮбһ¶бһ”бҹӢбһңбһ¶бһ“бҹ…бһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹ„бһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ `fit`бҹ–

```python
lin_reg = LinearRegression()
lin_reg.fit(X_train,y_train)
```

Object `LinearRegression` бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһ¶бһ“ `fit` бһҡбһҪбһ…бһҳбһ¶бһ“бһӮбһ»бһҖбҹҗбһ—бһёбһҹбҹ’бһҷбһ„бҹӢбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹғбһҡбҹҒбһ бҹ’бһӮбҹ’бһҡбҹҒбһҹбҹ’бһҷбһ»бһ„ бһҠбҹӮбһӣбһўбһ¶бһ…бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бһ…бҹ’бһ…бҹҒбһҖ `.coef_`бҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһҳбһ¶бһ“бһӮбһ»бһҖбҹҗбһ—бһёбһҹбҹ’бһҷбһ„бҹӢбһҸбҹӮбҹЎ бһҠбҹӮбһӣбһӮбһҪбһҡбһҸбҹӮбһ”бҹ’бһҡбһ бҹӮбһӣ `-0.017`бҹ” бһңбһ¶бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶ бһҸбһҳбҹ’бһӣбҹғбһҗбҹ’бһӣбҹғбһ‘бҹҶбһ“бһ·бһүбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһҖбҹҶбһ–бһ»бһ„бһ’бҹ’бһӣбһ¶бһҖбҹӢбһ”бһ“бҹ’бһҸбһ·бһ…бһ”бһ“бҹ’бһҸбһҪбһ…бһҮбһ¶бһҳбһҪбһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ·бһ“бһ…бһ¶бһүбҹӢбһҒбҹ’бһӣбһ¶бҹҶбһ„ бһҸбҹ’бһҡбһ№бһҳбһ”бҹ’бһҡбһ бҹӮбһӣ бҹў бһҹбҹҒбһ“бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҗбҹ’бһ„бҹғбһ‘бҹҒбҹ” бһҷбһҫбһ„бһҖбҹҸбһўбһ¶бһ…бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ…бҹҶбһҺбһ»бһ…бһҶбҹ’бһӣбһ„бһ•бҹ’бһҸбһҪбһ…бһ•бҹ’бһҸбһҫбһҳбһҡбһ”бһҹбҹӢбһҡбҹҒбһ бҹ’бһӮбҹ’бһҡбҹҒбһҹбҹ’бһҷбһ»бһ„бһҮбһ¶бһҳбһҪбһҷбһўбҹҗбһҖбҹ’бһҹ Y бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `lin_reg.intercept_` бһҠбҹӮбһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһ бҹӮбһӣ `21` бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҸбһҳбҹ’бһӣбҹғбһҗбҹ’бһӣбҹғбһӣбһҫбһҖбһҠбҹҶбһ”бһјбһ„бһ“бҹғбһҶбҹ’бһ“бһ¶бҹҶбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһҗбһ¶бһҳбҹүбһјбһҠбҹӮбһӣбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһҳбһ¶бһ“бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ”бҹүбһ»бһҺбһ¶ бһҷбһҫбһ„бһўбһ¶бһ…бһ‘бһ¶бһҷбһ‘бһҳбҹ’бһӣбҹғбһҗбҹ’бһӣбҹғбһ‘бҹҶбһ“бһ·бһүбһӣбһҫбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһңбһ¶бһҹбҹӢбһҗбһ¶бһҸбһҳбҹ’бһӣбҹғбһ‘бһ¶бһҷбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһҮбһ·бһҸбһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбһ”бҹүбһ»бһҺбҹ’бһҺбһ¶бҹ” бһңбһ¶бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҸбҹҶбһӣбҹғ MSE (mean square error) бһҠбҹӮбһӣбһҮбһ¶бһҳбһ’бҹ’бһҷбһҳбһ“бҹғбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһҸбһҳбҹ’бһӣбҹғбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖ бһ“бһ·бһ„бһҸбһҳбҹ’бһӣбҹғбһ‘бһ¶бһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбҹ’бһҡбҹ„бһҳбһ—бһ¶бһ–бһЎбһҫбһ„бһңбһ·бһүбҹ” 

```python
pred = lin_reg.predict(X_test)

mse = np.sqrt(mean_squared_error(y_test,pred))
print(f'Mean error: {mse:3.3} ({mse/np.mean(pred)*100:3.3}%)')
```
  
бһҖбҹҶбһ бһ»бһҹбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһҶбҹ’бһ„бһ¶бһҷбһ”бҹ’бһҡбһ бҹӮбһӣ бҹў бһ–бһ·бһ“бҹ’бһ‘бһ» бһҠбҹӮбһӣбһҮбһ¶бһ”бҹ’бһҡбһ бҹӮбһӣ ~17%бҹ” бһҳбһ·бһ“бһӣбҹ’бһўбһ–бҹҒбһҖбһ‘бҹҒбҹ” бһҹбһүбҹ’бһүбһ¶бһҳбһҪбһҷбһ‘бҹҖбһҸбһ“бҹғбһӮбһ»бһҺбһ—бһ¶бһ–бһҳбҹүбһјбһҠбҹӮбһӣбһӮбһә **coefficient of determination** бһҠбҹӮбһӣбһўбһ¶бһ…бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

```python
score = lin_reg.score(X_train,y_train)
print('Model determination: ', score)
```

бһҡбҹҒбһ бҹ’бһӮбҹ’бһҡбҹҒбһҹбҹ’бһҷбһ»бһ„бһ•бһјбһӣбһёбһҺбһјбһҳбҹ’бһҷм–јбһ”бһ„бҹ’бһҖбһҫбһҸбһҒбҹ’бһҹбҹӮбһ–бҹ„бһ„бһҖбҹ„бһ„ бһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһўбһ¶бһ…бһҹбһ¶бһҖбһҹбһҳбһ“бһ№бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ·бһ“бһҳбҹүбһ¶бһҹбҹ’бһҹбһ¶ linear бһ”бһ¶бһ“бһӣбҹ’бһўбһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһ„бһҳбһ»бһ“бҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ”бһүбҹ’бһ…бһјбһӣбһўбһҗбҹҒбһҡбһ–бһҺбҹҢбһңбһҸбҹ’бһҗбһ» `DayOfYear`^2 бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһүбҹ’бһ…бһјбһӣ бһҷбһҫбһ„бһӮбһҪбһҡбһҸбҹӮбһўбһ¶бһ…бһҸбҹҶбһЎбһҫбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһҳбһҪбһҷбһҒбҹ’бһҹбҹӮ parabola бһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһўбһ”бҹ’бһ”бһ”бһҡбһҳбһ¶бһ“бҹ…бһ–бҹҒбһӣбһҖбҹҶбһҺбһҸбҹӢбһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһҶбҹ’бһ“бһ¶бҹҶбҹ”

Scikit-learn бһҳбһ¶бһ“ [pipeline API](https://scikit-learn.org/stable/modules/generated/sklearn.pipeline.make_pipeline.html?highlight=pipeline#sklearn.pipeline.make_pipeline) бһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ бһҠбһҫбһҳбҹ’бһ”бһёбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбҹҶбһ бһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһҹбҹҒбһ„бҹ—бһӮбҹ’бһ“бһ¶бҹ” **pipeline** бһӮбһәбһҮбһ¶бһҹбһ„бҹ’бһҹбҹҗбһҷбһ“бҹғ **estimators**бҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһҷбһҫбһ„бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸ pipeline бһҠбҹӮбһӣбһҮбһ¶бһӣбһҫбһҖбһҠбҹҶбһ”бһјбһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһ•бһјбһӣбһёбһҺбһјбһҳбҹ’бһҷм–јбһ‘бҹ…бһҳбҹүбһјбһҠбҹӮбһӣ бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһҡбҹҒбһ бҹ’бһӮбҹ’бһҡбҹҒбһҹбҹ’бһҷбһ»бһ„бҹ–

pipeline бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ¶бһ“бһҠбһјбһ…бһҮбһ¶бһўбһ„бҹ’бһӮбһ—бһ¶бһ– `LinearRegression` бһҠбһҫбһҳ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮвҖӢбһҷбһҫбһ„бһўбһ¶бһ… `fit` pipeline бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ”бҹ’бһҡбһҫ `predict` бһ”бһ¶бһ“бһ•бһ„бһҠбҹӮбһҡ бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһ‘бһ¶бһҷбҹ” бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҮбһ¶бһҖбҹ’бһҡбһ¶бһ бҹ’бһңбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһҒбҹ’бһҹбҹӮбһ–бҹ„бһ„бһҖбҹ„бһ„бһҸбҹҶбһҺбһ¶бһ„бҹ–

бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҡбҹҒбһ бҹ’бһӮбҹ’бһҡбҹҒбһҹбҹ’бһҷбһ»бһ„бһ•бһјбһӣбһёбһҺбһјбһҳбҹ’бһҷм–ј бһҷбһҫбһ„бһўбһ¶бһ…бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“ MSE бһ‘бһ¶бһ”бһҮбһ¶бһ„бһ”бһ“бҹ’бһҸбһ·бһ… бһ“бһ·бһ„ coefficient of determination бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„бһ”бһ“бҹ’бһҸбһ·бһ… бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ·бһ“бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҺбһ¶бһҹбҹӢбһ‘бҹҒбҹ” бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҷбһҖбһ…бҹҶбһҺбһ»бһ…бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһ•бһ„бһҠбҹӮбһҡ!

> бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһҫбһҖбһ…бҹҶбһҺбһ¶бҹҶбһ”бһ¶бһ“бһҗбһ¶ бһҸбһҳбҹ’бһӣбҹғбһҗбҹ’бһӣбҹғбһўбһ„бҹ’бһӮбһҡбһҮбһ¶бһҳбһҪбһҷбһҒбҹ’бһ“бһ„бһ“бҹ„бҹҮбһҳбһ¶бһ“бһ…бһ“бҹ’бһӣбҹ„бҹҮбһ–бҹҒбһӣбһҠбҹӮбһӣбһҮбһ·бһҸбһҗбҹ’бһ„бҹғ Halloweenбҹ” бһҸбһҫбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ–бһ“бҹ’бһҷбһӣбҹӢбһҳбһјбһӣбһ бҹҒбһҸбһ»бһ“бҹҒбҹҮбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…?

рҹҺғ бһҹбһјбһҳбһўбһ”бһўбһҡбһҹбһ¶бһ‘бһҡ бһўбҹ’бһ“бһҖбһ‘бһҫбһ”бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣбһҳбһҪбһҷбһҠбҹӮбһӣбһўбһ¶бһ…бһҮбһҪбһҷбһ‘бһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҗбҹ’бһӣбҹғбһ“бҹҶбһ‘бҹҶбһ–бһ¶бҹҶбһ„бһ”бһ¶бһҷбһҮбһјбһ”бһ¶бһ“бҹ” бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡвҖӢбһҸбҹӮбһҳбһҪбһҷбһ“бҹҒбҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бҹҶбһ–бһ¶бҹҶбһ„бһ”бһ¶бһҷбһҮбһјбһӮбҹ’бһҡбһ”бҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ”бһ¶бһ“бһ•бһ„ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ¶бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһӣбҹҶбһ”бһ¶бһҖбҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһҡбҹҖбһ“бһҡбһ”бҹҖбһ”бһҷбһҖбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бҹҶбһ–бһ¶бҹҶбһ„бһ”бһ¶бһҷбһҮбһјбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһӮбҹҶбһҡбһјбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһҘбһӣбһјбһңбһ“бҹҒбҹҮ!

## бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҠбһјбһ…бһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘ (Categorical Features)

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ–бһ·бһ—бһ–бһ•бҹ’бһҸбһӣбҹӢбһҖбҹ’бһҸбһёбһҹбһ„бҹ’бһғбһ№бһҳ бһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһўбһ¶бһ…бһ‘бһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҗбҹ’бһӣбҹғбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бҹҶбһ–бһ¶бҹҶбһ„бһ”бһ¶бһҷбһҮбһјбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһҸбҹӮбһҳбһҪбһҷбҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҖбһ“бҹ’бһӣбҹӮбһ„бһҮбһҪбһҡбһҲбһҡ `Variety` бһҳбһ¶бһ“бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ–бһёбһҮбһҪбһҡбһҲбһҡ `Month` бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ”бһ„бҹ’бһҖбһ”бҹӢбһҸбһҳбҹ’бһӣбҹғбһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһҸбһҪбһӣбҹҒбһҒбһ‘бҹҒбҹ” бһҮбһҪбһҡбһҲбһҡбһҠбһјбһ…бһ“бҹҒбҹҮбһ бҹ…бһҗбһ¶ **categorical**бҹ”

> рҹҺҘ бһ…бһ»бһ…бһҡбһјбһ”бһ—бһ¶бһ–бһҒбһ¶бһ„бһӣбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһёбһҠбҹҒбһўбһјбһҮбһ¶бһҹбһ„бҹ’бһҒбҹҒбһ”бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҲбҹ’бһҳбҹ„бҹҮбһҠбһјбһ…бһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бҹ”

бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҫбһӣбһҗбһ¶бһҸбһҳбҹ’бһӣбҹғбһҗбҹ’бһӣбҹғбһҳбһ’бҹ’бһҷбһҳбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһ”бҹ’бһҡбһ—бҹҒбһ‘бҹ–

бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһҖбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһӮбҹҶбһҡбһјбһ”бһ¶бһ“ бһҸбҹ’бһҡбһјбһңбһ”бһҳбҹ’бһӣбҹӮбһ„бһңбһ¶бһ‘бҹ…бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһӣбҹҒбһҒбһ¬ **encode**бһңбһ¶бҹ” бһҳбһ¶бһ“бһңбһ·бһ’бһёбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбҹҲ

* **numeric encoding** бһҹбһ¶бһҳбһүбҹ’бһү бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҸбһ¶бһҡбһ¶бһ„бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ•бһ„бһҠбҹӮбһҡ бһ бһҫбһҷбһ”бһҳбҹ’бһӣбҹӮбһ„бһҲбҹ’бһҳбҹ„бҹҮбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҮбһ¶бһӣбҹҒбһҒбһҸбҹӮбһҳбҹ’бһҠбһ„бһҖбҹ’бһ“бһ»бһ„бһҸбһ¶бһҡбһ¶бһ„бҹ” бһ“бҹҒбҹҮбһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһӮбҹҶбһ“бһ·бһҸбһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбҹҒбһ бҹ’бһӮбҹ’бһҡбҹҒбһҹбҹ’бһҷбһ»бһ„бһҡбҹҒбһ бҹ’бһӮбҹ’бһҡбҹҒбһҹбҹ’бһҷбһ»бһ„ linear бһ‘бҹҒ бһ–бҹ’бһҡбҹ„бҹҮбһңбһ¶бһ”бҹ’бһҡбһҫбһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹ„бҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһӣбһ‘бҹ’бһ’бһ•бһӣ бһ”бһјбһҖбһ“бһ№бһ„бһӮбһ»бһҖбҹҗбһ—бһёбһҹбҹ’бһҷбһ„бҹӢбһ•бҹ’бһҹбҹҒбһ„бҹ— бһ бһҫбһҷбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҡбһңбһ¶бһ„бһӣбҹҒбһҒбһ“бһ·бһ„бһҸбһҳбҹ’бһӣбҹғбһҗбҹ’бһӣбҹғбһҳбһ·бһ“бһҳбҹӮбһ“ linear бһ‘бҹҒ бһ‘бҹ„бҹҮбһ”бһёбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһӣбҹҒбһҒбһҮбһ¶бһҸбһҳбҹ’бһӣбҹғбһӣбҹҶбһҠбһ¶бһ”бҹӢбһҖбҹ’бһҡбҹ„бһҳбһҖбҹҸбһҠбҹ„бһҷбҹ”
* **One-hot encoding** бһ“бһ№бһ„бһ”бһҳбҹ’бһӣбҹӮбһ„бһҮбһҪбһҡбһҲбһҡ `Variety` бһ‘бҹ…бһҮбһ¶бһҮбһҪбһҡбһҲбһҡбһ”бһҪбһ“бһӮбҹ’бһ“бһ¶ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ“бһёбһҳбһҪбһҷбҹ—бһӮбҹ’бһ“бһ¶бҹ” бһңбһ¶вҖӢбһўбһ¶бһ…бһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғ `1` бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҮбһҪбһҡбһҠбҹҒбһҖбһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ“бҹ„бҹҮ бһ бһҫбһҷ`0` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ” бһ“бҹҒбҹҮбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһ“бһ№бһ„бһҳбһ¶бһ“бһӮбһ»бһҖбҹҗбһ—бһёбһҹбҹ’бһҷбһ„бҹӢбһ”бһҪбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбҹҒбһ бҹ’бһӮбҹ’бһҡбҹҒбһҹбҹ’бһҷбһ»бһ„ linear бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бҹҶбһ–бһ¶бҹҶбһ„бһ”бһ¶бһҷбһҮбһј бһ”бҹҶбһ–бҹҒбһүбһҠбҹҶбһ”бһјбһ„бһ“бҹғбһҸбһҳбҹ’бһӣбҹғбһҗбҹ’бһӣбҹғбһ¬ "бһҸбһҳбҹ’бһӣбҹғбһ”бһ“бҹ’бһҗбҹӮбһҳ" бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ“бҹ„бҹҮбҹ”

бһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҷбһҫбһ„бһўбһ¶бһ…бһҖбһјбһҠ one-hot encoding бһ…бҹҶбһ–бҹ„бҹҮбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ”бһ¶бһ“бҹ–

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһҡбҹҒбһ бҹ’бһӮбҹ’бһҡбҹҒбһҹбҹ’бһҷбһ»бһ„ linear бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ—бҹҒбһ‘ бһҒбһҺбҹҲбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҖбһ”бҹ’бһҡбҹӮбһҮбһ¶ one-hot encoded бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣ бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһң initialize бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ `X` бһ“бһ·бһ„ `y` бһІбҹ’бһҷбһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ–

```python
X = pd.get_dummies(new_pumpkins['Variety'])
y = new_pumpkins['Price']
```
  
бһҖбһјбһҠбһ“бҹ…бһҹбһӣбҹӢбһҳбһ¶бһ“бһҠбһјбһ…бһҠбҹӮбһӣбһҷбһҫбһ„бһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ‘бҹ…бһҳбһ»бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣ Linear Regressionбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһңбһ¶бһўбһ¶бһ…бһғбһҫбһүбһҗбһ¶ mean squared error бһ”бҹ’бһҡбһ бҹӮбһӣбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҷбһҫбһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“ coefficient of determination бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„бһӮбҹҒ (~77%)бҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбһ¶бһҡбһ‘бһ¶бһҷбһӣбҹ’бһўбһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһ„бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҷбһҖбһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһ–бһ бһ» (categorical) бһ“бһ·бһ„ бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһ…бҹҶбһ“бһҪбһ“ (numeric) бһҠбһјбһ…бһҮбһ¶ `Month` бһ¬ `DayOfYear` бһ…бһјбһӣбһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶бһ…бҹ’бһҡбһҫбһ“бһ‘бҹҖбһҸбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһ¶бһ“бһўбһ¶бһҡбҹҒбһ…бһ¶бһҷбһ’бҹҶбһҳбһҪбһҷ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбһҖбҹ’бһҒбһҺбҹҲбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ `join`бҹ–

```python
X = pd.get_dummies(new_pumpkins['Variety']) \
        .join(new_pumpkins['Month']) \
        .join(pd.get_dummies(new_pumpkins['City'])) \
        .join(pd.get_dummies(new_pumpkins['Package']))
y = new_pumpkins['Price']
```
  
бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбһҷбһҫбһ„бһҖбҹҸбһҷбһҖбһ…бҹҶбһҺбһ»бһ… `City` бһ“бһ·бһ„бһ”бҹ’бһҡбһ—бҹҒбһ‘ `Package` бһ…бһјбһӣбһӮбһҺбһ“бһ¶бһ•бһ„ бһҠбҹ„бһҷбһҳбһ¶бһ“ MSE 2.84 (10%) бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ“бҹғбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ 0.94!

## бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹ’бһ“бһ¶бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ бһҷбһҫбһ„бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһҹбҹҶбһ”бҹ’бһҡбһ—бҹҒбһ‘ (one-hot encoded categorical) бһ“бһ·бһ„бһ…бҹҶбһ“бһҪбһ“ (numeric) бһ–бһёбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҒбһ¶бһ„бһӣбһҫбһҮбһ¶бһҳбһҪбһҷбһҡбҹҒбһ бҹ’бһӮбҹ’бһҡбҹҒбһҹбҹ’бһҷбһ»бһ„бһ•бһјбһӣбһёбһҺбһјбһҳбҹ’бһҷм–јбҹ” бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҮбһ¶бһҖбһјбһҠбһ–бҹҒбһүбһӣбҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһҪбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–

```python
# бһҡбҹҖбһ”бһ…бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣ
X = pd.get_dummies(new_pumpkins['Variety']) \
        .join(new_pumpkins['Month']) \
        .join(pd.get_dummies(new_pumpkins['City'])) \
        .join(pd.get_dummies(new_pumpkins['Package']))
y = new_pumpkins['Price']

# бһ”бҹҶбһ”бҹӮбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣ бһ“бһ·бһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=0)

# бһҡбҹҖбһ”бһ…бҹҶбһ–бһёбһ•бҹ’бһӣбһјбһңбһ”бҹҶбһҺбһ„ бһ“бһ·бһ„бһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣ
pipeline = make_pipeline(PolynomialFeatures(2), LinearRegression())
pipeline.fit(X_train,y_train)

# бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷбһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„
pred = pipeline.predict(X_test)

# бһӮбһҺбһ“бһ¶бһҡбҹүбһ¶бһҹбҹҠбһё MSE бһ“бһ·бһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһҖбҹҶбһҺбһҸбҹӢ
mse = np.sqrt(mean_squared_error(y_test,pred))
print(f'Mean error: {mse:3.3} ({mse/np.mean(pred)*100:3.3}%)')

score = pipeline.score(X_train,y_train)
print('Model determination: ', score)
```
  
бһ“бҹҒбҹҮбһӮбһҪбһҡбһҸбҹӮбһ•бҹ’бһҸбһӣбҹӢ coefficient of determination бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһ”бҹ’бһҡбһ бҹӮбһӣ 97% бһ бһҫбһҷ MSE=2.23 (~8% бһҖбҹҶбһ бһ»бһҹбһ‘бһ¶бһҷ)бҹ”

| бһҳбҹүбһјбһҠбҹӮбһӣ | MSE | бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶ |  
|-------|-----|---------------|

рҹҸҶ бһӣбҹ’бһўбһҺбһ¶бһҹбҹӢ! бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣбһҡбҹҒбһ бҹ’бһӮбҹ’бһҡбҹҒбһҹбҹ’бһҷбһ»бһ„бһ”бһҪбһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҳбҹҒбһҡбҹҖбһ“ бһ бһҫбһҷбһ”бһ„бҹ’бһҖбһҫбһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһҳбҹүбһјбһҠбҹӮбһӣбһҠбһӣбҹӢ 97%бҹ” бһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһўбҹҶбһ–бһёбһҡбҹҒбһ бҹ’бһӮбҹ’бһҡбҹҒбһҹбҹ’бһҷбһ»бһ„ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһўбҹҶбһ–бһё Logistic Regression бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бҹ”
