## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһ“бҹҒбҹҮбһӣбһҫбһҖбһ¶бһҡбһ§бһ‘бҹ’бһ‘бһ·бһҹбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ Regression бһҠбҹӮбһӣбһҮбһ¶бһҸбһ·бһ…бһҡбһ”бһҹбҹӢбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ ML _бһ”бһ»бһҡбһ¶бһҺ_, бһҷбһҫбһ„бһ“бһ№бһ„бһҹбҹ’бһӮбһ¶бһӣбҹӢ Logistic Regressionбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбһҳбһҫбһӣбһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ¶бһҷбһҖбһ¶бһҸбҹҒбһҖбһјбһҡбһёбһ–бһёбһҡбһ”бһ¶бһ“бҹ” бһҸбһҫбһҹбҹ’бһҖбһҡбһӮбһәбһҮбһ¶бһҹбҹ’бһңбһ¶бһҳбҹүбҹҒбһҸбһ«бһҳбһ·бһ“бһҳбҹӮбһ“? бһҸбһҫбһҮбһҳбҹ’бһ„бһәбһ“бҹҒбҹҮбһҶбҹ’бһӣбһ„бһ¬бһҳбһ·бһ“? бһҸбһҫбһўбһҸбһ·бһҗбһ·бһҮбһ“бһ“бҹҒбҹҮбһ“бһ№бһ„бһҡбһҫбһҹбһ•бһӣбһ·бһҸбһ•бһӣбһ“бҹҒбҹҮбһ«бһҳбһ·бһ“?

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бҹ–

- бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҗбҹ’бһҳбһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҳбһҫбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ
- бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ logistic regression

вң… бһҮбҹ’бһҡбһ¶бһ”бһ…бҹҶбһ бһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ regression бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ“бҹҒбҹҮбһҖбҹ’бһ“бһ»бһ„ [бһҳбһјбһҢбһ»бһӣбһҡбҹҖбһ“](https://docs.microsoft.com/learn/modules/train-evaluate-classification-models?WT.mc_id=academic-77952-leestott)

## бһҖбһ¶бһҡбһҸбҹ’бһҡбҹҖбһҳбһҒбҹ’бһӣбһҪбһ“

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһӣбҹҒбһ„бһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбһҳбҹ’бһҡбһ¶бһӣбһ•бҹ’бһӣбҹӮбһҲбһҫ pumpkin бһҷбһҫбһ„бһҳбһ¶бһ“бһҖбһ¶бһҡбһҹбҹ’бһӮбһ¶бһӣбҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһҷбһӣбҹӢбһҗбһ¶бһҳбһ¶бһ“бһҖбһ¶бһҸбҹҒбһҖбһјбһҡбһёбһ–бһёбһҡбһҳбһҪбһҷбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһӮбһә `Color`бҹ”

бһҳбһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣ logistic regression бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ¶бһҷбһҗбһ¶ бһ–бһёбһўбһҗбҹҒбһҡбһҒбҹ’бһӣбҹҮбҹ— _бһ–бһҺбҹҢбһҡбһ”бһҹбҹӢ pumpkin бһҳбһҪбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбһ…бһ„бҹ’бһўбһ»бһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ“бһ№бһ„бһҳбһ¶бһ“бһўбҹ’бһңбһёбһ“бҹ…бһӣбһҫбһ‘бҹ…_ (бһ‘бһ№бһҖбһҖбҹ’бһҡбһјбһ… рҹҺғ бһ¬ бһ–бһҺбҹҢбһҹ рҹ‘»)бҹ”

> бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҷбһҫбһ„бһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһ–бһёбһ…бҹҶбһҺбһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ–бһёбһҡ бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ“бһ№бһ„ regression? бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ—бһ¶бһҹбһ¶ бһ”бһҫбһ‘бҹ„бҹҮбһ”бһё logistic regression бһҮбһ¶бһңбһ·бһ’бһёбһ…бҹҶбһҺбһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҳбһҪбһҷ [бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ](https://scikit-learn.org/stable/modules/linear_model.html#logistic-regression) бһҖбҹҸбһҠбҹ„бһҷ бһҠбҹӮбһӣбһҳбһ¶бһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӣбһҫбһҒбҹ’бһҹбҹӮбһҸбһҪбҹ” бһҡбҹҖбһ“бһўбҹҶбһ–бһёбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹ…бһҳбҹҒбһҡбҹҖбһ“бһҖбҹ’бһҡбҹ„бһҷбҹ”

## бһҖбҹҶбһҺбһҸбҹӢбһҹбҹҶбһҺбһҪбһҡ

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһҷбһҫбһ„бһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ“бҹҒбҹҮбһҮбһ¶бһҖбһ¶бһҡбһ…бһ¶бһҸбҹӢбһҸбһ¶бһҳбһ–бһёбһҡбһҷбҹүбһ¶бһ„бҹ– 'бһ–бһҺбҹҢбһҹ' бһ¬ 'бһҳбһ·бһ“бһҳбҹӮбһ“бһ–бһҺбҹҢбһҹ'бҹ” бһҖбҹҸбһҳбһ¶бһ“бһҖбһ¶бһҸбҹҒбһҖбһјбһҡбһё 'бһ–бһҺбҹҢбһ”бҹҠбһ·бһ…' бһ•бһ„бһҠбҹӮбһӣбһҳбһ¶бһ“бһ“бҹ…бһӣбһҫdataset, бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ¶бһ“бһҖбһҡбһҺбһёбһҸбһ·бһ… бһҮбһ¶бһҗбҹ’бһҳбһёбһҷбһҫбһ„бһҳбһ·бһ“бһ”бҹ’бһҡбһҫбһңбһ¶бһ‘бҹҒбҹ” бһңбһ¶бһ“бһ№бһ„бһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһ“бҹ…бһ–бҹҒбһӣбһҷбһҫбһ„бһҠбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһӮбҹ’бһҳбһ¶бһ“бһўбҹ’бһңбһёбһ–бһё dataset бһ•бһ„бһҠбҹӮбһҡбҹ”

> рҹҺғ бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҡбһёбһҖбһҡбһ¶бһҷ бһӮбҹҒбһ–бҹҒбһӣбһҒбҹ’бһӣбҹҮбһ бҹ… pumpkin бһ–бһҺбҹҢбһҹбһҗбһ¶ pumpkin 'бһ—бҹ’бһ“бҹҶ'. бһңбһ¶бһҳбһ·бһ“бһ„бһ¶бһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бҹҶбһўбһ·бһ“ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһңбһ¶бһҳбһ·бһ“бһ–бҹҒбһүбһ“бһ·бһҷбһҳбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһҠбһјбһ… pumpkin бһ‘бһ№бһҖбһҖбҹ’бһҡбһјбһ…, бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ¶бһҳбһҫбһӣбһ‘бһ¶бһҖбҹӢбһ‘бһ¶бһүбһҺбһ¶бһҹбҹӢ! бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ бһҷбһҫбһ„бһўбһ¶бһ…бһҖбҹӮбһҹбҹҶбһҺбһҪбһҡбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһҮбһ¶бһҗбһ¶бҹ– 'бһ—бҹ’бһ“бҹҶ' бһ¬ 'бһҳбһ·бһ“бһ—бҹ’бһ“бҹҶ' рҹ‘»бҹ”

## бһўбҹҶбһ–бһё logistic regression

Logistic regression бһ•бҹ’бһҹбҹҒбһ„бһ–бһё linear regression бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ“бһҳбһ»бһ“бһ“бҹҒбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһүбҹ’бһ бһ¶бһҒбҹ’бһӣбҹҮбҹ”

[![ML for beginners - Understanding Logistic Regression for Machine Learning Classification](https://img.youtube.com/vi/KpeCT6nEpBY/0.jpg)](https://youtu.be/KpeCT6nEpBY "ML for beginners - Understanding Logistic Regression for Machine Learning Classification")

> рҹҺҘ бһ…бһ»бһ…бһӣбһҫбһҡбһјбһ”бһ—бһ¶бһ–бһҒбһ¶бһ„бһӣбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһёбһҠбҹҒбһўбһјбһҳбҹ„бһ‘бһ“бһ—бһ¶бһ–бһҒбҹ’бһӣбһёбһўбҹҶбһ–бһё logistic regressionбҹ”

### бһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ–бһёбһҡбһҮбһ¶бһ“бҹӢ

Logistic regression бһҳбһ·бһ“бһ•бҹ’бһҸбһӣбҹӢбһӣбһҖбҹ’бһҒбһҺбҹҲбһҠбһјбһ… linear regression бһ‘бҹҒбҹ” Logistic regression бһ•бҹ’бһҸбһӣбҹӢбһ‘бһ¶бһҷбһ“бҹҗбһҷбһўбҹҶбһ–бһёбһҖбһ¶бһҸбҹҒбһҖбһјбһҡбһёбһ–бһёбһҡбһҠбһјбһ…бһҮбһ¶ ("бһ–бһҺбҹҢбһҹ бһ¬ бһҳбһ·бһ“бһ–бһҺбҹҢбһҹ") бһҒбһҺбҹҲбһҠбҹӮбһӣ linear regression бһўбһ¶бһ…бһ‘бһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҮбһ¶бһ”бһ“бҹ’бһҸ (бһ”бһ·бһ‘бһ“бҹҒбҹҮбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҠбһҫбһҳ pumpkin бһ“бһ·бһ„бһ–бҹҒбһӣбһҖбһ¶бһ”бҹӢ бһ”бҹ’бһҡбһ бҹӮбһӣбһҗбһ¶ _бһҸбһҳбҹ’бһӣбҹғбһҡбһ”бһҹбҹӢбһңбһ¶бһ“бһ№бһ„бһЎбһҫбһ„бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“_)бҹ”

### бһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ•бҹ’бһҹбҹҒбһ„бҹ—

бһҳбһ¶бһ“бһ”бҹ’бһҡбһ—бҹҒбһ‘ logistic regression бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ бһҡбһҪбһҳбһҳбһ¶бһ“ multinomial бһ“бһ·бһ„ ordinalбҹ–

- **Multinomial** бһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҸбҹҒбһҖбһјбһҡбһёбһӣбһҫбһҹбһ–бһёбһҳбһҪбһҷбҹҲ "бһ‘бһ№бһҖбһҖбҹ’бһҡбһјбһ…, бһ–бһҺбҹҢбһҹ бһ“бһ·бһ„ бһ”бҹҠбһ·бһ…"бҹ”
- **Ordinal** бһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҸбҹҒбһҖбһјбһҡбһёбһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢ бһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҷбһҫбһ„бһ…бһ„бҹӢбһҸбһҳбҹ’бһҡбҹҖбһ”бһӣбһ‘бҹ’бһ’бһ•бһӣбһҷбҹүбһ¶бһ„бһҳбһ¶бһ“аё•аёЈаёЈаёҒаё°бһҠбһјбһ… pumpkin бһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһҖбҹҶбһҺбһҸбҹӢ (бһҸбһјбһ… бһ’бҹҶбһҳбһ’бҹ’бһҷбһҳ бһ’бҹҶ бһӣбҹҶбһҠбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶)бҹ”

бһ…бһ„бһ…бһ¶бҹҶбһҗбһ¶ linear regression бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһӣбҹ’бһўбһҮбһ¶бһҳбһҪбһҷбһўбһҗбҹҒбһҡбһҹбһҳбһ¶бһҹбһ—бһ¶бһ–бһӣбҹ’бһўбһҮбһ¶бһ„бһ‘бҹҒ? Logistic regression бһ•бҹ’бһ‘бһ»бһҷбһ‘бҹ…бһңбһ·бһү - бһўбһҗбҹҒбһҡбһҳбһ·бһ“бһ”бһ¶бһ…бҹӢбһҸбҹ’бһҡбһјбһңбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ“бһ№бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹҒбҹҮбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһ¶бһҹбһ—бһ¶бһ–бһўбһ“бҹӢбҹ”

Logistic regression бһ“бһ№бһ„бһ“бһ¶бҹҶбһІбҹ’бһҷбһ”бһ¶бһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһЎбһҫбһ„ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бҹ’бһҡбһҫбһ“; dataset бһҸбһјбһ…бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһҳбһ·бһ“бһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ„бһ¶бһҡбһ“бҹҒбҹҮбһ‘бҹҒ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҹбһјбһҳбһ…бһ„бһ…бһ¶бҹҶбҹ”

> рҹҺҘ бһ…бһ»бһ…бһӣбһҫбһҡбһјбһ”бһ—бһ¶бһ–бһҒбһ¶бһ„бһӣбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһёбһҠбҹҒбһўбһјбһҳбҹ„бһ‘бһ“бһ—бһ¶бһ–бһҒбҹ’бһӣбһёбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҸбҹ’бһҡбҹҖбһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ linear regression

Ordinal encoder: бһҹбһҳбһҹбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһҗбҹҒбһҡбһўбһ¶бһӮбһ»бһҷбһҡбһ”бһҹбҹӢ ordinal бһҠбҹӮбһӣбһҮбһ¶бһҖбһ¶бһҸбҹҒбһҖбһјбһҡбһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһӣбҹҶбһҠбһ¶бһ”бҹӢбһҠбһјбһ…бһҮбһ¶ `Item Size` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ dataset бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бҹ” бһңбһ¶бһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹӮбһ“бһ‘бһёбһҠбҹӮбһӣбһҡбһ¶бһӣбҹӢбһҖбһ¶бһҸбҹҒбһҖбһјбһҡбһёбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҸбҹҶбһҺбһ¶бһ„бһҠбҹ„бһҷбһӣбҹҒбһҒ бһҠбҹӮбһӣбһҮбһ¶бһӣбҹҶбһҠбһ¶бһ”бҹӢбһҡбһ”бһҹбҹӢбһҖбһ¶бһҸбҹҒбһҖбһјбһҡбһ“бҹ„бҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһЎбһ»бҹҶбҹ”

    ```python
    from sklearn.preprocessing import OrdinalEncoder

    item_size_categories = [['sml', 'med', 'med-lge', 'lge', 'xlge', 'jbo', 'exjbo']]
    ordinal_features = ['Item Size']
    ordinal_encoder = OrdinalEncoder(categories=item_size_categories)
    ```

2. Categorical encoder: бһҹбһҳбһҹбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһҗбҹҒбһҡбһўбһ¶бһӮбһ»бһҷбһҡбһ”бһҹбҹӢ nominal бһҠбҹӮбһӣбһҮбһ¶бһҖбһ¶бһҸбҹҒбһҖбһјбһҡбһёбһҠбҹӮбһӣбһҳбһ·бһ“бһҳбһ¶бһ“бһӣбҹҶбһҠбһ¶бһ”бҹӢбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһҠбһјбһ…бһҮбһ¶ бһӣбһҖбҹ’бһҒбһҺбҹҲбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҖбҹ’бһҡбҹ…бһ–бһё `Item Size` бһҖбҹ’бһ“бһ»бһ„ dataset бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бҹ” бһңбһ¶бһҮбһ¶ one-hot encoding бһҠбҹӮбһӣбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһҡбһ¶бһӣбҹӢбһҖбһ¶бһҸбҹҒбһҖбһјбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҸбҹҶбһҺбһ¶бһ„бһҠбҹ„бһҷбһҖбһјбһЎбһ»бҹҶбһ”бҹҠбһёбһҺбһ¶бһҡбһёбҹ– бһўбһҗбҹҒбһҡбһўбҹҠбһ·бһ“бһҖбһјбһҠбһҹбҹ’бһҳбһҫбһ“бһ№бһ„ 1 бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫ pumpkin бһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„ Variety бһ“бҹ„бҹҮ бһ“бһ·бһ„ 0 бһҳбһ·бһ“бһҠбһӣбҹӢбҹ”

    ```python
    from sklearn.preprocessing import OneHotEncoder

    categorical_features = ['City Name', 'Package', 'Variety', 'Origin']
    categorical_encoder = OneHotEncoder(sparse_output=False)
    ```
бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ `ColumnTransformer` бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһјбһӣ encoder бһ…бҹ’бһҡбһҫбһ“бһ‘бҹ…бһҮбһ¶бһҮбҹҶбһ бһ¶бһ“бһҸбҹӮбһҳбһҪбһҷ бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбһҫбһҖбһјбһЎбһ»бҹҶбһҠбҹӮбһӣбһҹбһҳбһҡбһҳбҹ’бһҷбҹ”

```python
    from sklearn.compose import ColumnTransformer
    
    ct = ColumnTransformer(transformers=[
        ('ord', ordinal_encoder, ordinal_features),
        ('cat', categorical_encoder, categorical_features)
        ])
    
    ct.set_output(transform='pandas')
    encoded_features = ct.fit_transform(pumpkins)
```
бһ•бҹ’бһ‘бҹғбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ encode бһҹбҹ’бһӣбһ¶бһҖ label бһҷбһҫбһ„бһ”бҹ’бһҡбһҫбһҗбҹ’бһ“бһ¶бһҖбҹӢ `LabelEncoder` бһҡбһ”бһҹбҹӢ scikit-learn бһҠбҹӮбһӣбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҮбһҪбһҷnormalize бһҹбҹ’бһӣбһ¶бһҖбһ”бҹӮбһ” бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҖбҹ’бһ“бһ»бһ„бһҡбһҖбһҳбһҪбһҷбһ–бһё 0 бһҠбһӣбҹӢ n_classes-1 (бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ 0 бһ“бһ·бһ„ 1)бҹ”

```python
    from sklearn.preprocessing import LabelEncoder

    label_encoder = LabelEncoder()
    encoded_label = label_encoder.fit_transform(pumpkins['Color'])
```
бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһё encode feature бһ“бһ·бһ„ label бһҡбһҪбһ… бһҷбһҫбһ„бһўбһ¶бһ…бһ”бһүбҹ’бһ…бһјбһӣбһ–бһҪбһҖбһңбһ¶бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ dataframe бһҗбҹ’бһҳбһё `encoded_pumpkins`бҹ”

```python
    encoded_pumpkins = encoded_features.assign(Color=encoded_label)
```
вң… бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ“бҹғбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ ordinal encoder бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһјбһЎбһ»бҹҶ `Item Size` бһҳбһ¶бһ“бһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮ?

### бһңбһ·бһ—бһ¶бһӮбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҡбһңбһ¶бһ„бһўбһҗбҹҒбһҡ

бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһҠбҹӮбһӣбһҷбһҫбһ„бһ”бһ¶бһ“бһ–бҹ’бһҷбһ¶бһ”бһ¶бһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһҳбһ»бһ“бһ бһҫбһҷ бһўбһ¶бһ…бһңбһ·бһ—бһ¶бһӮбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҡбһңбһ¶бһ„ feature бһ“бһ·бһ„ label бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһӣбҹӢбһ–бһёбһҗбһ¶ бһҳбҹүбһјбһҠбҹӮбһӣбһўбһ¶бһ…бһ‘бһ¶бһҷбһ”бһ¶бһ“бһӣбҹ’бһўбһ”бҹүбһ»бһҺбҹ’бһҺбһ¶ бһ“бҹ…бһ–бҹҒбһӣбһ•бҹ’бһҸбһӣбҹӢбһ…бҹҶбһҺбһјбһӣбһўбһҗбҹҒбһҡбҹ”
бһңбһ·бһ’бһёбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ“бҹҒбҹҮбһӮбһәбһӮбҹҶбһ“бһјбһҡбҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһҳбһ»бһҒбһ„бһ¶бһҡ catplot бһҡбһ”бһҹбҹӢ Seaborn бһңбһ·бһү бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҡбһңбһ¶бһ„ `Item Size`, `Variety` бһ“бһ·бһ„ `Color` бһҖбҹ’бһ“бһ»бһ„бһӮбҹҶбһ“бһјбһҡбһҹбһүбҹ’бһүбһ¶бһҖбһ¶бһҸбҹҒбһҖбһјбһҡбһёбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹҶбһ“бһјбһҡ бһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһҖбһјбһЎбһ»бҹҶ `Item Size` бһҠбҹӮбһӣбһ”бһ¶бһ“ encode бһ бһҫбһҷ бһ“бһ·бһ„бһҖбһјбһЎбһ»бҹҶ `Variety` бһҳбһ·бһ“бһ”бһ¶бһ“ encodeбҹ”

```python
    palette = {
    'ORANGE': 'orange',
    'WHITE': 'wheat',
    }
    pumpkins['Item Size'] = encoded_pumpkins['ord__Item Size']

    g = sns.catplot(
        data=pumpkins,
        x="Item Size", y="Color", row='Variety',
        kind="box", orient="h",
        sharex=False, margin_titles=True,
        height=1.8, aspect=4, palette=palette,
    )
    g.set(xlabel="Item Size", ylabel="").set(xlim=(0,6))
    g.set_titles(row_template="{row_name}")
```

### бһ”бҹ’бһҡбһҫбһҹбҹ’бһңбһ¶бһҳбһ”бҹ’бһҡбһ—бҹҒбһ‘бһӮбҹҶбһ“бһјбһҡбһ–бһҺбҹҢ

бһ–бҹ’бһҡбҹ„бҹҮ Color бһҮбһ¶бһҖбһ¶бһҸбҹҒбһҖбһјбһҡбһ–бһёбһҡ (бһ–бһҺбҹҢбһҹ бһ¬ бһҳбһ·бһ“) бһңбһ¶бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ '[бһңбһ·бһ’бһёбһ–бһ·бһҹбҹҒбһҹ](https://seaborn.pydata.org/tutorial/categorical.html?highlight=bar) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҳбһҫбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ'бҹ” бһҳбһ¶бһ“бһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҳбһҫбһӣбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҡбһ”бһҹбҹӢбһҖбһ¶бһҸбҹҒбһҖбһјбһҡбһ“бҹҒбҹҮбһҮбһ¶бһҳбһҪбһҷбһўбһҗбҹҒбһҡбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”

бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҫбһӣбһўбһҗбҹҒбһҡбһҮбһ¶бһўбҹҠбһ»бһҳбһҺбһ¶бһҳбһҪбһҷбһҡбһ¶бһ”бҹӢбһҳбһ»бһҒбһҮбһ¶бһҳбһҪбһҷбһӮбҹҶбһ“бһјбһҡ Seabornбҹ”

1. бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһӮбҹҶбһ“бһјбһҡбһҹбҹ’бһңбһ¶бһҳ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбҹ–

    ```python
    palette = {
    0: 'orange',
    1: 'wheat'
    }
    sns.swarmplot(x="Color", y="ord__Item Size", data=encoded_pumpkins, palette=palette)
    ```

**бһҹбһјбһҳбһ”бҹ’бһҡбһҷбҹҗбһҸбҹ’бһ“**: бһҖбһјбһҠбһҒбһ¶бһ„бһӣбһҫбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһ¶бһҡбһ–бҹ’бһҡбһҳбһ¶бһ“ бһ–бҹ’бһҡбҹ„бҹҮ seaborn бһҳбһ·бһ“бһўбһ¶бһ…бһ”бһ„бҹ’бһ бһ¶бһүбһ…бҹҶбһҺбһ»бһ…бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бҹ’бһҡбһҫбһ“бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһҖбҹ’бһ“бһ»бһ„бһӮбҹҶбһ“бһјбһҡбһҹбҹ’бһңбһ¶бһҳбһ”бһ¶бһ“бһ‘бҹҒбҹ” бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҳбһҪбһҷбһӮбһәбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ‘бҹҶбһ бҹҶбһҳбҹүбһ¶бһҖбһҜбһҖбһҹбһ¶бһҡ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ 'size'бҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҹбһјбһҳбһҠбһ№бһ„бһҗбһ¶ бһңбһ¶бһ“бһ№бһ„бһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһ‘бҹ…бһӣбһҫбһ—бһ¶бһ–бһўбһ¶бһ…бһўбһ¶бһ“бһ“бҹғбһӮбҹҶбһ“бһјбһҡбҹ”

> **рҹ§® бһ”бһ„бҹ’бһ бһ¶бһүбһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶**

> Logistic regression бһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһӮбҹҶбһ“бһ·бһҸ 'maximum likelihood' бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ [function sigmoid](https://wikipedia.org/wiki/Sigmoid_function)бҹ” 'Sigmoid Function' бһ“бҹ…бһӣбһҫбһӮбҹҶбһ“бһјбһҡбһҳбһҫбһӣбһ‘бҹ…бһҠбһјбһ…бһҮбһ¶бһҡбһјбһ” 'S'бҹ” бһңбһ¶бһ‘бһ‘бһҪбһӣбһҸбһҳбҹ’бһӣбҹғбһҳбһҪбһҷбһ бһҫбһҷбһ•бҹ’бһӮбһјбһҡбһ•бҹ’бһӮбһ„бһІбҹ’бһҷбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһ“бҹ…бһ…бһ“бҹ’бһӣбҹ„бҹҮ 0 бһ“бһ·бһ„ 1бҹ” бһҒбҹ’бһҹбҹӮбһңбһ¶бһҖбҹҸбһ бҹ…бһҗбһ¶ 'logistic curve'бҹ” бһҹбһҳбһёбһҖбһ¶бһҡбһҡбһјбһ”бһ“бҹҒбҹҮбһҳбһ¶бһ“бһҡбһјбһ”бһҡбһ¶бһ„бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–
>
> ![logistic function](../../../../translated_images/km/sigmoid.8b7ba9d095c789cf.webp)
>
> бһҠбҹӮбһӣ midpoint бһҡбһ”бһҹбҹӢ sigmoid бһҳбһ¶бһ“бһ“бҹ…бһ…бҹҶбһҺбһ»бһ… x = 0, L бһҮбһ¶бһҸбһҳбҹ’бһӣбҹғбһўбһҸбһ·бһ”бһҡбһҳбһ¶бһ“бҹғбһҒбҹ’бһҹбҹӮ, бһ“бһ·бһ„ k бһҮбһ¶бһҖбһҳбҹ’бһҡбһ·бһҸбһҖбһ¶бһ…бҹӢбһҮбҹ’бһҡбһ¶бһ”бһ“бҹғбһҒбҹ’бһҹбҹӮбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһӣбһ‘бҹ’бһ’бһ•бһӣбһ“бҹғ function бһӣбһҫбһҹ 0.5 бһҹбҹ’бһӣбһ¶бһҖбһ“бҹ„бҹҮбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ…бһ¶бһҸбҹӢбһҗбһ¶бһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘ '1' бһ“бҹғбһҮбһҳбҹ’бһҡбһҫбһҹбһ–бһёбһҡбҹ” бһ”бһҫбһҳбһ·бһ“бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ бһ“бһ№бһ„бһ…бһ¶бһҸбҹӢбһҗбһ¶бһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘ '0'бҹ”

## бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһҖбһ…бҹҶбһҺбһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ–бһёбһҡбһ“бҹҒбҹҮбһӮбһәбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„ Scikit-learnбҹ”

[![ML for beginners - Logistic Regression for classification of data](https://img.youtube.com/vi/MmZS2otPrQ8/0.jpg)](https://youtu.be/MmZS2otPrQ8 "ML for beginners - Logistic Regression for classification of data")

> рҹҺҘ бһ…бһ»бһ…бһӣбһҫбһҡбһјбһ”бһ—бһ¶бһ–бһҒбһ¶бһ„бһӣбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһёбһҠбҹҒбһўбһјбһҹбһ„бҹ’бһҒбҹҒбһ”бһҹбҹ’бһҸбһёбһ–бһёбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣ logistic regression

1. бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһўбһҗбҹҒбһҡбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бҹ’бһҡбһҫбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢ бһ бһҫбһҷбһ”бҹҶбһ”бҹӮбһҖ training бһ“бһ·бһ„ test бһҠбҹ„бһҷбһ бҹ… `train_test_split()`бҹ–

    ```python
    from sklearn.model_selection import train_test_split
    
    X = encoded_pumpkins[encoded_pumpkins.columns.difference(['Color'])]
    y = encoded_pumpkins['Color']

    X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=0)
    
    ```

2. бһҘбһЎбһјбһңбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһҺбҹ’бһҸбһ»бҹҮбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹ„бһҷбһ бҹ… `fit()` бһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ training бһ бһҫбһҷбһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбһ”бһҹбҹӢбһңбһ¶бҹ–

    ```python
    from sklearn.metrics import f1_score, classification_report 
    from sklearn.linear_model import LogisticRegression

    model = LogisticRegression()
    model.fit(X_train, y_train)
    predictions = model.predict(X_test)

    print(classification_report(y_test, predictions))
    print('Predicted labels: ', predictions)
    print('F1-score: ', f1_score(y_test, predictions))
    ```

    бһҳбһҫбһӣбһ‘бҹ…бһҖбҹ’бһҡбһЎбһ¶ scoreboard бһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбҹ” бһңбһ¶бһҳбһ·бһ“бһўбһ¶бһҖбҹ’бһҡбһ¶бһҸбһ‘бҹҒ бһ”бһҫбһӮбһ·бһҸбһҗбһ¶бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһ”бҹ’бһҡбһ бҹӮбһӣ 1000 бһҮбһҪбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ–

## бһҷбһӣбҹӢбһ”бһ¶бһ“бһӣбҹ’бһўбһҮбһ¶бһ„бһӮбҹҒбһҸбһ¶бһҳбһҡбһҷбҹҲ confusion matrix

бһӣбҹ„бһҖбһўбһ¶бһ…бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚ scoreboard бһҠбҹ„бһҷбһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһңбһҸбҹ’бһҗбһ»бһҒбһ¶бһ„бһӣбһҫ [terms](https://scikit-learn.org/stable/modules/generated/sklearn.metrics.classification_report.html?highlight=classification_report#sklearn.metrics.classification_report) бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҷбһӣбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ”бһ¶бһ“бһӣбһҝбһ“бһҮбһ¶бһ„бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ [confusion matrix](https://scikit-learn.org/stable/modules/model_evaluation.html#confusion-matrix) бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһӣбҹӢбһ–бһёбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ·бһ—бһ¶бһ–бһҳбҹүбһјбһҠбҹӮбһӣбҹ”

> рҹҺ“ '[confusion matrix](https://wikipedia.org/wiki/Confusion_matrix)' (бһ¬ 'error matrix') бһҮбһ¶бһҸбһ¶бһҡбһ¶бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ“бҹғбһҖбһ¶бһҡбһ‘бһ¶бһҷбһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһңбһ¶бһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ“бҹғбһ‘бһ¶бһҷбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бҹ”

1. бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫ confusion matrix, бһ бҹ… `confusion_matrix()`бҹ–

    ```python
    from sklearn.metrics import confusion_matrix
    confusion_matrix(y_test, predictions)
    ```

    бһҳбһҫбһӣбһ‘бҹ… confusion matrix бһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбҹ–

    ```output
    array([[162,   4],
           [ 11,  22]])
    ```

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Scikit-learn, бһҮбһҪбһҡбһҠбҹҒбһҖ (axis 0) бһҮбһ¶бһҹбҹ’бһӣбһ¶бһҖбһ–бһ·бһҸ бһ“бһ·бһ„бһҮбһҪбһҡбһҲбһҡ (axis 1) бһҮбһ¶бһҹбҹ’бһӣбһ¶бһҖбһ‘бһ¶бһҷбҹ”

|       |   0   |   1   |
| :---: | :---: | :---: |
|   0   |  TN   |  FP   |
|   1   |  FN   |  TP   |

бһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ? бһҷбһҫбһ„бһ“бһ·бһҷбһ¶бһҷбһҗбһ¶бһҳбҹүбһјбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҪбһҡбһІбҹ’бһҷбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢ pumpkin бһҡбһңбһ¶бһ„бһ”бҹүбһ¶бһҡбһ„бһ–бһёбһҡбһӮбһә 'бһ–бһҺбҹҢбһҹ' бһ“бһ·бһ„ 'бһҳбһ·бһ“бһ–бһҺбҹҢбһҹ'бҹ”

- бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбҹүбһјбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ‘бһ¶бһҷбһҗбһ¶ pumpkin бһҳбһ·бһ“бһҳбҹӮбһ“бһ–бһҺбҹҢбһҹ бһ“бһ·бһ„бһ“бҹ…бһ–бһ·бһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбһ¶бһҖбҹ’бһҡбһ»бһҳ 'бһҳбһ·бһ“бһ–бһҺбҹҢбһҹ', бһҷбһҫбһ„бһ бҹ…бһ“бҹҒбҹҮбһҗбһ¶ true negative бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҠбҹ„бһҷбһӣбҹҒбһҒбһҒбһ¶бһ„бһӣбһҫбһҶбҹ’бһңбҹҒбһ„бҹ”
- бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбҹүбһјбһҠбҹӮбһӣбһ‘бһ¶бһҷбһҗбһ¶ pumpkin бһҮбһ¶бһ–бһҺбҹҢбһҹ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ–бһ·бһҸбһҮбһ¶бһҖбҹ’бһ“бһ»бһ„бһҖбҹ’бһҡбһ»бһҳ 'бһҳбһ·бһ“бһ–бһҺбҹҢбһҹ' бһҷбһҫбһ„бһ бҹ…бһҗбһ¶ false negative бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҠбҹ„бһҷбһӣбҹҒбһҒбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҶбҹ’бһңбҹҒбһ„бҹ”
- бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбҹүбһјбһҠбҹӮбһӣбһ‘бһ¶бһҷбһҗбһ¶ pumpkin бһҳбһ·бһ“бһҳбҹӮбһ“бһ–бһҺбҹҢбһҹ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ–бһ·бһҸбһҮбһ¶ 'бһ–бһҺбҹҢбһҹ', бһҷбһҫбһ„бһ бҹ…бһҗбһ¶ false positive бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҠбҹ„бһҷбһӣбҹҒбһҒбһҒбһ¶бһ„бһӣбһҫбһҹбҹ’бһҸбһ¶бҹҶбҹ”
- бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбҹүбһјбһҠбҹӮбһӣбһ‘бһ¶бһҷбһҗбһ¶ pumpkin бһҮбһ¶бһ–бһҺбҹҢбһҹ бһ бһҫбһҷбһ–бһ·бһҸбһҮбһ¶ 'бһ–бһҺбҹҢбһҹ', бһҷбһҫбһ„бһ бҹ…бһҗбһ¶ true positive бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҠбҹ„бһҷбһӣбҹҒбһҒбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбҹ’бһҸбһ¶бҹҶбҹ”
бһҷбҹүбһ¶бһ„бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ¶бһ“бһӮбһ·бһҸ бһҳбһ»бһ“бһ“бҹҒбҹҮбһңбһ¶бһҮбһ¶бһҖбһ¶бһҡбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҳбһ¶бһ“бһ…бҹҶбһ“бһҪбһ“ TP (true positives) бһ“бһ·бһ„ TN (true negatives) бһ…бҹ’бһҡбһҫбһ“ бһ бһҫбһҷбһҳбһ¶бһ“бһ…бҹҶбһ“бһҪбһ“ FP (false positives) бһ“бһ·бһ„ FN (false negatives) бһҸбһ·бһ…  бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶бһҳбҹүбһјбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһӣбҹ’бһўбһҮбһ¶бһ„бҹ”

бһҸбһҫ matrix бһҡбһүбҹ’бһҮбһҪбһҷбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ‘бҹ…бһ“бһ№бһ„ precision бһ“бһ·бһ„ recall бһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…? бһ…бһјбһҡбһ…бһ„бһ…бһ¶бҹҶбһҗбһ¶ бһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһ“бҹ…бһӣбһҫ бһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһё precision (0.85) бһ“бһ·бһ„ recall (0.67)бҹ”

Precision = tp / (tp + fp) = 22 / (22 + 4) = 0.8461538461538461

Recall = tp / (tp + fn) = 22 / (22 + 11) = 0.6666666666666666

вң… бһҹбҹҶбһ“бһҪбһҡбҹ– бһҷбҹ„бһ„бһҸбһ¶бһҳ matrix бһҡбһүбҹ’бһҮбһҪбһҷ бһҳбҹүбһјбһҠбҹӮбһӣбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҠбҹҒбһ…? бһҠбһ¶бһ…бҹӢбһ…бһ·бһҸбҹ’бһҸбҹ– бһҳбһ·бһ“бһўбһ¶бһҖбҹ’бһҡбһҖбҹӢбһ‘бҹҒ; бһҳбһ¶бһ“бһ…бҹҶбһ“бһҪбһ“ true negatives бһ…бҹ’бһҡбһҫбһ“бһӣбҹ’бһў бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ“бҹ…бһҸбҹӮбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“ false negatives бһ•бһ„бһҠбҹӮбһҡбҹ”

бһ…бһјбһӣбһ‘бҹ…бһңбһ·бһүбһңбһ·бһӣбһ‘бһҹбҹ’бһҹбһ“бһ¶ бһӣбһҖбҹ’бһҒбһҺбҹҲбһҠбҹӮбһӣбһҷбһҫбһ„бһ”бһ¶бһ“бһғбһҫбһүбһ–бһёбһҳбһ»бһ“ бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҮбҹҶбһ“бһҪбһҷбһҖбҹҶбһҺбһҸбҹӢбһ‘бһёбһҸбһ¶бҹҶбһ„ TP/TN бһ“бһ·бһ„ FP/FN бһ“бҹғ matrix бһҡбһүбҹ’бһҮбһҪбһҷбҹ–

рҹҺ“ Precision: TP/(TP + FP) бһ—бһ¶бһӮбһҡбһҷбһ“бҹғбһҖбһҡбһҺбһёбһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбҹ„бһҳбһҖбһҡбһҺбһёбһҠбҹӮбһӣбһ”бһ¶бһ“бһҷбһҖ (бһ§. бһҹбҹ’бһӣбһ¶бһҖбһҺбһ¶бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбҹ’бһӣбһ¶бһҖбһ”бһ¶бһ“бһӣбҹ’бһў)

рҹҺ“ Recall: TP/(TP + FN) бһ—бһ¶бһӮбһҡбһҷбһ“бҹғбһҖбһҡбһҺбһёбһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„ бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҷбһҖ бһҳбһ·бһ“бһҗбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбҹ’бһӣбһ¶бһҖбһ”бһ¶бһ“бһӣбҹ’бһўбһ¬бһўбһҸбҹӢ

рҹҺ“ f1-score: (2 * precision * recall)/(precision + recall) бһҳбһ’бҹ’бһҷбһҳ бһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–бһ“бҹғ precision бһ“бһ·бһ„ recall бһҠбҹӮбһӣбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһӮбһә 1 бһ“бһ·бһ„бһўбһ¶бһҖбҹ’бһҡбһҖбҹӢбһ”бҹҶбһ•бһ»бһҸбһӮбһә 0

рҹҺ“ Support: бһ…бҹҶбһ“бһҪбһ“бһҖбһҡбһҺбһёбһ“бҹғбһҹбҹ’бһӣбһ¶бһҖбһ“бһёбһҳбһҪбһҷбҹ—бһҠбҹӮбһӣбһ”бһ¶бһ“бһҷбһҖ

рҹҺ“ Accuracy: (TP + TN)/(TP + TN + FP + FN) бһ—бһ¶бһӮбһҡбһҷбһ“бҹғбһҹбҹ’бһӣбһ¶бһҖбһҠбҹӮбһӣбһ”бһ¶бһ“бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹҶбһҡбһјбҹ”

рҹҺ“ Macro Avg: бһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһҳбһ’бҹ’бһҷбһҳбһ”бҹ’бһҡбһ¶бһҖбҹӢбһҳбһ·бһ“бһ”бһ„бҹ’бһӮбһ”бһ„бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһӣбһ¶бһҖбһ“бһёбһҳбһҪбһҷбҹ— бһҠбҹ„бһҷбһҳбһ·бһ“бһӮбһ·бһҸбһ–бһёбһ—бһ¶бһ–бһҳбһ·бһ“бһҹбһҳбһҡбһҳбҹ’бһҷбһ“бҹғбһҹбҹ’бһӣбһ¶бһҖбҹ”

рҹҺ“ Weighted Avg: бһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһҳбһ’бҹ’бһҷбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһӣбһ¶бһҖбһ“бһёбһҳбһҪбһҷбҹ— бһҠбҹ„бһҷбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһӣбһҫбһ—бһ¶бһ–бһҳбһ·бһ“бһҹбһҳбһҡбһҳбҹ’бһҷбһ“бҹғбһҹбҹ’бһӣбһ¶бһҖбһҠбҹ„бһҷбһңбһ¶бһҹбҹӢбһҸбҹҶбһӣбҹғбһҸбһ¶бһҳбһҖбһҳбҹ’бһҡбһ·бһҸбһҹбҹ’бһ“бҹҒбҹҮ (бһ…бҹҶбһ“бһҪбһ“бһҖбһҡбһҺбһёбһҹбһ–бҹ’бһңбһҗбҹ’бһ„бҹғбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһӣбһ¶бһҖбһ“бһёбһҳбһҪбһҷбҹ—)бҹ”

вң… бһҸбһҫбһўбҹ’бһ“бһҖбһӮбһ·бһҸбһҗбһ¶бһҸбһҳбҹ’бһӣбҹғбһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһ“бҹ…бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһІбҹ’бһҷбһҳбҹүбһјбһҠбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бһ“бҹ’бһҗбһҷбһ…бҹҶбһ“бһҪбһ“ false negatives?

## бһ”бһ„бҹ’бһ бһ¶бһүбһҡбһ¶бһ„бһҖбҹ„бһ„ ROC бһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ“бҹҒбҹҮ

[![ML for beginners - Analyzing Logistic Regression Performance with ROC Curves](https://img.youtube.com/vi/GApO575jTA0/0.jpg)](https://youtu.be/GApO575jTA0 "ML for beginners - Analyzing Logistic Regression Performance with ROC Curves")

> рҹҺҘ бһ…бһ»бһ…бһӣбһҫбһҡбһјбһ”бһ—бһ¶бһ–бһҒбһ¶бһ„бһӣбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһёбһҠбҹҒбһўбһјбһ…бҹҶбһӣбһ„бһҒбҹ’бһӣбһёбһўбҹҶбһ–бһёбһҡбһ¶бһ„бһҖбҹ„бһ„ ROC

бһҷбһҫбһ„бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһ”бһ„бҹ’бһ бһ¶бһүбһҳбһҪбһҷбһ‘бҹҖбһҸ бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһҡбһ¶бһ„бһҖбҹ„бһ„бһҠбҹӮбһӣбһ бҹ…бһҗбһ¶ вҖҳROCвҖҷбҹ–

```python
from sklearn.metrics import roc_curve, roc_auc_score
import matplotlib
import matplotlib.pyplot as plt
%matplotlib inline

y_scores = model.predict_proba(X_test)
fpr, tpr, thresholds = roc_curve(y_test, y_scores[:,1])

fig = plt.figure(figsize=(6, 6))
plt.plot([0, 1], [0, 1], 'k--')
plt.plot(fpr, tpr)
plt.xlabel('False Positive Rate')
plt.ylabel('True Positive Rate')
plt.title('ROC Curve')
plt.show()
```

бһ”бҹ’бһҡбһҫ Matplotlib бһҠбһҫбһҳбҹ’бһ”бһёбһӮбһјбһҡбһҡбһ¶бһ„бһҖбҹ„бһ„ [Receiving Operating Characteristic](https://scikit-learn.org/stable/auto_examples/model_selection/plot_roc.html?highlight=roc) бһ¬ ROC бһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбҹ” бһҡбһ¶бһ„бһҖбҹ„бһ„ ROC бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбһ”бһҹбҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһ…бҹҶбһҺбһ»бһ… positive бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„ false positivesбҹ” "бһҡбһ¶бһ„бһҖбҹ„бһ„ ROC бһҮбһ¶бһ’бһҳбҹ’бһҳбһҸбһ¶бһ”бһ„бҹ’бһ бһ¶бһүбһўбһҸбҹ’бһҡбһ¶ true positive бһ“бҹ…бһӣбһҫбһўбҹҗбһҖбҹ’бһҹ Y бһ бһҫбһҷ false positive бһ“бҹ…бһӣбһҫбһўбҹҗбһҖбҹ’бһҹ X"бҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ, бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбһҡбһ”бһҹбҹӢбһҡбһ¶бһ„бһҖбҹ„бһ„ бһ“бһ·бһ„бһ…бһ“бҹ’бһӣбҹ„бҹҮбһҡбһңбһ¶бһ„бһҒбҹ’бһҹбҹӮвҖӢбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһҺбҹ’бһҸбһ¶бһӣ бһ“бһ·бһ„бһҡбһ¶бһ„бһҖбҹ„бһ„бһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бһ№бһ„бҹ– бһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бһ¶бһ“бһҡбһ¶бһ„бһҖбҹ„бһ„бһҳбһҪбһҷбһҡбҹҶбһҖбһ·бһӣбһЎбһҫбһ„бһӣбһҝбһ“ бһ бһҫбһҷбһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһҒбҹ’бһҹбҹӮбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһҡбһ”бһҹбҹӢбһҷбһҫбһ„ there are false positives to start with, and then the line heads up and over properly:

![ROC](../../../../translated_images/km/ROC_2.777f20cdfc4988ca.webp)

бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ бһ”бҹ’бһҡбһҫ Scikit-learnвҖҷs [`roc_auc_score` API](https://scikit-learn.org/stable/modules/generated/sklearn.metrics.roc_auc_score.html?highlight=roc_auc#sklearn.metrics.roc_auc_score) бһҠбһҫбһҳбҹ’бһ”бһёбһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ 'бһ•бҹ’бһ‘бҹғбһҖбҹ’бһҡбҹ„бһҳбһҡбһ¶бһ„бһҖбҹ„бһ„' (AUC)бҹ–

```python
auc = roc_auc_score(y_test,y_scores[:,1])
```
