# бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҹбһ¶бһҷбһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣ ML

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣ MLбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһҖбһ–бһёбһ–бһ·бһ—бһ–бһӣбҹ„бһҖбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бҹ– _бһҖбһ¶бһҡбһҳбһҫбһӣбһғбһҫбһү UFO бһӣбһҫбһҡбһҷбҹҲбһ–бҹҒбһӣбһҳбһҪбһҷбһҹбһҸбһңбһҸбҹ’бһҹаё—аёөа№Ҳаёңа№ҲаёІаёҷаёЎаёІ_ бһҠбҹӮбһӣбһ”бһ¶бһ“бһҷбһҖбһ–бһёбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ NUFORCбҹ”

бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бҹ–

- бһҡбһ”бҹҖбһ” 'pickle' бһҳбҹүбһјбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ
- бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһ“бҹ„бҹҮбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Flask

бһҷбһҫбһ„бһ“бһ№бһ„бһ”бһ“бҹ’бһҠбһ”бҹ’бһҡбһҫбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҳбҹ’бһўбһ¶бһҸбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣбһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҷбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹҒбҹҮбһ‘бҹ…бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳ бһҠбҹ„бһҷбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣ 'бһ“бҹ…бһҸбһ¶бһҳбһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ·' бһ“бһ·бһҷбһ¶бһҷбһҳбҹ’бһ“бһ¶бһҠбһјбһ…бһҮбһ¶ бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҹбһ¶бһҷбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҹбһ¶бһҷбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Flaskбҹ”

## бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё

бһҳбһ¶бһ“бһңбһ·бһ’бһёбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҹбһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбҹҖбһ“бҹ” бһңбһ·бһ’бһ¶бһ“бһ¶бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһңбҹҒбһ”бһҹбһ¶бһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһ¶бһ“бһҘбһ‘бҹ’бһ’бһ·бһ–бһӣбһӣбһҫбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҳбҹүбһјбһҠбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбҹ” бһҹбһјбһҳбһ“бһ·бһҷбһ¶бһҷбһҗбһ¶бһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳбһҳбһҪбһҷ бһҠбҹӮбһӣбһҖбҹ’бһҡбһ»бһҳбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһ–бһҪбһҖбһӮбҹҒбһ…бһ„бҹӢбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһҪбһҷбҹ”

### бһ–бһ·бһ…бһ¶бһҡбһҺбһ¶

бһҳбһ¶бһ“бһҹбҹҶбһҺбһҪбһҡбһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҹбһҪбһҡбҹ–

- **бһңбһ¶бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҹбһ¶бһҷбһ¬бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бһӣбҹҗбһҸ?** бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бһӣбҹҗбһҸ бһ¬бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘ IoT бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ [TensorFlow Lite](https://www.tensorflow.org/lite/) бһ бһҫбһҷбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Android бһ¬ iOS бҹ”
- **бһҳбҹүбһјбһҠбҹӮбһӣбһ“бһ№бһ„бһ•бҹ’бһ‘бһ»бһҖбһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһҺбһ¶?** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ–бһ–бһҖбһ¬бһҖбҹ’бһ“бһ»бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢ ?
- **бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҜбһҖбһҸбһ¶бҹ”** бһҸбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҸбҹ’бһҡбһјбһңбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ“бҹ’бһҸбһҡбһ–бҹҒбһӣбһўбһҸбҹӢбһ…бһјбһӣбһ”бһҺбҹ’бһҠбһ¶бһүбһҡбһәбһ‘бҹҒ?
- **бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһўбҹ’бһңбһёбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣ?** бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹбһўбһ¶бһ…бһҳбһ¶бһ“бһҘбһ‘бҹ’бһ’бһ·бһ–бһӣбһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбҹ”
    - **бһ”бҹ’бһҡбһҫ TensorFlowбҹ”** бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ TensorFlow бһҮбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһўбҹҒбһҖбһјбһҹбһ»бһёбһҹбҹ’бһҸбҹӮбһҳбһ“бҹ„бҹҮбһ•бҹ’бһҸбһӣбҹӢбһҮбһјбһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ”бһҳбҹ’бһӣбҹӮбһ„бһҳбҹүбһјбһҠбҹӮбһӣ TensorFlow бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ [TensorFlow.js](https://www.tensorflow.org/js/)бҹ”
    - **бһ”бҹ’бһҡбһҫ PyTorchбҹ”** бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҠбһјбһ…бһҮбһ¶ [PyTorch](https://pytorch.org/), бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ“бһ¶бҹҶбһ…бҹҒбһүбһңбһ¶бһ‘бҹ…бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ [ONNX](https://onnx.ai/) (Open Neural Network Exchange) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ” JavaScript бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ [Onnx Runtime](https://www.onnxruntime.ai/)бһ”бһ¶бһ“бҹ” бһҮбһҳбҹ’бһҡбһҫбһҹбһ“бҹҒбҹҮбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ“бҹ…бһҳбҹҒбһҡбҹҖбһ“бһҒбһ¶бһ„бһҳбһ»бһҒбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҠбҹ„бһҷ Scikit-learnбҹ”

бһўбҹ’бһ“бһҖбһҖбҹҸбһҳбһ¶бһ“бһұбһҖбһ¶бһҹбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ” Flask бһ–бҹҒбһүбһӣбҹҒбһүбһҳбһҪбһҷбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһңбҹҒбһ”бһ•бһ„бһҠбҹӮбһҡбҹ” бһ“бҹҒбҹҮбһҖбҹҸбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ TensorFlow.js бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘ JavaScriptбҹ”

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһҠбҹ„бһҷбһҹбһ¶бһҡбһҷбһҫбһ„бһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„ Python, бһҷбһҫбһ„бһ“бһ№бһ„бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһёбһҮбҹҶбһ бһ¶бһ“бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫ бһҠбһҫбһҳбҹ’бһ”бһёбһ“бһ¶бҹҶбһ…бҹҒбһүбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһ–бһёбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶ бһ…бҹҒбһүбһ‘бҹ…бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһҠбҹӮбһӣбһўбһ¶бһ…бһўбһ¶бһ“бһ”бһ¶бһ“бһҠбҹ„бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҠбҹӮбһӣбһ”бһ¶бһ“бһҹбҹ’бһҸбһ¶бһ”бҹӢбһҠбҹ„бһҷ Pythonбҹ”

## бһ§бһ”бһҖбһҡбһҺбҹҚ

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ§бһ”бһҖбһҡбһҺбҹҚбһ–бһёбһҡбһӮбһә Flask бһ“бһ·бһ„ Pickle бһҠбҹӮбһӣбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһҡбһҸбҹӢбһӣбһҫ Pythonбҹ”

вң… Flask бһӮбһәбһҮбһ¶бһўбҹ’бһңбһё? бһ“бһ·бһҷбһ¶бһҷбһҗбһ¶ 'micro-framework' бһҠбҹ„бһҷбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһңбһ¶, Flask бһ•бҹ’бһҸбһӣбҹӢбһӣбһҖбҹ’бһҒбһҺбҹҲбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғ framework бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбҹҒбһ”бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Python бһ“бһ·бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ•бҹ’бһҹбҹҶбһ‘бҹҶбһ–бҹҗбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бҹҶбһ–бҹҗбһҡбһңбҹҒбһ”бҹ” бһҹбһјбһҳбһҳбһҫбһӣ [бһҳбһјбһҢбһ»бһӣбһҡбҹҖбһ“бһ“бҹҒбҹҮ](https://docs.microsoft.com/learn/modules/python-flask-build-ai-web-app?WT.mc_id=academic-77952-leestott) бһҠбһҫбһҳбҹ’бһ”бһёбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶бһҳбһҪбһҷ Flaskбҹ”

вң… Pickle бһӮбһәбһҮбһ¶бһўбҹ’бһңбһё? Pickle рҹҘ’ бһӮбһәбһҮбһ¶бһҳбҹүбһјбһҢбһ»бһӣ Python бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҒбһҡбҹҖбһӣбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһ”бһ„бҹ’бһңбһ·бһӣбһҹбҹҒбһҡбҹҖбһӣбһҖбһҳбҹ’бһҳбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ objects Pythonбҹ” бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖ 'pickle' бһҳбҹүбһјбһҠбҹӮбһӣ бһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһҹбҹҒбһҡбҹҖбһӣбһҖбһҳбҹ’бһҳ бһ¬бһ”бҹ’бһӣбһ¶бһҸбһҳбҹүбһјбһҠбҹӮбһӣбһ“бҹҒбҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһ“бҹ…бһӣбһҫбһңбҹҒбһ”бҹ” бһҹбһјбһҳбһ”бҹ’бһҡбһҷбҹҗбһҸбҹ’бһ“бҹ– pickle бһҳбһ·бһ“бһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһ‘бҹҒ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҹбһјбһҳбһ”бҹ’бһҡбһҷбҹҗбһҸбҹ’бһ“ бһ–бҹҒбһӣбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһң unzip бһ¬ 'un-pickle' бһҜбһҖбһҹбһ¶бһҡбҹ” бһҜбһҖбһҹбһ¶бһҡ pickle бһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„ .pklбҹ”

## бһӣбҹҶбһ бһ¶бһҸбҹӢ - бһҹбһҳбҹ’бһўбһ¶бһҸбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһҖбһ¶бһҡбһҳбһҫбһӣбһғбһҫбһү UFO бһҮбһ¶бһ„ 80,000 бһҠбһ„ бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһҳбһјбһӣбһЎбһҫбһ„бһҠбҹ„бһҷ [NUFORC](https://nuforc.org) (бһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣбһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚ UFO бһҮбһ¶бһҸбһ·)бҹ” бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹҒбҹҮбһҳбһ¶бһ“бһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҳбһҫбһӣ UFO бһҠбһјбһ…бһҮбһ¶бҹ–

- **бһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһңбҹӮбһ„бҹ”** "бһ”бһ»бһҡбһҹбһҳбҹ’бһ“бһ¶бһҖбҹӢбһЎбһҫбһ„бһ–бһёбһҖбһ¶бҹҶбһҡбһҹбҹ’бһҳбһёбһ—бҹ’бһӣбһәбһҳбһҪбһҷбһҠбҹӮбһӣбһ—бҹ’бһӣбһәбһӣбһҫбһҠбһёбһЎбһ„бҹӢбһҳбҹүбһ¶бһҹ бһҹбҹ’бһҡбҹӮбһ–бҹ’бһҡбҹғбһ“бҹ…бһҷбһ”бҹӢ бһ бһҫбһҷбһӮбһ¶бһҸбҹӢбһҡбһҸбҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҹбһҪбһ“бһ…бһҸбҹӢбһҷбһ¶бһ“бһҷбһ“бҹ’бһҸ Texas Instruments"бҹ”
- **бһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҒбҹ’бһӣбһёбҹ”** "бһ–бһ“бҹ’бһӣбһәбһ”бһ¶бһ“бһңбһ·бһӣбһҸбһ¶бһҳбһҷбһҫбһ„"бҹ”

1. бһ”бһҳбҹ’бһӣбҹӮбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ ufos бһ‘бҹ…бһҖбһ¶бһ“бҹӢ dataframe бһҸбһјбһ…бһҳбһҪбһҷбһҠбҹ„бһҷбһҳбһ¶бһ“бһ…бҹҶбһҺбһ„бһҮбһҫбһ„бһҗбҹ’бһҳбһёбҹ” бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҸбһҳбҹ’бһӣбҹғбһҠбһ¶бһ…бҹӢбһҒбһ¶бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһ¶бһӣ `Country` бҹ”

    ```python
    ufos = pd.DataFrame({'Seconds': ufos['duration (seconds)'], 'Country': ufos['country'],'Latitude': ufos['latitude'],'Longitude': ufos['longitude']})
    
    ufos.Country.unique()
    ```

1. бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ бһҠбҹ„бһҷбһ”бҹ„бҹҮбһ”бһ„бҹӢбһҸбһҳбҹ’бһӣбҹғ null бһ“бһ·бһ„бһ“бһ¶бҹҶбһ…бһјбһӣбһҸбҹӮбһ…бҹҶбһ“бһҪбһ“бһңбҹҒбһӣбһ¶бһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһңбһ¶бһ„ 1-60 бһңбһ·бһ“бһ¶бһ‘бһёбһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ–

    ```python
    ufos.dropna(inplace=True)
    
    ufos = ufos[(ufos['Seconds'] >= 1) & (ufos['Seconds'] <= 60)]
    
    ufos.info()
    ```

1. бһ“бһ¶бҹҶбһ…бһјбһӣбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ `LabelEncoder` бһҡбһ”бһҹбҹӢ Scikit-learn бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҳбҹ’бһӣбҹӮбһ„бһҸбһҳбҹ’бһӣбҹғбһўбһҖбҹ’бһҹбһҡбһҡбһ”бһҹбҹӢбһ”бҹ’бһҡбһ‘бҹҒбһҹбһ‘бҹ…бһҮбһ¶бһӣбҹҒбһҒбҹ–

    вң… LabelEncoder бһӮбһә encode бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢбһўбһҖбҹ’бһҒбһҡбһ¶бһңбһ·бһҡбһ»бһ‘бҹ’бһ’бҹ”

    ```python
    from sklearn.preprocessing import LabelEncoder
    
    ufos['Country'] = LabelEncoder().fit_transform(ufos['Country'])
    
    ufos.head()
    ```

    бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһ”бһ„бҹ’бһ бһ¶бһүбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

## бһӣбҹҶбһ бһ¶бһҸбҹӢ - бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҸбҹ’бһҡбҹҖбһҳбһҒбҹ’бһӣбһҪбһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣ бһҠбҹ„бһҷбһ”бҹҶбһ”бҹӮбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһҖбҹ’бһҡбһ»бһҳбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ бһ“бһ·бһ„бһҖбҹ’бһҡбһ»бһҳбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бҹ”

1. бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһӣбһҖбҹ’бһҒбһҺбҹҲбһ”бһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бһҺбҹ’бһҸбһ»бҹҮбһҮбһ¶бһҸбһҳбҹ’бһӣбҹғ X бһ бһҫбһҷбһҸбһҳбҹ’бһӣбҹғ y бһҠбһјбһ…бһҮбһ¶ `Country`бҹ” бһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһўбһ¶бһ…бһ”бһүбҹ’бһ…бһјбһӣ `Seconds`, `Latitude` бһ“бһ·бһ„ `Longitude` бһ бһҫбһҷбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһӣбҹҒбһҒбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ”бҹ’бһҡбһ‘бҹҒбһҹбһңбһ·бһүбҹ”

    ```python
    from sklearn.model_selection import train_test_split
    
    Selected_features = ['Seconds','Latitude','Longitude']
    
    X = ufos[Selected_features]
    y = ufos['Country']
    
    X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=0)
    ```

1. бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ logistic regressionбҹ–

    ```python
    from sklearn.metrics import accuracy_score, classification_report
    from sklearn.linear_model import LogisticRegression
    model = LogisticRegression()
    model.fit(X_train, y_train)
    predictions = model.predict(X_test)
    
    print(classification_report(y_test, predictions))
    print('Predicted labels: ', predictions)
    print('Accuracy: ', accuracy_score(y_test, predictions))
    ```

бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҳбһ·бһ“бһўбһ¶бһҖбҹ’бһҡбһҖбҹӢбһ‘бҹҒ **(бһ”бҹ’бһҡбһ бҹӮбһӣ 95%)** бһўбҹ’бһңбһёбһҠбҹӮбһӣбһӮбҹ’бһҳбһ¶бһ“бһ—бһ¶бһ–бһ—бҹ’бһүбһ¶бһҖбҹӢбһ•бҹ’бһўбһҫбһӣ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ `Country` бһ“бһ·бһ„ `Latitude/Longitude` бһҳбһ¶бһ“бһҹбһҳбһ¶бһҳбһ¶бһҸбҹ’бһҡбһ…бҹҶбһҡбһјбһ„бҹ”

бһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһҗбҹ’бһҳбһёбһӮбҹ’бһҡбһ”бҹӢа»ҒбһӣбһҖбһ‘бҹҒ бһ–бҹ’бһҡбҹ„бҹҮбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһўбһ„бҹ’бһҖбҹҒбһҸбһғбһҫбһүбһ”бҹ’бһҡбһ‘бҹҒбһҹбһ–бһё `Latitude` бһ“бһ·бһ„ `Longitude` бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ бһҸбҹӮбһңбһ¶бһҮбһ¶бһӣбҹҶбһ бһ¶бһҸбҹӢбһӣбҹ’бһўбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһ–бһёбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбһҫбһҳбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҹбһҳбҹ’бһўбһ¶бһҸбһ“бһ·бһ„бһ“бһ¶бҹҶбһ…бҹҒбһү бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһ“бҹҒбҹҮбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҹбһ¶бһҷбҹ”

## бһӣбҹҶбһ бһ¶бһҸбҹӢ - 'pickle' бһҳбҹүбһјбһҠбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһӮбһәбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶ _pickle_ бһҳбҹүбһјбһҠбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ! бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһңбһ¶ бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһјбһҠбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһҒбҹ’бһҹбҹӮбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһңбһ¶ _pickle_ бһҡбһҪбһ… бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ•бҹ’бһ‘бһ»бһҖбһҳбҹүбһјбһҠбҹӮбһӣ pickle бһ“бһ·бһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһңбһ¶бһҮбһ¶бһҳбһҪбһҷбһўбһ¶бһҡбҹҒ бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғ seconds, latitude бһ“бһ·бһ„ longitudeбҹ”

```python
import pickle
model_filename = 'ufo-model.pkl'
pickle.dump(model, open(model_filename,'wb'))

model = pickle.load(open('ufo-model.pkl','rb'))
print(model.predict([[50,44,-12]]))
```

## бһӣбҹҶбһ бһ¶бһҸбҹӢ - бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Flask

бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Flask бһҠбһҫбһҳбҹ’бһ”бһёбһ бҹ…бһҳбҹүбһјбһҠбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһҸбҹ’бһҡбһЎбһ”бҹӢбһ“бһјбһңбһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһҸбҹӮбһҳбһ¶бһ“бһҡбһјбһ”бһҡбһ¶бһ„бһҹбһҳбҹ’бһҡбһҹбҹӢбһҮбһ¶бһ„бһҳбһ»бһ“бҹ”

1. бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһҗбһҸбһҺбһ¶бһҳбһҪбһҷбһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮ **web-app** бһ“бҹ…бһҮбһ¶бһ”бҹӢбһҜбһҖбһҹбһ¶бһҡ _notebook.ipynb_ бһҠбҹӮбһӣбһҜбһҖбһҹбһ¶бһҡ _ufo-model.pkl_ бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбҹ”

1. бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһ“бҹ„бҹҮбһ”бһ„бҹ’бһҖбһҫбһҸбһҗбһҸбһ”бһёбһ‘бҹҖбһҸбҹ– **static** бһҠбҹӮбһӣбһҳбһ¶бһ“бһҗбһҸ **css** бһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„ бһ“бһ·бһ„ **templates**бҹ” бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһҳбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡ бһ“бһ·бһ„бһҗбһҸбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

1. бһҜбһҖбһҹбһ¶бһҡбһҠбҹҸбһҠбҹҶбһ”бһјбһ„бһҖбҹ’бһ“бһ»бһ„бһҗбһҸ _web-app_ бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһә **requirements.txt**бҹ” бһҠбһјбһ…бһҮбһ¶ _package.json_ бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё JavaScript, бһҜбһҖбһҹбһ¶бһҡбһ“бҹҒбҹҮбһҡбһ¶бһҷбһ”бһүбҹ’бһҮбһёбһҖбһ¶бһҡбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһҠбҹӮбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҮбһ¶ dependencyбҹ” бһҖбҹ’бһ“бһ»бһ„ **requirements.txt** бһ”бһүбҹ’бһ…бһјбһӣбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбҹ–

    бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбҹҖбһҳбһ”бҹ’бһҡбһҫбһҜбһҖбһҹбһ¶бһҡ python бһҠбҹӮбһӣбһҠбһ№бһҖбһ“бһ¶бҹҶбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣ бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбҹ–

1. бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ `app.py` бһ”бһ“бҹ’бһҗбҹӮбһҳбҹ–

    ```python
    import numpy as np
    from flask import Flask, request, render_template
    import pickle
    
    app = Flask(__name__)
    
    model = pickle.load(open("./ufo-model.pkl", "rb"))
    
    
    @app.route("/")
    def home():
        return render_template("index.html")
    
    
    @app.route("/predict", methods=["POST"])
    def predict():
    
        int_features = [int(x) for x in request.form.values()]
        final_features = [np.array(int_features)]
        prediction = model.predict(final_features)
    
        output = prediction[0]
    
        countries = ["Australia", "Canada", "Germany", "UK", "US"]
    
        return render_template(
            "index.html", prediction_text="Likely country: {}".format(countries[output])
        )
    
    
    if __name__ == "__main__":
        app.run(debug=True)
    ```

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘бһҳбһ»бһҒбһҡбһ”бһҡ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһүбһҗбһ¶бһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһӣбҹ’бһўбһӮбһәбһ‘бһ¶бһҳбһ‘бһ¶бһҡбһҮбһ¶бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҡбһңбһ¶бһ„бһҳбһ“бһ»бһҹбҹ’бһҹбһҠбҹӮбһӣбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһңбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ” бһ¬бһ…бһӣбҹҗбһҸбҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһҷбһҫбһ„ бһӮбһәбһҳбһ“бһ»бһҹбҹ’бһҹбһҳбҹ’бһ“бһ¶бһҖбҹӢбһӮбһҸбҹӢ бһӮбһәбһўбҹ’бһ“бһҖ!
