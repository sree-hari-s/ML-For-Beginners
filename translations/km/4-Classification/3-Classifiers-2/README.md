    - бһҷбһҫбһ„бһўбһ¶бһ…бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ вңЁ KNeighbors Classifier

Support-Vector clustering (SVC) бһӮбһәбһҮбһ¶бһҖбһјбһ“бһҒбҹ’бһӣбһҪбһ“бһҳбһҪбһҷбһ“бҹғбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡбһ§бһ”бһҖбһҡбһҺбҹҚбһҳбҹүбһ¶бһҹбһёбһ“бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· Support-Vector (бһҡбҹҖбһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһңбһ¶бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ)бҹ” бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҮбҹ’бһҡбһҫбһҹ `'kernel'` бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҳбҹ’бһҡбҹҒбһ…бһҗбһ¶бһҸбһҫбһ…бҹӮбһҸбһјбһ…Labelsбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бҹ” бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ `'C'` бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶ `'regularization'` бһҮбһ¶бһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҘбһ‘бҹ’бһ’бһ·бһ–бһӣбһ“бҹғбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбҹ” Kernel бһўбһ¶бһ…бһҮбһ¶бһҸбһҪбһўбҹ’бһ“бһҖбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ [бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“](https://scikit-learn.org/stable/modules/generated/sklearn.svm.SVC.html#sklearn.svm.SVC); бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ бһҷбһҫбһ„бһҖбҹҶбһҺбһҸбҹӢбһңбһ¶бһҗбһ¶ `'linear'` бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҗбһ¶бһҷбһҫбһ„бһ”бҹ’бһҡбһҫ linear SVCбҹ” Probability бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ№бһҳ `'false'`; бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ бһҷбһҫбһ„бһҖбҹҶбһҺбһҸбҹӢбһңбһ¶бһҗбһ¶ `'true'` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҳбһјбһӣбһҖбһ¶бһҡбһ”бҹүбһ¶бһ“бҹӢбһҹбҹ’бһҳбһ¶бһ“бһ”бҹ’бһҡбһјбһ”бһ¶бһ”бҹҠбһёбһӣбһёбһ‘бһёбҹ” бһҷбһҫбһ„бһҖбҹҶбһҺбһҸбҹӢ random state бһ‘бҹ…бһҮбһ¶ `'0'` бһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҶбһӣбҹ„бһ—бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһјбһ”бһ¶бһ”бҹҠбһёбһӣбһёбһ‘бһёбҹ”

### бһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҺ - бһўбһ“бһ»бһңбһҸбҹ’бһҸ Linear SVC

бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһ¶бһҡбҹүбҹҒ (array) бһ“бҹғбһ§бһ”бһҖбһҡбһҺбҹҚбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһүбҹ’бһ…бһјбһӣбһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһўбһ¶бһҡбҹүбҹҒбһ“бҹҒбҹҮ бһҠбҹ„бһҷбһҮбһ¶бһҖбһ¶бһҡбһ”бһ“бҹ’бһҸбһҮбһ¶бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ–бҹҒбһӣбһҠбҹӮбһӣбһҷбһҫбһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бҹ”

1. бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷ Linear SVCбҹ–

    ```python
    C = 10
    # бһ”бһ„бҹ’бһҖбһҫбһҸбһўбҹ’бһ“бһҖбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶бҹ”
    classifiers = {
        'Linear SVC': SVC(kernel='linear', C=C, probability=True,random_state=0)
    }
    ```

2. бһ”бһҺбҹ’бһҸбһ»бҹҮбһҳбҹүбһјбһҠбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһҳбһҪбһҷ Linear SVC бһ бһҫбһҷбһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҳбһҪбһҷбҹ–

    ```python
    n_classifiers = len(classifiers)
    
    for index, (name, classifier) in enumerate(classifiers.items()):
        classifier.fit(X_train, np.ravel(y_train))
    
        y_pred = classifier.predict(X_test)
        accuracy = accuracy_score(y_test, y_pred)
        print("Accuracy (train) for %s: %0.1f%% " % (name, accuracy * 100))
        print(classification_report(y_test,y_pred))
    ```

## бһ§бһ”бһҖбһҡбһҺбҹҚбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢ K-Neighbors

K-Neighbors бһӮбһәбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ“бҹғбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡбһ–бһёбһҡ вҖңneighborsвҖқ бһ“бҹғбһӮбһҳбҹ’бһҡбҹ„бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹ„бһҷбһҳбһ¶бһ“бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚ бһ“бһ·бһ„бһӮбҹ’бһҳбһ¶бһ“бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбҹ” бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ“бҹҒбҹҮ бһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹҶбһ“бһҪбһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ“бҹғбһ…бҹҶбһҺбһ»бһ…бһҳбһҪбһҷбһ бһҫбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҳбһјбһӣбһҮбһ»бҹҶбһңбһ·бһүбһ…бҹҶбһҺбһ»бһ…бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ…бһ‘бһ¶бһҷбһ”бһ¶бһ“бһҹбҹ’бһӣбһ¶бһҖбһ‘бһјбһ‘бҹ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

### бһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҺ - бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢ K-Neighbors

бһ§бһ”бһҖбһҡбһҺбҹҚбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҳбһ»бһ“бһӮбһәбһӣбҹ’бһў бһ бһҫбһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбҹ’бһўбһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһҷбһҫбһ„бһўбһ¶бһ…бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһӣбҹ’бһўбһҮбһ¶бһ„бһ“бҹҒбҹҮбһ‘бҹҖбһҸбҹ” бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢ K-Neighborsбҹ”

1. бһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһҪбһҷбһҮбһҪбһҡбһҠбһҠбҹӮбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһўбһ¶бһҡбҹүбҹҒбһ§бһ”бһҖбһҡбһҺбҹҚбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ (бһ”бһ“бҹ’бһҗбҹӮбһҳбһҒбҹ’бһ‘бһ„бҹӢбһҖбҹ’бһҡбҹ„бһҷбһҳбһ»бһҒбһҡбһ”бһҹбҹӢ Linear SVC)бҹ–

## Support Vector Classifier

Support-Vector classifiers бһӮбһәбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ“бҹғбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡбһҡбһ”бһҹбҹӢ [Support-Vector Machine](https://wikipedia.org/wiki/Support-vector_machine) бһ“бҹғбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢ бһ“бһ·бһ„бһҡбҹүбҹҒбһ бҹ’бһӮбҹ’бһҡбҹҒбһҹбҹ’бһҷбһ»бһ„бҹ” SVMs "бһ•бҹӮбһ“бһ‘бһёбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ…бҹҶбһҺбһ»бһ…бһҖбҹ’бһ“бһ»бһ„бһӣбҹҶбһ " бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһ“бһ…бһҳбҹ’бһ„бһ¶бһҷбһҡбһңбһ¶бһ„бһҖбҹ’бһҡбһ»бһҳбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ–бһёбһҡбҹ” бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹӮбһ“бһ‘бһёбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһӣбҹҶбһ бһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ…бһ‘бһ¶бһҷбһҮбһ¶бһҖбҹ’бһҡбһ»бһҳбһ”бҹ’бһҡбһ—бҹҒбһ‘бҹ”

бһҹбһјбһҳбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһӣбҹ’бһўбһҮбһ¶бһ„бһ“бҹҒбҹҮбһҮбһ¶бһҳбһҪбһҷ Support Vector Classifierбҹ”

- [Random Forest](https://scikit-learn.org/stable/modules/ensemble.html#forest), бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҮбһ¶бһҳбһ’бҹ’бһҷбһҳбһҳбһҪбһҷ бһ”бһ„бҹ’бһҖбһҫбһҸ 'бһ–бҹ’бһҡбҹғ' бһ“бҹғ 'бһҠбһҫбһҳбһҲбһҫбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸ' бһҠбҹӮбһӣбһ”бһүбҹ’бһ…бһјбһӣбһҖбһҡбһҺбһёбһ…бҹғбһҠбһ“бҹ’бһҷбһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҡбһҪбһҳбһҒбҹ’бһӣбһҪбһ“бҹ” бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ n_estimators бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ‘бҹ…бһ…бҹҶбһ“бһҪбһ“бһҠбһҫбһҳбһҲбһҫбҹ”

- [AdaBoost](https://scikit-learn.org/stable/modules/generated/sklearn.ensemble.AdaBoostClassifier.html) бһ”бһҺбҹ’бһҸбһ»бҹҮбһ§бһ”бһҖбһҡбһҺбҹҚбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ‘бҹ…бһҮбһ¶бһӣбҹҶбһ“бһ¶бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһ…бһҳбҹ’бһӣбһ„бһ“бҹғбһ§бһ”бһҖбһҡбһҺбҹҚбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ“бҹ„бҹҮбһ‘бҹ…бһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбһҠбҹӮбһӣбҹ” бһңбһ¶бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ‘бһҳбҹ’бһ„бһ“бҹӢбһ“бҹғбһ’бһ¶бһҸбһ»бһҠбҹӮбһӣбһ”бһ¶бһ“бһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ‘бһ¶бһ“бҹӢбһҸбҹ’бһҡбһјбһңбһҒбһ»бһҹ бһ бһҫбһҷбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹӮбһҸбһҳбҹ’бһҡбһјбһңбҹ”
