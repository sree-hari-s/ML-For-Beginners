# бһҖбһ¶бһҡбһ‘бһ¶бһҷбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҮбһ¶бһҳбһҪбһҷ ARIMA

бһ“бҹ…бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҳбһ»бһ“ бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ“бһ–бһёбһҖбһ¶бһҡбһ‘бһ¶бһҷбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶ бһ бһҫбһҷбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһ“бҹғбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһўбһӮбҹ’бһӮбһ·бһҹбһ“бһёбһҮбһ¶бһҳбһҪбһҷбһҡбһҷбҹҲбһ–бҹҒбһӣбҹ”

[![Introduction to ARIMA](https://img.youtube.com/vi/IUSk-YDau10/0.jpg)](https://youtu.be/IUSk-YDau10 "Introduction to ARIMA")

> рҹҺҘ бһ…бһ»бһ…бһҡбһјбһ”бһ—бһ¶бһ–бһҒбһ¶бһ„бһӣбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһёбһҠбҹҒбһўбһјбҹ– бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҒбҹ’бһӣбһёбһўбҹҶбһ–бһёбһҳбҹүбһјбһҠбҹӮбһӣ ARIMAбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһҖбҹ’бһ“бһ»бһ„ R бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһӮбҹҶбһ“бһ·бһҸбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһӮбһәбһ‘бһјбһ‘бҹ…бҹ”

## [бһ–бһ·бһ“бҹ’бһ‘бһ»бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҳбһ»бһ“бһҗбҹ’бһ“бһ¶бһҖбҹӢбһ”бһ„бҹ’бһҡбҹҖбһ“](https://ff-quizzes.netlify.app/en/ml/)

## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһёбһңбһ·бһ’бһёбһ–бһ·бһҹбҹҒбһҹбһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣбһҮбһ¶бһҳбһҪбһҷ [ARIMA: *A*uto*R*egressive *I*ntegrated *M*oving *A*verage](https://wikipedia.org/wiki/Autoregressive_integrated_moving_average)бҹ” бһҳбҹүбһјбһҠбҹӮбһӣ ARIMA бһҹбһҳ Д§afnaбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһү [non-stationarity](https://wikipedia.org/wiki/Stationary_process)бҹ”

## бһӮбҹҶбһ“бһ·бһҸбһ‘бһјбһ‘бҹ…

бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ ARIMA бһҳбһ¶бһ“бһӮбҹҶбһ“бһ·бһҸбһҒбҹ’бһӣбҹҮбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҠбһ№бһ„бҹ–

- рҹҺ“ **Stationarity**бҹ” бһ–бһёбһҡбһҷбҹҲбһ–бҹҒбһӣбһҹбҹ’бһҗбһ·бһҸбһ· бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– stationarity бһӮбһәбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ–бһёбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷбһҳбһ·бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ“бҹ…бһ–бҹҒбһӣбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ–бҹҒбһӣбҹ” бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһҳбһ·бһ“бһҳбҹӮбһ“ stationarity бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһ•бҹ’бһӣбһјбһңбһ“бҹғбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһңбһ·бһ—бһ¶бһӮбһ”бһ¶бһ“бҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһҡбһҠбһјбһңбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҡбһҮбһ¶бһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ бһҫбһҷбһўбһ¶бһ…бһҠбһҖбһ…бҹҒбһүбһ”бһ¶бһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ…бһҗбһ¶ 'seasonal-differencing'бҹ”

- рҹҺ“ **[Differencing](https://wikipedia.org/wiki/Autoregressive_integrated_moving_average#Differencing)**бҹ” бһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (differencing) бһ–бһёбһҖбһ¶бһҡбһҹбҹ’бһҗбһ·бһҸбһ·бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бһҷбҹҚбһҳбһ·бһ“бһҳбҹӮбһ“ stationarity бһ‘бҹ…бһҮбһ¶ stationarity бһҠбҹ„бһҷбһҷбһҖбһ…бҹҒбһүбһ–бһёбһ•бҹ’бһӣбһјбһңбһҠбҹӮбһӣбһҳбһ·бһ“бһҳбҹүбҹ„бһ„бһҗбҹҒбһҡбҹ” "Differencing бһ”бҹҶбһ”бһ¶бһҸбҹӢбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҡбһҷбҹҲбһ–бҹҒбһӣ бһ бһҫбһҷбһҠбһҖбһ…бҹҒбһүбһ•бҹ’бһӣбһјбһңбһ“бһ·бһ„бһҡбһҠбһјбһңбһ”бһҖбҹ’бһҹ бһҠбһјбһ…бһ“бҹҒбҹҮбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ–бһёбһҡбһҷбҹҲбһ–бҹҒбһӣбһ“бҹ„бҹҮбһҳбһ¶бһ“бһҳбһ’бҹ’бһҷбһҳбһҗбҹҒбһҡбҹ”" [бһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“ Shixiong et al](https://arxiv.org/abs/1904.07632)

## ARIMA бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘бһ“бҹғбһҲбһ»бһҸбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶

бһҳбһҖбһңбһ·бһүбһ–бһ“бҹ’бһҷбһӣбҹӢбһ•бҹ’бһ“бҹӮбһҖбһҡбһ”бһҹбҹӢ ARIMA бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһӣбҹӢбһ”бһ¶бһ“бһӣбҹ’бһўбһҗбһ¶бһңбһ¶бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҷбһҫбһ„бһҹбһ·бһҖбҹ’бһҹбһ¶ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһ‘бһ¶бһҷбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ“бһ·бһ„бһҮбһҪбһҷбһҷбһҫбһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһ‘бһ¶бһҷбҹ”

- **AR - бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ AutoRegressive**бҹ” бһҳбҹүбһјбһҠбҹӮбһӣ autoregressive бһҠбһјбһ…бһҲбҹ’бһҳбҹ„бҹҮбһңбһ¶ бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһҳбһҫбһӣбһҸбҹ’бһҡбһЎбһ”бҹӢбһҖбҹ’бһҡбҹ„бһҷбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶ бһҠбһҫбһҳбҹ’бһ”бһёбһңбһ·бһ—бһ¶бһӮбһҸбһҳбҹ’бһӣбҹғбһҳбһ»бһ“бҹ—бһҖбҹ’бһ“бһ»бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһ“бҹ’бһҳбһҸбҹӢбһўбҹҶбһ–бһёбһңбһ¶бҹ” бһҸбһҳбҹ’бһӣбҹғбһҳбһ»бһ“бҹ—бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ бҹ…бһҗбһ¶ 'lags'бҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһӮбһәбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһӣбһҖбҹӢбһҒбҹ’бһҳбҹ…бһҒбҹ’бһҳбҹ…бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҒбҹӮбҹ” бһҸбһҳбҹ’бһӣбҹғбһӣбһҖбҹӢбһӮбҹ’бһҡбһ”бҹӢбһҒбҹӮбһӮбһәбһҮбһ¶бһ•бһӣбһ”бһ“бҹ’бһӣбһ¶бһҹбҹӢбһҖбһ¶бһҡбһ…бһӣбҹҗбһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҲбһ»бһҸбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” бһҳбҹүбһјбһҠбҹӮбһӣбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷбһҖбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„ вҖңбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бһ“бҹ’бһӣбһ¶бһҹбҹӢбһ“бҹғбһўбһҗбҹҒбһҡбһҠбҹӮбһӣбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҮбһ¶бһҳбһҪбһҷбһӣбһҫбһҸбһҳбҹ’бһӣбҹғ lag бһҳбһҪбһҷбһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“бҹ”вҖқ [wikipedia](https://wikipedia.org/wiki/Autoregressive_integrated_moving_average)

- **I - бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Integrated**бҹ” бһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһҳбҹүбһјбһҠбҹӮбһӣ ARMA бһҹбҹ’бһҡбһҠбҹҖбһ„бһӮбҹ’бһ“бһ¶ бһўбһ„бҹӢбһӮбҹ’бһӣбҹҒбһҒбһ¶бһ„ I бһҖбҹ’бһ“бһ»бһ„ ARIMA бһҹбҹҶбһҠбҹ…бһӣбһҫбһ•бҹ’бһ“бҹӮбһҖ *[integrated](https://wikipedia.org/wiki/Order_of_integration)* бһҡбһ”бһҹбҹӢбһңбһ¶бҹ” бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“ 'integrated' бһ“бҹ…бһ–бҹҒбһӣбһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣ differencing бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбһҫбһҳбҹ’бһ”бһёбһҠбһҖбһ…бҹҒбһү non-stationarityбҹ”

- **MA - бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Moving Average**бҹ” бһ•бҹ’бһ“бҹӮбһҖ [moving-average](https://wikipedia.org/wiki/Moving-average_model) бһ“бҹғбһҳбҹүбһјбһҠбҹӮбһӣбһ“бҹҒбҹҮ бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһўбһҗбҹҒбһҡбһ…бҹҒбһүбһҠбҹӮбһӣбһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһҖбһ¶бһҡбһҹбһ„бҹ’бһҖбҹҒбһҸбһҸбһҳбҹ’бһӣбҹғбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ бһ“бһ·бһ„бһҳбһ»бһ“бҹ—бһ“бҹғ lagбҹ”

бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ– ARIMA бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҳбҹүбһјбһҠбҹӮбһӣбһҹбһҳбһҹбҹ’бһҡбһ”бһҸбһ¶бһҳбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ–бһ·бһҹбҹҒбһҹбһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸбҹ”

    > бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбҹ– бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮбһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһҷбһҫбһ„бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҳбһҳбһјбһӣбһҳбҹүбһјбһҠбҹӮбһӣ ARIMA бһўбһ“бһ»бһңбһҸбҹ’бһҸ validation бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӣбһҫбһҖбһҠбҹҶбһ”бһјбһ„ (in-sample) бһҖбҹ’бһҡбҹ„бһҳбһҖбһ¶бһҡбһҹбһҳбһҳбһјбһӣ бһҷбһҫбһ„бһ“бһ№бһ„бһҳбһ·бһ“бһ”бҹ’бһҡбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ pour validationбҹ”

### бһўбһ“бһ»бһңбһҸбҹ’бһҸ ARIMA

бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ‘бһҹбҹ’бһҹбһ“бһ¶ ARIMA! бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ `statsmodels` бһҠбҹӮбһӣбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бһҳбһ»бһ“бҹ”

бһҘбһЎбһјбһңбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбҹҶбһ бһ¶бһ“бһҒбҹ’бһӣбҹҮбҹ—бҹ–

   1. бһҖбҹҶбһҺбһҸбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹ„бһҷбһ бҹ… `SARIMAX()` бһ бһҫбһҷбһ”бһүбҹ’бһҮбһјбһ“бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбҹ– p, d, бһ“бһ·бһ„ q бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„ P, D, бһ“бһ·бһ„ Q бҹ”
   2. бһҡбҹҖбһ”бһ…бҹҶбһҳбҹүбһјбһҠбҹӮбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҠбҹ„бһҷбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ fit()бҹ”
   3. бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ‘бһ¶бһҷбһҠбҹ„бһҷбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ `forecast()` бһ“бһ·бһ„бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ…бҹҶбһ“бһҪбһ“бһҮбҹҶбһ бһ¶бһ“ (horizon) бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ‘бһ¶бһҷбҹ”

> рҹҺ“ бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҳбһ¶бһ“бһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮ? бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһјбһҠбҹӮбһӣ ARIMA бһҳбһ¶бһ“бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ…бҹҶбһ“бһҪбһ“ 3 бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһҳбһҳбһјбһӣбһҡбһҝбһ„бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҖбҹ’бһ“бһ»бһ„бһҲбһ»бһҸбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бҹ– бһҡбһҠбһјбһңбһҖбһ¶бһӣ, бһ•бҹ’бһӣбһјбһң, бһ“бһ·бһ„бһҹбҹҶбһЎбҹҒбһ„бҹ” бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһӮбһәбҹ–

`p`: бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ“бҹӮбһҖ autoregressive бһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҸбһҳбҹ’бһӣбҹғ *бһ–бһёбһҳбһ»бһ“*бҹ”
`d`: бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ“бҹӮбһҖ integrated бһҠбҹӮбһӣбһҳбһ¶бһ“бһҘбһ‘бҹ’бһ’бһ·бһ–бһӣбһӣбһҫбһҖбһҳбҹ’бһҡбһ·бһҸбһ“бҹғ *differencing* (рҹҺ“ бһ…бһ„бһ…бһ¶бҹҶ differencing рҹ‘Ҷ?) бһҠбҹӮбһӣбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбһҫбһҲбһ»бһҸбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бҹ”
`q`: бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ“бҹӮбһҖ moving-average бҹ”

> бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбҹ– бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһҡбһҠбһјбһңбһҖбһ¶бһӣ - бһҠбҹӮбһӣбһ“бҹҒбҹҮбһ’бҹ’бһңбһҫ - бһҷбһҫбһ„бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣ ARIMA бһҡбһҠбһјбһң (SARIMA)бҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҗбҹ’бһҳбһё P, D, бһ“бһ·бһ„ Q бһҠбҹӮбһӣбһҹбһҳбһ“бһ№бһ„ p, d, бһ“бһ·бһ„ q бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ”бһ„бҹ’бһ бһ¶бһүбһӣбһҖбҹ’бһҒбһҺбҹҲбһҡбһҠбһјбһңбһҖбһ¶бһӣбҹ”

1. бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғ horizon бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бһ¶бһ“бҹ” бһҳбһҖбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ 3 бһҳбҹүбҹ„бһ„бҹ–

    ```python
    # бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ…бҹҶбһ“бһҪбһ“бһҮбҹҶбһ бһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ¶бһҷбһўбһ“бһ¶бһӮбһҸ
    HORIZON = 3
    print('Forecasting horizon:', HORIZON, 'hours')
    ```

    бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҸбһҳбҹ’бһӣбҹғбһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ ARIMA бһўбһ¶бһ…бһҮбһ¶бһҖбһ¶бһҡбһӣбҹҶбһ”бһ¶бһҖ бһ–бҹ’бһҡбҹ„бҹҮбһңбһ¶бһҳбһ¶бһ“бһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһ“бһ·бһ„бһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҳбһ»бһҒбһ„бһ¶бһҡ `auto_arima()` бһ–бһёбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ [`pyramid`](https://alkaline-ml.com/pmdarima/0.9.0/modules/generated/pyramid.arima.auto_arima.html)бҹ”

1. бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҒбһӣбһ“бҹҒбҹҮ бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҠбҹ„бһҷбһҠбҹғбһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҳбҹүбһјбһҠбҹӮбһӣбһӣбҹ’бһўбһҳбһҪбһҷбҹ”

    ```python
    order = (4, 1, 0)
    seasonal_order = (1, 1, 0, 24)

    model = SARIMAX(endog=train, order=order, seasonal_order=seasonal_order)
    results = model.fit()
    ```

бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹҶбһ”бһјбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ! бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһңбһ·бһ’бһёбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһңбһ¶бҹ”

### бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҳбҹүбһјбһҠбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

бһҠбһҫбһҳбҹ’бһ”бһёбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҳбҹүбһјбһҠбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһўбһ“бһ»бһңбһҸбҹ’бһҸ `walk forward` validationбҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ бһҳбҹүбһјбһҠбҹӮбһӣбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһЎбһҫбһ„бһңбһ·бһүбһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһҳбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹ’бһҳбһёбҹ” бһңбһ¶бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҳбҹүбһјбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһ‘бһ¶бһҷбһ”бһ¶бһ“бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҮбҹҶбһ бһ¶бһ“бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бҹ”

бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһҠбһҫбһҳбһҲбһ»бһҸбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһ“бҹҒбҹҮ бһ”бһҺбҹ’бһҸбһ»бҹҮбһҳбҹүбһјбһҠбҹӮбһӣбһӣбһҫбһҲбһ»бһҸбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ‘бһ¶бһҷбһ“бҹ…бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ” бһҖбһ¶бһҡбһ‘бһ¶бһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ”бһ„бҹ’бһҖбһ”бҹӢбһҸбһ¶бһҳбһҸбһҳбҹ’бһӣбҹғбһ–бһ·бһҸбҹ” бһҲбһ»бһҸбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһ”бһ“бҹ’бһҸбһ–бһ„бҹ’бһҡбһёбһҖбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҸбһҳбҹ’бһӣбҹғбһ–бһ·бһҸ бһ бһҫбһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһЎбһҫбһ„бһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбҹ”

> бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбҹ– бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһҡбһҖбҹ’бһҹбһ¶бһ”бһ„бҹ’бһўбһҪбһ…бһҲбһ»бһҸбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһ“бҹ…бһҳбһ·бһ“бһ”бҹ’бһҠбһјбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһҹбһ„бҹ’бһҖбҹҒбһҸбһҗбҹ’бһҳбһёбһ…бһјбһӣбһ‘бҹ… бһӮбһҪбһҡбһҷбһҖбһҖбһ¶бһҡбһҹбһ„бҹ’бһҖбҹҒбһҸбһ…бһ¶бһҹбҹӢбһ…бҹҒбһүбһ–бһёбһҠбһҫбһҳбһҲбһ»бһҸбҹ”

бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһҖбһ¶бһҡбһ”бҹүбһ¶бһ“бҹӢбһ”бҹ’бһҡбһҳбһ¶бһҺбһҡбһ№бһ„бһҳбһ¶бҹҶбһҗбһ¶бһҳбҹүбһјбһҠбҹӮбһӣбһ“бһ№бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһҠбҹ„бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһ…бҹҶбһҺбһ¶бһҷбһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһ–бҹ„бҹҮбһ”бһ»бһ–бҹ’бһңбһҮбҹҶбһ“бһ¶бһүбҹ” бһңбһ¶бһҮбһ¶бһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһҷбһҖбһ”бһ¶бһ“ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбһјбһ… бһ¬бһҳбҹүбһјбһҠбҹӮбһӣбһҹбһ¶бһҳбһүбҹ’бһү бһ”бҹүбһ»бһ“бҹ’бһҸбһ¶бһўбһ¶бһ…бһҮбһ¶бһ”бһүбҹ’бһ бһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ’бҹҶбҹ”

Walk-forward validation бһӮбһәбһҮбһ¶бһҳбһ¶бһҸбҹ’бһҡбһҠбҹ’бһӢбһ¶бһ“бһҳбһ¶бһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҳбҹүбһјбһҠбҹӮбһӣбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶ бһ бһҫбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҺбҹӮбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһҳбҹ’бһҡбҹ„бһ„бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

1. бһҮбһ¶бһӣбһҫбһҖбһҠбҹҶбһ”бһјбһ„ бһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹҶбһҺбһ»бһ…бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһ¶бһӣбҹӢбһҮбҹҶбһ бһ¶бһ“ HORIZONбҹ”

    ```python
    test_shifted = test.copy()

    for t in range(1, HORIZON+1):
    ```

    ```python
    train_ts = train['load']
    test_ts = test_shifted

    history = [x for x in train_ts]
    history = history[(-training_window):]

    predictions = list()

    order = (2, 1, 0)
    seasonal_order = (1, 1, 0, 24)

    for t in range(test_ts.shape[0]):
        model = SARIMAX(endog=history, order=order, seasonal_order=seasonal_order)
        model_fit = model.fit()
        yhat = model_fit.forecast(steps = HORIZON)
        predictions.append(yhat)
        obs = list(test_ts.iloc[t])
        # бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһёбһ”бһ„бҹ’бһўбһҪбһ…бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ
        history.append(obs[0])
        history.pop(0)
        print(test_ts.index[t])
        print(t+1, ': predicted =', yhat, 'expected =', obs)
    ```

### бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҳбҹүбһјбһҠбҹӮбһӣ

бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹ„бһҷбһңбһ¶бһҹбҹӢ mean absolute percentage error (MAPE) бһӣбһҫбһҖбһ¶бһҡбһ‘бһ¶бһҷбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ”

> **рҹ§® бһ”бһ„бҹ’бһ бһ¶бһүбһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶**
>
> ![MAPE](../../../../translated_images/km/mape.fd87bbaf4d346846.webp)
>
>  [MAPE](https://www.linkedin.com/pulse/what-mape-mad-msd-time-series-allameh-statistics/) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ“бҹғбһҖбһ¶бһҡбһ‘бһ¶бһҷбһҮбһ¶бһӮбҹҶбһ“бһјбһҹбһҠбҹӮбһӣбһҖбҹҶбһҺбһҸбҹӢбһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҒбһ¶бһ„бһӣбһҫбҹ” бһ”бһҳбҹ’бһҡбҹӮбһ”бһҳбҹ’бһҡбһҪбһӣбһҡбһңбһ¶бһ„ actual<sub>t</sub> бһ“бһ·бһ„ predicted<sub>t</sub> бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹӮбһ„бһ…бҹӮбһҖбһҠбҹ„бһҷ actual<sub>t</sub>бҹ” "бһҸбһҳбҹ’бһӣбҹғ absolute бһ“бҹғбһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһјбһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһ¶бһӣбҹӢбһ…бҹҶбһҺбһ»бһ…бһҠбҹӮбһӣбһ”бһ¶бһ“бһ‘бһ¶бһҷ бһ бһҫбһҷбһ…бҹӮбһҖбһҸбһ¶бһҳбһ…бҹҶбһ“бһҪбһ“бһ…бҹҶбһҺбһ»бһ… fitted nбҹ”" [wikipedia](https://wikipedia.org/wiki/Mean_absolute_percentage_error)
1. Express equation in code:

    ```python
    if(HORIZON > 1):
        eval_df['APE'] = (eval_df['prediction'] - eval_df['actual']).abs() / eval_df['actual']
        print(eval_df.groupby('h')['APE'].mean())
    ```

1. Calculate one step's MAPE:

    ```python
    print('One step forecast MAPE: ', (mape(eval_df[eval_df['h'] == 't+1']['prediction'], eval_df[eval_df['h'] == 't+1']['actual']))*100, '%')
    ```

    One step forecast MAPE:  0.5570581332313952 %

1. Print the multi-step forecast MAPE:

    ```python
    print('Multi-step forecast MAPE: ', mape(eval_df['prediction'], eval_df['actual'])*100, '%')
    ```

    ```output
    Multi-step forecast MAPE:  1.1460048657704118 %
    ```

    A nice low number is best: consider that a forecast that has a MAPE of 10 is off by 10%.

    ```python
        eval_df.plot(x='timestamp', y=['actual', 'prediction'], style=['r', 'b'], figsize=(15, 8))

    else:
        ## бһӮбһјбһҹбһӮбҹҶбһ“бһјбһҹбһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһ бһ¶бһ“

            ax.plot(x, y, color='blue', linewidth=4*math.pow(.9,t), alpha=math.pow(0.8,t))
    ```
