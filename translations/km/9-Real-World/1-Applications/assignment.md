# ប្រកួតស្វែងរក ML

## សេចក្ដីណែនាំ

នៅក្នុងមេរៀននេះ អ្នកបានរៀនអំពីករណីការប្រើប្រាស់ជាក់ស្តែងជាច្រើនដែលត្រូវបានដោះស្រាយដោយប្រើ ML សាមញ្ញ។ ខណៈពេលដែលការប្រើប្រាស់ deep learning បច្ចេកវិទ្យាថ្មីៗ និងឧបករណ៍នៅក្នុង AI ហើយការចូលប្រើបណ្តាញច្រើនជាន់បានជួយអោយដំណើរការផលិតឧបករណ៍រហ័សឡើងដើម្បីជួយនៅវិស័យទាំងនេះ ML សាមញ្ញដោយប្រើបច្ចេកទេសក្នុងវគ្គសិក្សានេះនៅតែមានតំលៃខ្ពស់។

ក្នុងការងារផ្ដល់នេះ សូមសន្យាថាអ្នកកំពុងចូលរួមក្នុងការប្រកួត hackathon។ ប្រើអ្វីដែលអ្នកបានរៀនក្នុងវគ្គសិក្សា ដើម្បីផ្ដល់យោបល់ដំណោះស្រាយដោយប្រើ ML សាមញ្ញសម្រាប់ដោះស្រាយបញ្ហាមួយ នៅក្នុងវិស័យដែលបានពិភាក្សានៅក្នុងមេរៀននេះ។ បង្កើតសេចក្ដីបង្ហាញដែលអ្នកពិភាក្សាអំពីរបៀបដែលអ្នកនឹងអនុវត្តគំនិតរបស់អ្នក។ ពិន្ទុបន្ថែមប្រសិនបើអ្នកអាចចងក្រងទិន្នន័យគំរូ និងកសាងម៉ូដែល ML ដើម្បីគាំទ្រគំនិតរបស់អ្នក!
