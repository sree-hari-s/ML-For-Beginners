# ការរួមចំណែក

គម្រោងនេះស្វាគមន៍ការចូលរួម និងការផ្ដល់យោបល់។ ការចូលរួមភាគច្រើនតម្រូវឱ្យអ្នកយល់ព្រមលើសន្ធិសញ្ញាអ្នករួមចំណែក (CLA) ដែលប្រកាសថាអ្នកមានសិទ្ធិ ហើយពិតប្រាកដថា អ្នកផ្ដល់សិទ្ធិឱ្យយើងប្រើប្រាស់ការរួមចំណែករបស់អ្នក។ សម្រាប់ព័ត៌មានលម្អិត ទៅកាន់ https://cla.microsoft.com។

> សំខាន់៖ ពេលបកប្រែអត្ថបទនៅក្នុងហាងសម្រង់នេះ សូមប្រាកដថា អ្នកមិនប្រើការបកប្រែដោយម៉ាស៊ីនទេ។ យើងនឹងបញ្ជាក់ការបកប្រែតាមសហគមន៍ ដូច្នេះសូមចូលរួមបកប្រែតែភាសាដែលអ្នកមានជំនាញប៉ុណ្ណោះ។

ពេលអ្នកដាក់សំណើ pull request មួយ CLA-bot នឹងកំណត់ដោយស្វ័យប្រវត្តិនិងកំណត់ថាតើអ្នកត្រូវផ្ដល់ CLA ឬអត់ ហើយតុបតែង PR តាមដែលសមរម្យ (ឧ. ស្លាក, មតិ)។ គ្រាន់តែតាមដានសេចក្តីណែនាំដែលប៉ុស្តិ៍ផ្ដល់ឱ្យ។ អ្នកត្រូវធ្វើបែបនេះតែមួយដងតែមួយសម្រាប់គ្រប់ហាងសម្រង់ដែលប្រើ CLA របស់យើង។

គម្រោងនេះបានទទួលយក [Microsoft Open Source Code of Conduct](https://opensource.microsoft.com/codeofconduct/)។
សម្រាប់ព័ត៌មានបន្ថែម សូមមើល [Code of Conduct FAQ](https://opensource.microsoft.com/codeofconduct/faq/)
ឬទំនាក់ទំនង [opencode@microsoft.com](mailto:opencode@microsoft.com) សំរាប់សំណួរឬមតិយោបល់បន្ថែម។
