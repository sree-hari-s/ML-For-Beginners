      бһ“бҹҒбҹҮбһҮбһ¶бһ‘бһёбһҖбҹ’бһҡбһ»бһ„ бһ“бһ·бһ„бһ”бҹ’бһҡбһ‘бҹҒбһҹбһҸбҹӮбһҳбһҪбһҷбҹ—бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ–

      ```python
      def replace_address(row):
          if "Netherlands" in row["Hotel_Address"]:
      ```

2. бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһҪбһҡбһҲбһҡбһҹбһ„бҹ’бһҒбҹҒбһ”бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡ

  1. бһ”бҹ„бҹҮбһ”бһ„бҹӢ `Additional_Number_of_Scoring`

  1. бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ `Total_Number_of_Reviews` бһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһҹбһҡбһ»бһ”бһ“бҹғбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһ“бҹ„бҹҮбһҠбҹӮбһӣбһҳбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“

  1. бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ `Average_Score` бһҮбһ¶бһ–бһ·бһ“бҹ’бһ‘бһ»бһҒбҹ’бһӣбһҪбһ“бһҜбһ„бһҠбҹӮбһӣбһӮбһҺбһ“бһ¶бһЎбһҫбһ„

  ```python
  # бһҠбһҖ `Additional_Number_of_Scoring` бһ…бҹҒбһү
  df.drop(["Additional_Number_of_Scoring"], axis = 1, inplace=True)
  # бһҮбҹҶбһ“бһҪбһҹ `Total_Number_of_Reviews` бһ“бһ·бһ„ `Average_Score` бһҮбһ¶бһҳбһҪбһҷбһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһҷбһҫбһ„бһӮбһҺбһ“бһ¶бһЎбһҫбһ„бһҜбһ„
  df.Total_Number_of_Reviews = df.groupby('Hotel_Name').transform('count')
  df.Average_Score = round(df.groupby('Hotel_Name').Reviewer_Score.transform('mean'), 1)
  ```

бһҮбһҪбһҡбһҲбһҡ `Tag` бһӮбһәбһҳбһ¶бһ“бһ”бһүбҹ’бһ бһ¶ бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮбһңбһ¶бһҮбһ¶бһ”бһүбҹ’бһҮбһё (бһҖбҹ’бһ“бһ»бһ„бһ‘бҹҶбһҡбһ„бҹӢбһўбһҸбҹ’бһҗбһ”бһ‘) бһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһҮбһҪбһҡбһҲбһҡбҹ” бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһҹбһ„бҹ’бһғбһ№бһҳбһҗбһ¶бһӣбҹҶбһҠбһ¶бһ”бҹӢ бһ“бһ·бһ„бһ…бҹҶбһ“бһҪбһ“бһўбһҗбҹҒбһҡбһҡбһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҮбһҪбһҡбһҲбһҡбһ“бҹҒбҹҮбһҸбҹӮбһҳбһҪбһҷбһӮбҹ’бһ“бһ¶бһ‘бҹҒбҹ” бһңбһ¶бһ–бһ·бһ”бһ¶бһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһ“бһ»бһҹбҹ’бһҹбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбһ№бһ„бһ–бһёбһўбһҸбҹ’бһҗбһ“бҹҗбһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҠбҹӮбһӣбһӮбһҪбһҡбһІбҹ’бһҷбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮбһҳбһ¶бһ“бһҮбһҪбһҡбһҠбҹҒбһҖ бҹҘбҹЎбҹҘ,бҹ бҹ бҹ  бһ“бһ·бһ„бһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡ бҹЎбҹӨбҹўбҹ§ бһ“бһ·бһ„бһ“бһёбһҳбһҪбһҷбҹ—бһҳбһ¶бһ“бһҮбһҳбҹ’бһҡбһҫбһҹбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҒбҹ’бһӣбҹҮбҹ—бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһўбһ¶бһ…бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ”бһ¶бһ“бҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹӮбһӣ NLP бһҹбҹ’бһҳбҹ„бҹҮбһҸбҹ’бһҡбһ„бҹӢбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҹбҹ’бһҖбҹҒбһ“бһўбһҸбҹ’бһҗбһ”бһ‘ бһ“бһ·бһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһӮбҹҶбһ“бһ·бһҸбһҠбҹӮбһӣбһ–бҹҒбһүбһ“бһ·бһҷбһҳбһ”бҹҶбһ•бһ»бһҸ бһ бһҫбһҷбһҡбһ¶бһ”бҹӢбһңбһ¶бҹ”

бһўбһҹбһјбһҡбһҺбһ¶бһҹбҹӢ бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһӣбһҫбһ–бһ¶бһҖбҹ’бһҷбһҸбҹӮбһҳбһҪбһҷбһ‘бҹҒ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһӣбһҫбһ–бһ¶бһҖбҹ’бһҷбһ…бҹ’бһҡбһҫбһ“бһ–бһ¶бһҖбҹ’бһҷ (бһ§. *Business trip*)бҹ” бһҡбһҸбҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ—бһ¶бһ–бһүбһ№бһҖбһүбһ¶бһ”бҹӢбһ–бһ¶бһҖбҹ’бһҷбһ…бҹ’бһҡбһҫбһ“бһ–бһ¶бһҖбҹ’бһҷбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бҹ’бһҡбһҫбһ“бһ”бҹӮбһ”бһ“бҹҒбҹҮ (бҹҰ,бҹ§бҹҰбҹў,бҹҰбҹӨбҹҰ бһ–бһ¶бһҖбҹ’бһҷ) бһўбһ¶бһ…бһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҖбҹ’бһҡбҹ„бһҷбһҳбһ·бһ“бһҳбһҫбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһңбһ¶бһ бһ¶бһҖбҹӢбһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһ…бҹҶбһҺбһ¶бһҷбһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹӮбһӣбһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹ’бһҳбһҫбһҹбҹ’бһҳбһ¶бһүбһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһғбһҫбһү бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғ Tags бһҠбһјбһ…бһҮбһ¶ `[' Business trip  ', ' Solo traveler ', ' Single Room ', ' Stayed 5 nights ', ' Submitted from  a mobile device ']` бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҹбһҪбһҡбһҗбһ¶ бһҸбһҫбһўбһ¶бһ…бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҠбҹӮбһҡбһ¬бһ‘бҹҒбҹ” бһҹбҹҶбһҺбһ¶бһ„бһӣбҹ’бһў бһңбһ¶бһўбһ¶бһ…бһ”бһ¶бһ“ вҖ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹ’бһҡбҹ’бһҸбһҸбҹ’бһҡбһјбһңбһҮбҹҶбһ бһ¶бһ“бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢ Tags бһҠбҹӮбһӣбһӮбһҪбһҡбһІбҹ’бһҷбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбҹ”

### бһ…бһҳбҹ’бһҡбҹ„бҹҮ Tags

бһ…бһјбһҡбһ…бһ„бһ…бһ¶бҹҶбһҗбһ¶бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҜбһҖбһҹбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбһәбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ бһ“бһ·бһ„бһҮбһҪбһҡбһҲбһҡбһҠбҹӮбһӣбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ (бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҒбҹ’бһӣбһҪбһ“бһҜбһ„ бһ¬бһўбһҸбһ·бһҗбһ·бһҮбһ“бһҠбҹӮбһӣбһ…бһ„бҹӢбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸ bot бһ•бҹ’бһҸбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚбһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡ)бҹ” бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҹбһҪбһҡбһҒбҹ’бһӣбһҪбһ“бһҜбһ„бһҗбһ¶ Tags бһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ¬бһўбһҸбҹӢбһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҖбһ¶бһҡбһ–бһ“бҹ’бһҷбһӣбҹӢбһҳбһҪбһҷ (бһ”бһҫбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҜбһҖбһҹбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ бҹҒбһҸбһ»бһ•бһӣбһ•бҹ’бһҹбҹҒбһ„ Tags бһҒбҹ’бһӣбҹҮбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ¬бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҪбһ…бһ…бҹҒбһүбһ–бһёбһҖбһ¶бһҡвҖӢ selection)бҹ–

1. бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҠбҹҶбһҺбһҫбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’ бһ бһҫбһҷбһӮбһҪбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ
2. бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҖбҹ’бһҡбһ»бһҳбһ—бҹ’бһүбҹҖбһңбһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢ бһ бһҫбһҷбһӮбһҪбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ
3. бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ”бһ“бҹ’бһ‘бһ”бҹӢ бһҹбҹҠбһ»бһҷбһҸ бһ¬бһҹбҹ’бһ‘бһјбһҢбһёбһҷбҹ„бһҠбҹӮбһӣбһ—бҹ’бһүбҹҖбһңбһ”бһ¶бһ“бһҹбҹ’бһ“бһ¶бһҖбҹӢбһ“бҹ…бһӮбҹ’бһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢ (бһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҳбһ¶бһ“бһ”бһ“бҹ’бһ‘бһ”бҹӢбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ•бҹ’бһ‘бһ¶бһӣбҹӢ)
4. бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһҸбҹҶбһ”бһ“бҹӢбһҳбһҸбһ·бһӮбҹ’бһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢ
5. бһ…бҹҶбһ“бһҪбһ“бһҷбһ”бҹӢбһҠбҹӮбһӣбһ—бҹ’бһүбҹҖбһңбһҹбҹ’бһ“бһ¶бһҖбҹӢбһ“бҹ… *бһўбһ¶бһ…*бһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбҹ’бһ“бһ¶бһҖбҹӢбһ“бҹ…бһҷбһјбһҡбһҮбһ¶бһҳбһҪбһҷбһ…бҹҶбһҺбһ„бҹӢбһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸбһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡбһӣбһҫбһҹ а№Ғаё•а№Ҳбһңбһ¶бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳ бһ бһҫбһҷбһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһӮбҹ’бһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ

бһҮбһ¶бһ—бһ¶бһӮбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ **бһҡбһҖбҹ’бһҹбһ¶ Tags бҹў бһ”бҹ’бһҡбһ—бҹҒбһ‘ бһ“бһ·бһ„бһҠбһҖ Tags бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ…бҹҒбһү**бҹ”

```Python
# бһҠбһҖбһҹбһүбҹ’бһүбһ¶бһӮбһјбһ”бһҪбһҹбһ”бһҫбһҖ бһ“бһ·бһ„бһ”бһ·бһ‘бһ…бҹҒбһү

# бһҠбһҖбһҹбһүбҹ’бһүбһ¶бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһҹбһ¶бҹҶбһ„бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ•бһ„бһҠбҹӮбһҡ
df.Tags = df.Tags.str.replace(" ', '", ",", regex = False)
```

бһҖбһ¶бһҡбһҠбһҖбһ–бһ¶бһҖбҹ’бһҷбһҲбһ”бҹӢ бһ¬бһ–бһ¶бһҖбҹ’бһҷбһҮбһ¶бһ—бһ¶бһҹбһ¶бһўбһ„бҹӢбһӮбҹ’бһӣбҹҒбһҹбһ’бһҳбҹ’бһҳбһҸбһ¶бһҠбҹӮбһӣбһҳбһ·бһ“бһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһ“бҹғбһ”бҹ’бһҡбһҷбҹ„бһӮ бһӮбһәбһҮбһ¶бһҮбҹҶбһ бһ¶бһ“бһ‘бһёбһҳбһҪбһҷбҹ” бһҠбҹ„бһҷбһҠбһҖбһ–бһҪбһҖбһңбһ¶бһ…бҹҒбһү бһўбҹ’бһ“бһҖбһңбһ·бһ—бһ¶бһӮбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһӮбһҪбһҡбһҡбһҸбҹӢбһ”бһ¶бһ“бһӣбһҝбһ“бһҮбһ¶бһ„бһҳбһ»бһ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҸбһ·бһ…бһҮбһ¶бһ„бһҳбһ»бһ“бһЎбһҫбһҷ (бһ–бҹ’бһҡбҹ„бҹҮбһ–бһ¶бһҖбҹ’бһҷбһҲбһ”бҹӢбһҳбһ·бһ“бһҳбһ¶бһ“бһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ–бһҪбһҖбһңбһ¶бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһҷбһәбһҸ)бҹ”

бһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҸбһ·бһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“бһңбҹӮбһ„бһ”бҹҶбһ•бһ»бһҸбһҳбһ¶бһ“ бҹЈбҹ©бҹҘ бһ–бһ¶бһҖбҹ’бһҷ бһ”бҹүбһ»бһ“бҹ’бһҸбҹ’бһҡбҹ’бһ‘бһ…бһ¶бһ”бҹӢбһҸбһ¶бҹҶбһ„бһ–бһёбһҠбһҖбһ–бһ¶бһҖбҹ’бһҷбһҲбһ”бҹӢбһ…бҹҒбһү бһӮбһәбһ“бҹ… бҹЎбҹ©бҹҘ бһ–бһ¶бһҖбҹ’бһҷбҹ”

бһҖбһ¶бһҡбһҠбһҖбһ–бһ¶бһҖбҹ’бһҷбһҲбһ”бҹӢбһҖбҹҸбһҮбһ¶бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбһҝбһ“бһҠбҹӮбһҡ бһҖбһ¶бһҡбһҠбһҖбһ–бһ¶бһҖбҹ’бһҷбһҲбһ”бҹӢбһ–бһёбһҮбһҪбһҡбһҲбһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҸбһ· бҹў бһҮбһҪбһҡбһ–бһёбһҮбһҪбһҡбһҠбҹҒбһҖ бҹҘбҹЎбҹҘ,бҹ бҹ бҹ  бһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣ бҹЈ.бҹЈ бһңбһ·бһ“бһ¶бһ‘бһёбһ“бҹ…бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бҹ” бһңбһ¶бһўбһ¶бһ…бһҷбһәбһҸбһ¬бһҶбһ¶бһ”бҹӢбһҮбһ¶бһ„бһ“бҹҒбҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖаёӮаё¶а№үаёҷбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ CPU бһҳбһҪбһҷ RAM бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚ бһҳбһ¶бһ“бһңбһҸбҹ’бһҗбһ»бһ…бһӣбҹҗбһҸ SSD бһ¬бһӮбҹ’бһҳбһ¶бһ“бһ бһҫбһҷбһ”бҹүбһ»бһ“бҹ’бһҸбһ¶бһҷбҹүбһ¶бһ„бһҺбһ¶бһҖбҹҸбһҠбҹ„бһҷбҹ” бһҡбһҷбҹҲбһ–бҹҒбһӣбһҒбҹ’бһӣбһёбһ“бҹҒбҹҮбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһ”бһҫбһңбһ¶бһҮбһҪбһҷбһ”бһ„бҹ’бһҖбһҫбһ“бһӣбҹ’бһ”бһҝбһ“бһңбһ·бһ—бһ¶бһӮбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ бһңбһ¶бһӮбһҪбһҡбһҸбҹӮбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбҹ”

```python
from nltk.corpus import stopwords

# бһҠбһ¶бһҖбҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ•бҹ’бһүбһҫбһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡбһ–бһё CSV
df = pd.read_csv("../../data/Hotel_Reviews_Filtered.csv")

# бһӣбһ»бһ”бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҲбһ”бҹӢ - бһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһҹбҹҠбһёбһҮбһҳбҹ’бһҡбҹ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһҸбҹ’бһҗбһ”бһ‘бһ…бҹ’бһҡбһҫбһ“!
# Ryan Han (ryanxjhan бһӣбһҫ Kaggle) бһҳбһ¶бһ“бһўбһҸбҹ’бһҗбһ”бһ‘бһӣбҹ’бһўбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһңбһ¶бһҹбҹӢбһңбҹӮбһ„бһӣбһ‘бҹ’бһ’бһ—бһ¶бһ–бһ“бҹғбһңбһ·бһ’бһёбһӣбһ»бһ”бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҲбһ”бҹӢбһ•бҹ’бһҹбҹҒбһ„бҹ—
# https://www.kaggle.com/ryanxjhan/fast-stop-words-removal # бһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҠбҹӮбһӣ Ryan бһ”бҹ’бһҡбһ¶бһ”бҹӢбһҠбһ¶бһҖбҹӢ
start = time.time()
cache = set(stopwords.words("english"))
def remove_stopwords(review):
    text = " ".join([word for word in review.split() if word not in cache])
    return text

# бһӣбһ»бһ”бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҲбһ”бҹӢбһ–бһёбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ
df.Negative_Review = df.Negative_Review.apply(remove_stopwords)   
df.Positive_Review = df.Positive_Review.apply(remove_stopwords)
```

NLTK бһ•бҹ’бһҸбһӣбҹӢбһўбҹ’бһ“бһҖбһңбһ·бһ—бһ¶бһӮбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһ•бҹ’бһҹбҹҒбһ„бҹ—бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбҹҖбһ“ бһ бһҫбһҷбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҮбҹҶбһ“бһҪбһҹбһ–бһҪбһҖбһңбһ¶бһ”бһ¶бһ“ бһ бһҫбһҷбһҳбһҫбһӣбһҗбһ¶ бһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҳбһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҒбҹ’бһӣбҹҮ бһ¬бһҖбҹҚбһҒбҹ’бһңбҹҮбҹ” бһңбһ·бһ—бһ¶бһӮбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ VADER бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбҹ”
> Hutto, C.J. & Gilbert, E.E. (2014). VADER: бһҳбҹүбһјбһҠбҹӮбһӣбһҡбҹҖбһ”бһ…бҹҶбһӣбһҖбҹ’бһҒбһҒбҹҗбһҺбҹ’бһҢбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбһ¶бһҳбһүбҹ’бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһ·бһ—бһ¶бһӮбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһ“бҹғбһўбһҸбҹ’бһҗбһ”бһ‘бһ”бһҺбҹ’бһҸбһ¶бһүбһҹбһ„бҹ’бһӮбһҳбҹ” бһҖбһ·бһ…бҹ’бһ…бһҹбһ“бҹ’бһ“бһ·бһ”бһ¶бһҸбһўбһ“бҹ’бһҠбһҡбһҮбһ¶бһҸбһ·бһҮбһ¶бһӣбһҫбһҖбһ‘бһёбһ”бҹ’бһҡбһ¶бҹҶбһ”бһёбһӣбһҫбһ”бһҺбҹ’бһҸбһ¶бһүбһўбҹҠбһёбһ“бһ’бһәбһҡбһҺбҹӮбһҸ бһ“бһ·бһ„бһ”бһҺбҹ’бһҸбһ¶бһүбһҹбһ„бҹ’бһӮбһҳ (ICWSM-14)бҹ” Ann Arbor, MI, бһҒбҹӮбһҳбһ·бһҗбһ»бһ“бһ¶ бҹўбҹ бҹЎбҹӨбҹ”

```python
from nltk.sentiment.vader import SentimentIntensityAnalyzer

# бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбһ·бһ—бһ¶бһӮбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ vader (бһҳбһ¶бһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ NLTK бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бһ¶бһ“бһ•бһ„бһҠбҹӮбһҡ)
vader_sentiment = SentimentIntensityAnalyzer()
# Hutto, C.J. & Gilbert, E.E. (2014). VADER: бһҳбҹүбһјбһҠбҹӮбһӣбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ…бҹ’бһ”бһ¶бһ”бҹӢбһҸбһ·бһ…бһҸбһҪбһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһ·бһ—бһ¶бһӮбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһ“бҹғбһўбһҸбҹ’бһҗбһ”бһ‘бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһҹбһ„бҹ’бһӮбһҳбҹ” бһҹбһ“бҹ’бһ“бһ·бһҹбһёбһ‘бһўбһ“бҹ’бһҸбһҡбһҮбһ¶бһҸбһ·бһӣбһҫбһҖбһ‘бһёбһ”бҹ’бһҡбһ¶бҹҶбһ”бһҪбһ“бһўбҹҶбһ–бһёбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ”бһҺбҹ’бһҠбһ¶бһүбһ“бһ·бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһҹбһ„бҹ’бһӮбһҳ (ICWSM-14)бҹ” бһўбһ¶бһ“бҹӢбһўбһ¶бһ”бҹӢбһ”бҹҠбһҫбһҡ, MI, бһҒбҹӮбһҳбһ·бһҗбһ»бһ“бһ¶ 2014бҹ”

# бһҳбһ¶бһ“бһҮбһҳбҹ’бһҡбһҫбһҹ 3 бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбҹ–
# бһңбһ¶бһўбһ¶бһ…бһҮбһ¶ "бһӮбҹ’бһҳбһ¶бһ“бһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“", бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ“бҹҒбҹҮ бһҹбһјбһҳбһ”бһ„бҹ’бһңбһ·бһӣ 0
# бһңбһ¶бһўбһ¶бһ…бһҮбһ¶ "бһӮбҹ’бһҳбһ¶бһ“бһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“", бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ“бҹҒбҹҮ бһҹбһјбһҳбһ”бһ„бҹ’бһңбһ·бһӣ 0
# бһңбһ¶бһўбһ¶бһ…бһҮбһ¶бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҳбһҪбһҷ, бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ“бҹҒбҹҮ бһӮбһҺбһ“бһ¶бһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһ“бҹ„бҹҮ
def calc_sentiment(review):    
    if review == "No Negative" or review == "No Positive":
        return 0
    return vader_sentiment.polarity_scores(review)["compound"]    
```

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһўбһ“бһ»бһңбһҸбҹ’бһҸбһңбһ¶бһ‘бҹ…бһӣбһҫбһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҸбһ·бһҳбһҪбһҷбһҳбһҪбһҷбһҠбһјбһ…бһҸбһ‘бҹ…бҹ–

```python
# бһ”бһ“бҹ’бһҗбҹӮбһҳбһҮбһҪбһҡбһҲбһҡбһ–бһёбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ бһ“бһ·бһ„бһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“
print("Calculating sentiment columns for both positive and negative reviews")
start = time.time()
df["Negative_Sentiment"] = df.Negative_Review.apply(calc_sentiment)
df["Positive_Sentiment"] = df.Positive_Review.apply(calc_sentiment)
end = time.time()
print("Calculating sentiment took " + str(round(end - start, 2)) + " seconds")
```

бһ“бҹҒбҹҮбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһ”бҹ’бһҡбһ бҹӮбһӣ бҹЎбҹўбҹ  бһңбһ·бһ“бһ¶бһ‘бһёбһ“бҹ…бһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ¶бһ“бһ№бһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҸбһ¶бһҳбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ“бһёбһҳбһҪбһҷбҹ—бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһӣбһ‘бҹ’бһ’бһ•бһӣ бһ бһҫбһҷбһҳбһҫбһӣбһҗбһ¶бһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҹбһҳбһҹбҹ’бһҡбһ”бһ“бһ№бһ„бһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҸбһ·бһҷбҹүбһ¶бһ„бһҠбҹӮбһҡбһ¬бһ‘бҹҒбҹ–

```python
df = df.sort_values(by=["Negative_Sentiment"], ascending=True)

```
