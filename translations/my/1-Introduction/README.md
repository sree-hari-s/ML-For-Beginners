# machine learning အကြောင်းအကျဉ်း

အခု သင်ခန်းစာပိုင်းကြီးမှာတော့ machine learning ရဲ့ အခြေခံ အယူအဆတွေ၊ ၎င်းဘာလဲ၊ သမိုင်းကြောင်းနဲ့ သုတေသနပညာရှင်တွေက ရှာဖွေ အသုံးချတဲ့ နည်းပညာတွေကို တွေ့ဆုံလေ့လာမှာဖြစ်ပါတယ်။ ML ရဲ့ စိတ်လှုပ်ရှားစရာကမ္ဘာကို အတူတူဖော်ထုတ်ကြရအောင်!
