# Panimula sa machine learning

Sa seksyong ito ng kurikulum, ipinapakilala sa iyo ang mga pangunahing konsepto ng machine learning, kabilang ang kung ano ito, ang kasaysayan nito, at ang mga teknik na ginagamit ng mga mananaliksik upang ilapat ito sa mga tunay na sitwasyon. Tuklasin natin nang sama-sama ang kapanapanabik na mundong ito ng ML!
