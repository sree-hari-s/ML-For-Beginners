# Introduktion till maskininlärning

I denna del av kursplanen kommer du att introduceras till de grundläggande begreppen inom maskininlärning, inklusive vad det är, dess historia, och de tekniker forskare använder för att tillämpa det i verkliga scenarier. Låt oss utforska denna spännande värld av ML tillsammans!
