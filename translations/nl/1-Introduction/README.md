# Introductie tot machine learning

In dit gedeelte van het curriculum word je geïntroduceerd in de fundamentele concepten van machine learning, inclusief wat het is, de geschiedenis ervan, en de technieken die onderzoekers gebruiken om het toe te passen in scenario's uit de echte wereld. Laten we samen deze opwindende wereld van ML verkennen!
