# 機械学習入門

このカリキュラムのセクションでは、機械学習の基本概念、つまりそれが何か、その歴史、そして研究者が実世界のシナリオに適用するために使用する技術について紹介します。さあ、このワクワクするMLの世界を一緒に探検しましょう！
